# 第309封信丨信息技术有什么独特之处？

## 读者，你好！

今天和你分享一下，我从《创新者》这本书中看到的两个独到而有趣的观点。

《创新者》这本书的作者是艾萨克森（Walter Isaacson，《乔布斯传》的作者），这本书实际上是过去两个世纪以来信息技术的编年史，书中讲述了很多 IT 发明的故事，对于那些主要发明者，书中虽然介绍了他们的工作，但对他们为什么能做出重大发明，其实分析得不是很透彻。

但是，透过作者讲的故事，你可以理解信息技术以及由它产生的相关产业，和传统的工业产业有非常大的不同之处。

先来看第一个特点。你可能已经注意到了， *IT 产业不仅发展迅速，而且经过了近两个世纪的发展，至今看不到放缓的迹象。* 这和其他产业大不相同，无论是交通运输、航空航天，还是化工、医学和能源等领域，通常在高速发展一段时间后，发展的速度就趋于平稳了。

比如今天的轮船比两百年前快不了多少，今天我们使用的依然是半个世纪前设计的波音  747 飞机，在过去的 30 年里，人类对癌症的认识并没有多少提高，治疗手段进步也有限。

但是，信息领域不一样，它总是不断地给我们带来惊喜，这一点大家都是受益者，就不需要我再赘述了。从这一点我们至少可以得出一个结论，年轻人投身 IT 行业，你不用担心今后没有饭吃。

 *信息产业第二个特点是，相应的产品和服务价格在不断下降，这是它能够渗入到社会生活方方面面的根本原因。* 虽然每一种工业品在生产率提高后，价格相对于我们的收入并没有提高，但是绝对的价格还是在上涨的，从日用品到汽车、电冰箱或者医疗仪器都是如此。在传统的服务行业价格的上涨更是明显。

但是，在 IT 领域，价格总是不断下降的，而提供的产品的性能，服务的质量却在提升，这是在人类历史上第一次出现的反常现象。

出现了反常现象必有原因。通常，我们将上述现象归结为人类源源不断的创造力，更具体地讲是技术的创新。但是问题又来了，为什么这种创造力和创新集中体现在 IT 领域呢？对于这个问题的答案可谓是仁者见仁，智者见智。

艾萨克森在书中就试图通过研究两个世纪以来 IT 技术的发展，给予大家一些新的理解这个现象的方法。下面，我就和你讲一下其中的两个。

第一个是数学对 IT 行业产生了不可估量的影响，而其他行业，受到数学的影响不是那么大。我们知道世界上最早试图制造计算机解决复杂问题的是数学家巴贝奇（Charles Babbage），他设计了一种能够解决微积分中差分问题的计算机。

这台计算机里面包含无数个齿轮。虽然巴贝奇并没有真正做出来这台计算机，但是后人根据他的设计复制出来了，并且证明是可以工作的。而影响巴贝奇来做这件事的是谁呢？她是大诗人拜伦爵士的女儿埃达·拜伦，一位对数学痴迷的贵妇名媛。

从那个时候起，人们就在考虑数学＋机械＝智力这样的事情。因此计算机这个作为 IT 革命的代表性产品，从一开始就和数学思维联系在一起。

那么哪些带有智能水平的事情可以通过计算解决，什么不可以呢？或者说，人和机器的边界在哪里呢？一直在思考这个问题的是数学家图灵（Alan Mathison Turing）。

而图灵则又受到另一位数学大师冯·诺依曼（John von Neumann）的影响，图灵的很多灵感来自冯·诺依曼的《量子力学的数学基础》一书。图灵甚至想，我们自由的意志，是否就是因为量子力学中的不确定性所导致的。当然这件事情很难证实。

就在图灵苦思冥想思维和（巴贝奇机械计算机）运算的边界的时候，另一个著名数学家希尔伯特（David Hilbert）的一些思想给了图灵启发，图灵顺着这些数学家的思路，终于想清楚什么是可计算性这件事，并且提出了可计算理论，在此基础上他建立了计算的数学模型。我在之前的第 301 封信中也谈到了当年图灵的这个思考过程。

知道什么样的问题可以转化成计算问题后，大家自然就会问怎么让机器实现这些计算呢？其实在图灵之前另一位天才已经解决了这个问题，他就是大名鼎鼎的信息论的发明人香农（Claude Elwood Shannon）。

香农在做硕士论文时，就设计了数字逻辑电路，这种电路能够实现所有的计算。今天的集成电路，无论多么复杂，里面基本的单元都是香农的逻辑电路。因此香农的这篇只有十几页纸的论文，也被誉为 20 世纪最重要的硕士论文。

好了，现在万事俱备，只欠东风，需要有人将这些数学理论变成计算机。这个任务当时交给了莫奇莱（Mauchly）和艾克特（Eckert）师徒二人，遗憾的是他们设计的计算机不是通用的，只能解一道题。

所幸的是，数学家冯·诺依曼因为一个偶然的机会（他自己也要用计算机）加入了进来，帮助他们将计算机的系统结构改成了通用的，这就是冯·诺依曼系统结构，今天所有的计算机都是这种结构（和它的一个变种）。

因此可以讲， *将一切问题变成数学问题是过去两个世纪以来 IT 革命的核心。或者说数学的基因，数学家的思维方式，是 IT 行业能够快速发展的重要原因。*

今天，计算机不仅速度快，而且非常聪明，以至于很多人担心计算机的智能会超过人。其实这种担心是杞人忧天，我在很多场合解释了原因，这里我从另一个角度来解释一下。

如果说数学是整个 IT 革命的基础，那么是否存在不可计算的问题呢？答案是肯定的，图灵就找出了这样的问题。而这些不是靠计算解决的问题，人能够给出答案（虽然答案未必完美），但是计算机对此无能为力。了解了计算的这个本质，你就不必担心人工智能的发展了。

对于 IT 技术不断发展起了关键作用的另一个因素来自于一种文化，或者说是一种政策。艾萨克森在书中把这一切归功于一个人，麻省理工学院的教授万尼瓦尔·布什（ Vannevar Bush）。

布什过去曾经发明了模拟计算机，当然这种技术后来被数字计算机所取代，二战时，布什是曼哈顿计划的主要倡导者，二战后他是主导了政府科技政策的学者。

布什为美国确立了走政府、大学和企业的合作来发展 IT 产业的道路，而不是采用由国家出钱建立大型实验室的方式。这对于美国计算机的普及，半导体产业的发展，以至于后来互联网的出现和繁荣，起到了至关重要的作用。

相比之下，整个欧洲虽然在二战前科技水平不逊于美国，但是它在 IT 时代的发展模式出了问题，一切由政府主导，以至于落后于美国。

在艾萨克森看来，没有任何一个美国人对科技发展的影响能超越布什。正是在他的思路指导下，美国才能做到 IT 巨子辈出，盖茨、乔布斯等大批创新者，都是在这样的环境下诞生的。

然而，大部分国家在发展科技时，看重的是盖茨、乔布斯这样的人，但是他们的贡献是相对孤立的，他们的出现也具有偶然性。但是，当一个国家在文化（和制度）层面对创新有了正确的理解时，盖茨和乔布斯这样的人就会成批出现。

上述两个观点，是我读《创新者》这本书的收获。总的来讲这本书是介绍 IT 历史的，我想你今后如果真的去读它的话，会更关注这方面的内容。而对于我来讲，它讲述的 IT 的故事我早知道了， *我所感兴趣的是他看问题的视角，这可以帮助我提高认知水平。*

这本书对你能有什么启示呢？如果你不是政策的制定者，万尼瓦尔·布什的作用你是发挥不了的。但是每一个人都可以通过了解 IT 产业的数学基因知道 IT 的边界。

数学让 IT 很纯粹，可以不受很多限制快速发展，但是也让它有局限性，毕竟世界上还有数学之外的东西。

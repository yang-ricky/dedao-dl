# 第316封信丨明白的大妈和糊涂的记者

## 读者，你好！

最近在微信上看到了这样一段对话，很有意思，我先转述一下：

> 记者：马云推出无人超市了，您怎么看？
> 
> 
> 
> 大妈：那啥？超市都没人啦，那还不关门干嘛？
> 
> 
> 
> 记者：大妈，无人超市不是没有人这个意思，而是说，超市里没有售货员、收银员等员工了。
> 
> 
> 
> 大妈：那应该叫无员工超市啊！哎，就你们这语文水平，还当记者呢！
> 
> 
> 
> 记者：是是，大妈说得对，应该叫无员工超市。大妈，那您对这种新型的超市有什么看法呢？
> 
> 
> 
> 大妈：超市不需要养员工了，那东西是不是更便宜啦？
> 
> 
> 
> 记者：这个……我们暂时还没了解到。
> 
> 
> 
> 大妈：瞧瞧你们这些记者怎么当的？老百姓最关心的问题，你们不去了解，整天只会关心马云又弄啥玩意了。我们老百姓最关心的是什么？有没有假货，东西是不是更便宜啦！超市里有没有员工，关我啥事？
> 
> 
> 
> 记者：您不觉得无人超市的推出将会改变我们传统的购物方式吗？
> 
> 
> 
> 大妈：改变啥哟？买东西不花钱啦？刷支付宝那也是花钱呐！
> 
> 
> 
> 记者：大妈，看来您还是不能理解时代的发展潮流。
> 
> 
> 
> 大妈：哟，弄个没有员工的超市就是时代潮流啦？每天都弄些专门裁减底层员工的玩意儿算啥本事？有本事弄个没有老板的超市啊！ 
> 
> 
> 
> 记者：大妈，您对马云是不是有意见啊？
> 
> 
> 
> 大妈：我不是对马云有意见，是对你这种无聊的记者有意见，问问题从来都问不到点上。马云改变了我们的生活，但我们要的不仅仅是改变，而是带来幸福的改变。现在很多改变不仅没有增添我们的幸福，还增添了许多烦恼！这才是你们记者应该关注的问题。
> 
> 
> 
> 记者：……

这段对话的背景是前一阵子阿里巴巴发布了推出无人超市的消息，可能你已经读过这则新闻，如果没有的话可以在网上搜一下。不少记者得知这条新闻之后很嗨，甚至叫板亚马逊——你们没做到的事情我们中国人率先做到了。

我看了这则报道和相关的很多评论后当时并不以为然，因为那个所谓的“超市”，不过是大一点的自动贩卖机而已。我既不是马云的粉丝，也不是他的反对者，当时对这件事情也没有太上心。

事实上马云自己也没有把它太当回事，因为没有见他在各种场合到处解读它，这和百度的几个负责人到处解读人工智能的做法完全不同。

有趣的是很多记者（和一些“吃瓜群众”）的反应，把这一件并不大的事作了很多夸大的解读。相比之下，倒是对话中的那个大妈几乎每句话都一语中的，因此我甚至认为那个大妈其实是某个资深科技人士或者商业人士的化身。

这位大妈讲的第一个观点道出了消费者真实关心的是什么。是购物时的放心（没有假货）以及价格，而并非通过什么方式购买东西。网购之所以能够兴起，只是因为它价格低，省时间，并非因为它很酷。

因此，大妈问没有了售货员，价格是否可以便宜，则是这种无售货员商场能否成功的关键，而不在于它的技术本身。如果没有了售货员，顾客可能会毁坏里面的货物，或者变相偷盗一些商品，那么这种无售货员商店里东西的价格还可能会增加。

在美国，很多人在超市偷盗小商品的行为并不完全是因为无钱支付商品，从好莱坞明星到财富 500 强公司高管都曾经被抓到过有偷盗行为，前者顺了个几十美元的化妆品，后者随手拿了一个一两美元的打火机。

在美国、欧洲和澳大利亚，并非没有无售货员的计价交款柜台，这确实省了一些人手，但是柜台需要由一个保安或者售货员监控。因此，有没有售货员不是问题的核心，是否利用新技术提高了效率，降低了成本才是核心。

今天很多科技记者，混淆了技术在目的和手段之间应有的位置。 *技术从来都是手段而不是目的，搞不清楚这个关系，就会为了技术而研发技术。很多时候，我们知道一个需求可能是刚需，但是对这个需求的解决方案并不一定是刚需，甚至不因为某个解决方案使用了新技术就称为刚需。*

在美国，FDA（美国食品药品监督管理局）每年都会勒令下架一批药，包括很多新药，其原因除了个别的药副作用太大，大部分是因为没有用途。也就是说虽然治疗相应的疾病是刚需，但是解决方案却未必是。

对老百姓来讲，购物是刚需，但无售货员的超市却未必是，尽管这是一个新概念。类似地，无论是刷卡，还是用手机支付，花钱这件事并不会因此省略掉，要想过好日子，从本质上讲，是要获得更多的钱，而不是方便花钱。

在上面的对话中，最后记者抛出一句话，“看来您还是不能理解时代的发展潮流”。在论战中，这样的话常常是败下阵来之前最后说的话，类似的话还有“你又不是这方面的专家”，“和你说了你也不懂”，等等。

当一个人抛出这样的话时，说服别人的努力就已经失败了九成。具体到这件事，一个时代发展的潮流并不是记者说了算的。

什么时候该采用自动化，什么时候根本不需要，有些时候不能仅仅考虑技术的因素。

爱迪生爱讲他第一个重要发明后来失败了的例子。当时电的使用刚刚开始，爱迪生就发明了一种供议会使用的自动投票机，并且获得了他的第一个专利。

爱迪生以为有了自动投票机，议员们就可以加快议会投票的过程，提高效率。但是，当他带着这项专利来到国会时，议员们却告诉他这东西根本没有用，然后就把他打发走了。

后来他才了解到，议会出于公平的考虑，要给少数派足够的时间来说服其他人，因此出于决策流程的考虑，国会投票的过程并不需要加快。

这次碰壁以后，爱迪生懂得了一个道理， *这个世界不仅需要技术，而且技术还需要有用，有市场。* 从此，爱迪生抛弃了单纯发明家的思维方式，即要把所有的事情都自动化，转而领悟到了企业家的思维方式，并且一生再也没有做任何没有市场的发明了。

在爱迪生的时代，并不缺乏好的发明家，特斯拉等人便是如此。所不同的是，很多发明家虽然想法很好，甚至可以讲非常超前，但是想法不切实际，最后的结局都不算好。

而那个时代真正留下姓名的大发明家，比如爱迪生、贝尔和德国的西门子，都不是为了发明而发明，而是通过发明切实解决了人类的现实问题。

我们在每一个时代，都希望用技术对社会进行改变。但是仅仅谈改变是没有意义的，变好才是目的。2008 年奥巴马竞选总统时，高呼改变的口号，吸引了很多对现实不满的选民。

当时，朱利安尼(纽约市前市长)就指出这种说法的问题。他讲， *改变本身不是目的，因为有好的改变和坏的改变之分，搞好才是目的。* 朱利安尼认为，像奥巴马这样没有任何管理经验的人难以改良社会。事实证明，当初投奥巴马票的绝大部分人，八年后变得更加贫穷了。

在我们现实生活中，总有人希望通过“改变”来让自己的命运变得更好。比如有读者问我，自己到了 30 多岁，感觉事业前途一般，是否该开始学习编程，这样可以踏上技术革命的快车。

虽然我不能否认极个别的人可能有过去没有发现的计算机的天赋，但是这种可能性很小，更何况要成为一个优秀的计算机工程师，除了要有近万小时的刻意练习，更重要的是学会计算机科学特殊的思维方式。

今天大部分科班出身的计算机从业者尚且没有掌握这里面的精髓，更何况30多岁改行学计算机的人了。因此，这样的改变几乎可以肯定并不能比原来改得更好，他们多少有点为了改变而改变的意味。

爱因斯坦讲，“真理就是在经验面前站得住脚的东西”。对话中的大妈并不知道什么高科技，却懂得很多朴素的道理，即那些在经验面前不断被验证过的道理。

相反，那位记者的想法则需要在经验面前验证，看看是否站得住脚，才能下结论——也就是那种放大了的自动售卖机是否能代表商业的“时代发展潮流”。

 **最后，总结一下今天的内容：**

1. 新的商业模式是否可行，要看能否让东西更便宜，购物更方便。

2. 技术是手段，不是目的，不能为了发明而发明。

3. 对老百姓来讲，具有更多的钱，比更方便地花钱更重要。

4. 改变不是目的，改变有好有坏，我们需要的是变得更好。

5. 真理就是在经验面前站得住脚的东西。

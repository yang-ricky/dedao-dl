# 第082封信 | 为什么世界长期实行“金本位”的货币制度？

昨天我们讲到了最早的货币的两个特性， **自身的价值和记账工具，而能够同时满足这个特性在早期只有大宗商品了。** 但是大宗商品难以运输，不便于开展商业，于是，就需要使用有价值同时又便于携带的物品。今天我们来讲讲真正意义上货币的诞生，以及为什么世界长期实行的金本位很重要。

虽然很多人都同意贝币是最早的便于携带的货币，但是贝币不具备普遍意义，并非所有的文明都使用了贝币。因此贝币并非货币出现的一个必经阶段。

最早的货币源于何时何地，学者们至今观点看法不一。主要原因是在它作为记账工具和交易工具上的分歧。如果强调货币的记账功能，最早的货币应该是古代埃及、美索不达米亚、印度以及中国宫殿和神庙的一些存单。在那些文明中，宫殿和神庙会有仓库或者养殖场，保存大宗商品。当然那些大宗商品并非都属于国王或者神庙，一些贵族或者有钱人也会把自己的财产存在那里，而王宫和神庙就会给他们一些凭证。

此外，在美索不达米亚地区，大家彼此借一些东西也会打欠条。一些学者认为，所有这些凭证，就是最早的非实物商品货币。最早的这一类凭据出现在美索不达米亚，距今至少有四千年。在《汉谟拉比法典》和更早的《乌尔纳姆法典》中，都记载了欠债要还利息。这也被认为是有了货币的证据。

但是，很多学者反对这种说法，因为那些凭单只具有记账特性，不能用于交换。因此他们认为最早的货币应该是由国家铸造的金属货币，因为它们具有流通性。但不论采用哪一种说法，至少大家在两点上是有共识的。

其一，那就是 **货币需要由权威机构发行，** 这个共识一直延续至今。其二， **不论是根据抵押的大宗商品发行的票据，还是铸造的金属货币，其实都是广义上的金本位。** 在上述两种货币起源的理论中，我更倾向于接受后一种，即铸造的金属货币才是真正意义上最早的货币。不过，第一种观点也有意义，它揭示了货币发行的一个基本条件——抵押品。

我们知道任何历史都需要是连续的，不可能跳变。从大宗商品到金属货币需要有一个过渡的阶段。在这个过渡阶段，金属成为了一种新的大宗商品，取代了谷物、牛羊等，作为衡量物品价值的中介。据考证，在公元前第四个千年纪，古埃及人就用金块作为实物货币进行交易，同时代美索不达米亚人用白银作为实物货币，进行交易。不过，这种交易并不普遍。今天，（支持后一种观点的）一些学者则认为从实物货币到金属货币这个过渡最早出现在中国的商代。

在商代，中国的青铜制造已经很发达了，但是青铜依然很珍贵。因此，它同时具有了大宗商品和贵重金属双重属性。不过，说它是大宗商品，也只有贵族才能使用。因此，天子和贵族使用青铜来给予奖赏或者发放俸禄。当然，如果青铜只是用作这两个功能，那还只是具有记账功能，还不具有流通性。到了西周早期，周恭王时代（公元前1000年左右）所铸造的曶鼎的铭文上有这样的记载，“用囗卖丝五夫，用百寽”。寽是古代重量单位。 **学者们认为这一条记录了当时用一定数量的铜交换了人，这时铜就成了流通货币了。**

当然，作为流通货币不能够每次都称重量，于是铸造出大小重量相同的金属货币就成为了必须。有些学者认为，3000年前中国已经出现了青铜的刀币，甚至有人认为更早的商代的铜贝也是货币，但是这种说法的证据并不是很充足。不过从出土的文物来看，自公元前7世纪开始，在美索不达米亚、古希腊和古代中国，几乎同时出现了真正的金属货币。

金属货币有三个好处。

首先， **它比较贵重，本身就具有价值，不易贬值。** 其次， **它便于保存、携带。** 第三， **它的制造需要一定的技术，原材料也可以控制，因此很容易被官方垄断。** 就这样，在很长的时间里，金属货币成为了全世界所有文明共同使用的货币，而且这些文明在这一点上并没有彼此商量。

但是，金属货币也有三个缺陷。

首先因为 **货币的原材料稀缺，因此数量有限，当商品经济发展起来之后，钱就不够用了，** 这会带来经济上的一系列问题。比如中国在明朝时期，经济情况就和从美洲输入的白银数量息息相关。不同地区、不同时期铸造的金属币的重量和成色不一，因此很快会出现劣币驱除良币的现象。最后，虽然它们相比大宗商品便于携带，但是毕竟交易量大了，携带起来还是很不方便。于是， **当人们把货币的记账功能强调出来，把本身的价值淡化，就出现了纸币。**

今天能找到的最早的纸币出现在宋代的四川省，就是大家都听说过的交子。交子是古代四川话，意思是票券、凭证。为什么交子会出现在四川呢？原因是宋太祖的一项政策。当时四川是后蜀的地盘，被统一进宋朝的时间较晚，因此宋太祖在占据四川后，就把当地的金、银和铜都运往了汴梁，于是四川人只能用铁铸钱了。由于铁原料便宜，要想有点价值，就必须铸得很重，当时1000文铁钱重25斤，出门买个东西要用车子拉钱，十分不方便。于是，大家将铁钱寄存在“交子铺”中，换取票据交子，替代铁钱作交易。

如果我们对比一下交子出现的过程，它和古代文明存储大宗商品的凭证有很多相似之处，和雅浦岛上的费币的使用也颇为相像，都是利用了钱作为记账工具的特点。但是和早期文明存储大宗商品的凭证所不同的是，交子这种凭证用于了流通，因此它是真正意义上的货币了。

交子出现后，今天纸币的所有缺陷它马上都具有了。

 **首先是容易造假，** 于是官府采用了红黑套印的方式印刷，并用秘密记号和图样防伪。官府甚至定期作废旧币，强制百姓换新币，以达到防伪的目的。即便如此，造假依然有。

其次，虽然开始的时候每一贯交子纸钞都对应一贯铁钱，但是很快交子铺就将寄存的财物用于投资赚钱了，于是多次发生挤兑，而且从此面值一贯的纸币，很快贬值成只有七八百钱的价值。再到后来，北宋中央政权干脆收回了交子和另一种纸币钱引的发行权，并且从哲宗朝开始乱发货币。所幸的是，两宋经济比较发达，虽然纸币不断贬值，倒也没有引起严重的经济危机。

 **交子从完全由金属货币抵押到由中央政府随意发放，在无意中完成了纸币进化的过程，即从广义上的金本位（包括其它金属），到以国家的信用作抵押发行货币的转变。** 在古代中国以国家信用发行货币的条件并不成熟，因此到了明代，又恢复到以白银加铜钱的货币体系。在西方，直到20世纪初，货币的发行依然普遍采用金本位。也就是说，发多少纸币，从理论上讲需要在中央银行抵押等量的黄金。

最后对今天的内容做一个小结：

 *1. 从方便记账，方便流通的目的出发，货币本身并不需要是具有价值的金属币。* 而且由于纸币可以大量发行，更便于宏观调控经济，因此当商品经济高度发达之后，采用纸币要比金属货币更合理。

 *2. 纸币难免会被多发，因此需要有政府和自身有价值的金属货币背书，* 否则相应的金融系统会崩溃。

同时提示一下后面要讨论的话题，比特币是否能流通，要看是否符合上述原则，更加便利，有更好的背书。当然，在讲比特币之前，我们还要介绍一个人类在金融上的大发明——平衡表。

 **辩论赛参谋薛兆丰老师：**

腾讯的 Q 币不算货币。我们在《薛兆丰的经济学课》中，从第一讲开始就解释过，货币是价值的一个公共尺度，它既可以免除人们在物物交换中必须满足的“双重偶然性”要求，从而增加交易的机会，也可以让人们避免在交易中付出不必要的质检成本，从而减少交易的难度。

从上述两个标准看，腾讯 Q 币基本不具备上述功能。目前，用户只能用法币（如人民币）兑换 Q 币，而 Q 币却不能兑换成法币；同时，用户只能用 Q 币为自己或别人向腾讯平台购买虚拟物品或会员服务，但用户之间不能用 Q 币购买对方提供的商品或服务。

这些限制从根本上阻止了 Q 币成为一般意义上的货币。以其有限的“购买对象”和单程的“流通方向”判断，Q 币只是腾讯平台内部流通的一种功能非常有限的服务票证而已。

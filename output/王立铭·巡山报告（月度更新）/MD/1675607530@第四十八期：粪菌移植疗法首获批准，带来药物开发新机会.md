# 粪菌移植疗法首获批准，带来药物开发新机会

你好，我是王立铭。2023年2月6日，第四十八期《巡山报告》又和你见面了。

不知不觉，《巡山报告》已经进行了四年，希望给你带来了一些收获和启发。从下期开始，我们一同开始第五年的巡山。

2022年11月30日，美国药监局正式批准了一款粪菌移植产品（简称FMT，fecal microbiota transplantation）进入临床应用。 **这款商品名为Rebyota的产品是有史以来第一款被正式批准的此类产品，标志着利用人体肠道菌群治疗疾病这个全新医疗模式的巨大突破。**

借此机会，我们这期《巡山报告》就来系统谈谈这背后的科学原理和药物开发机会。

# 首款粪菌移植药物Rebyota

首先，我们具体介绍一下Rebyota。

这款药物的开发者是美国生物技术公司Rebiotix。这款药物的特征与很多传统的药物类型，比如小分子化学药物、重组蛋白药物、单克隆抗体药物、细胞药物都存在显著的差异。

简而言之，它是将健康人的粪便收集起来，逐步清洗过滤，收集其中的细菌组分，再加入特殊溶液后保存备用；使用的时候，将这些细菌悬液直接通过灌肠送入患者肠道。 *和传统药物最大的差别可能是，粪菌移植是一类“活”药物，药物的有效成分是来自多达几百上千种活的细菌。*

 **Rebyota这款药物获批的适应症是避免艰难梭菌感染（简称CDI，Clostridioides difficile infection）的复发。**

艰难梭菌在自然环境中广泛存在，但在成年人肠道中比较罕见。一般情况下，即便少数艰难梭菌通过各种路径——特别是食物——进入人体消化道，它们的生产繁殖也会被人体免疫系统和肠道中的菌群抑制。

但在少数情况下，特别是服用大量抗生素之后，人体肠道内的细菌被快速清除，于是艰难梭菌就有机会繁殖、释放毒素并引起肠道的剧烈免疫反应。患者往往会出现腹泻、发烧、腹痛的症状，更严重者可能丧命。以美国为例，每年会出现大约50万例艰难梭菌感染，直接导致接近3万人死亡 [1]。

大部分艰难梭菌感染在停止使用抗生素之后会很快好转，严重的艰难梭菌感染往往需要动用甲硝唑、万古霉素、非达霉素等药物来治疗。不过，艰难梭菌感染的一个棘手问题是反复发作。有研究认为，复发概率可能高达35% [2]。患者的病情往往会伴随着多次复发而逐渐加重，而相关的抗生素却可能因为细菌耐药而逐渐失效。因此，如何有效预防艰难梭菌的感染就成为了临床急需解决的问题。

既然艰难梭菌感染很大程度上是因为人体肠道的天然菌群被破坏，那么如果把健康人的肠道菌群补充进去，重建患者被严重摧毁的肠道菌群，也许就可以治疗和预防。这正是粪菌移植疗法的逻辑基础。

1958年，美国科罗拉多医学院的医生利用粪菌移植手段治疗多个伪膜性肠炎患者，取得了很好的疗效 [3]，但当时人们并不完全理解这种方法为何有效。到了1978年，人们意识到，伪膜性肠炎的病因正是艰难梭菌感染，于是粪菌移植的合理性才得到科学支持。1983年，瑞典医生为一位65岁的复发性艰难梭菌感染患者移植了分离自她丈夫的粪菌样本，成功治疗了她的肠道疾病，并且在9个月的随访期内没有观察到疾病复发 [4]。

从那之后，世界各国的不少医疗机构都在尝试开展类似的粪菌移植治疗，并且在超过90%的患者中看到了良好的疗效 [5]。2013年，在一项对照研究中，接受粪菌移植的艰难梭菌感染者治愈率远高于使用万古霉素的常规治疗方案[6]。

但是相比其他药物，粪菌移植的特殊性大大限制了它在临床的广泛应用。

和其他药物较为单纯的有效成分、严格的生产和质量控制程序不同，粪菌移植的成分是从健康人粪便中提取的细菌。每一份粪菌药物，到底有哪些种类的细菌、数量和活性多少、进入肠道后哪些可以成功定植，从源头上就是难以控制的。而且，粪菌移植还涉及较为复杂的手术操作，比如灌肠、结肠镜、胃镜等，具体的移植部位和用量也多有不同。

因此在2013年，美国药监局出台新规，将粪菌移植疗法正式纳入药品监管范畴，而不仅仅将它看作医疗机构进行的常规手术操作 [7]。

这在一定程度上限制了医疗机构开展粪菌移植治疗的频率。因为在这项规定下，如果想要开展粪菌移植，就要向药监局提交正式的临床试验申请，获得批准后才可以按照申请中的设计方案进行操作。

但与此同时，这项规定也意味着，粪菌移植手段如果经历了药监局的严格审查，将有可能成为上市药品，从而获得更大的应用空间。

Rebyota的上市正是在这条路径下成功实现的。在2022年结束的一项三期随机对照双盲试验中，接受Rebyota治疗的患者治愈率显著高于使用安慰剂灌肠的患者 [8]，同时药物的安全性良好。正是基于这些数据，美国药监局正式批准了Rebyota的上市申请。这意味着，全美各地的医疗机构都可以使用由Rebiotix公司生产的粪菌产品进行移植。

# 人体共生菌和人类疾病

Rebyota的成功上市当然具有历史意义，但在行业内部可能并不算多么重大的新闻。毕竟， **除了Rebyota，还有多款用于治疗复发性艰难梭菌感染的粪菌移植药物也已经非常接近于得到监管机构批准。**

比如，美国SERES公司的粪菌产品SER-109也已经在三期临床试验中取得了积极疗效，而且和Rebyota不同的是，这款产品是口服使用，方便程度大大提高 [9]。Finch公司的口服粪菌产品CP101、Vedanta公司的口服粪菌产品VE303，也在二期临床中取得了不错的成绩 [10] [11]。

 *真正让学术界和业界都感到兴奋的，是粪菌类产品在艰难梭菌感染这种单一疾病之外的想象空间。*

要想说清楚这一点，我们首先要回到人体共生菌（commensial bacteria）这一概念。在人体各个器官，比如皮肤、生殖器、口腔、以及肠道内部，都寄生着大量细菌，种类可能多达1000种，数量更是可能高达100万亿个，超过了人体细胞的总数 [12]。这些细菌被称为“人体共生菌”。

在传统的生物医学研究中，共生菌群的作用往往是被忽略的。人们只关注到少数场合下特定共生菌的具体作用。比如，乳酸菌和双歧杆菌能够在肠道内合成维生素B和维生素K，是人体这些维生素的重要来源之一。

但在过去20年间，人们越来越多的意识到： *人体共生菌作为一个整体，在人体健康和疾病中起到的作用是非常深刻和广泛的。在免疫系统疾病、代谢系统疾病、神经系统疾病、肿瘤和心血管疾病中我们都能看到，人体共生菌发生了系统性的变化，起到了意想不到的正向或负向作用。* 基于这些发现，我们甚至可以把共生菌看成是维护人体活动至关重要的一个“外部”器官。

为了方便你理解，我用两个典型的案例来拆解一下人体共生菌和疾病之间的关系，以及这些关系如何指导临床治疗。

 **一个例子是共生菌和癌症之间的关系，或者更具体的说，是共生菌和癌症免疫疗法的关系。**

你很有可能听说过癌症免疫疗法，它可以被看成是癌症治疗领域半个世纪以来最具革命性的进展。代表性的药物是靶向PD-1/PD-L1通路的纳武单抗（Nivolumab，“O药”）和帕博丽珠单抗（Pembrolizumab，“K药”），以及靶向CTLA通路的伊匹单抗（lpilimumab）等。相关概念的提出者已经获得了2018年度的诺贝尔生理学或医学奖。

这类药物的作用机制，我们可以这么粗略地理解：癌细胞作为一种异常细胞，想要成功繁殖复制，需要通过某些特定的信号为人体免疫系统“踩刹车”，从而逃逸免疫系统的识别和追杀。而如果设计药物反其道而行之，破坏这些刹车的功能，就有可能唤起免疫系统的活动，杀伤癌细胞。

从这个角度看的话，癌症免疫药物和共生菌实在是完全不相关的两个生物医学概念。但在2013年，两个实验室发现，在小鼠肿瘤模型中，多种常用肿瘤化疗药物的作用都依赖于共生菌的存在，在无菌小鼠体内，化疗药物的作用被大大减弱 [13][14]。这是人们首次意识到共生菌和癌症的关系。

2015年，研究者们又在小鼠模型中发现，肿瘤免疫药物的作用同样需要共生菌的存在 [15][16]。

到了2018年，研究者们更是在人体中进行了确认：在黑色素瘤患者中，人体对癌症免疫药物的反应程度也和共生菌群的特征有关 [17][18]。相反，使用抗生素会降低癌症患者对癌症免疫药物的响应 [19]。

在上面的研究中我们已经知道，共生菌的一些特征，比如菌群的多样性、某些特定共生菌的丰度，可能起到了关键作用。这些菌群的特征可能会影响人体免疫系统的活动，从而改变癌症患者对多种癌症免疫药物的响应。但更细节的机制，比如到底是哪些细菌、通过什么方式、影响了哪些免疫细胞的什么功能，目前仍然需要进一步的深入研究 [20]。

 **第二个例子显得更为惊世骇俗——人体共生菌和自闭症之间的关系。**

自闭症主要表现为社交和人际关系障碍、语言表达困难、情绪控制障碍、刻板行为等，严重的自闭症患者可能还会出现智力障碍。目前，全球范围内自闭症发病率约为1% [21]，美国等发达国家更是超过了2% [22]。自闭症一般被认为是一种和大脑发育缺陷相关的疾病，为什么会和共生菌产生关系呢？

2010年，医生首先确认，许多自闭症儿童会出现肠道功能方面的问题，比如饮食习惯异常、消化不良等 [23]。

到了2013年，研究者们首次在小鼠模型上证明了自闭症和肠道共生菌之间的关系 [24]。研究者发现，如果母鼠在怀孕期间遭遇病毒感染或者免疫刺激，生下来的小鼠就会出现自闭症的典型特征，包括行为障碍、消化系统异常等。而如果给这些小鼠的肠道内植入一种共生菌（脆弱拟杆菌），上面这些异常都会得到缓解。这是人们首次在自闭症和共生菌之间建立因果关系。

随后的几年中，人们比较了自闭症患者和健康人的肠道共生菌群特征，确实发现两者存在显著的差别 [25]。而在2019年，人们又进一步确认，如果把自闭症患者的肠道共生菌群植入小鼠体内，小鼠也会出现自闭症的行为异常 [26]。甚至人们还发现，只要小鼠妈妈的肠道菌群出现紊乱，小鼠宝宝就有可能让患上自闭症 [27]。

当然，必须强调的是，在具体机制层面，人们还并不十分清楚肠道菌群的紊乱到底是如何影响大脑发育、从而导致自闭症出现的。细菌分泌的代谢物质可能直接影响大脑发育；细菌可能影响了免疫系统的功能，从而激发了大脑中的炎症反应，都是可能的解释 [28]。实际上，也有研究者认为，肠道菌群并不导致自闭症，相反，是自闭症患者的饮食异常导致了肠道菌群的紊乱 [29]。

但无论如何，肠道菌群和人类精神疾病存在千丝万缕的联系这一点还是相当可靠的，也有不少小规模的临床研究在直接测试粪菌移植是否可以改善自闭症患者的病情 [30]。

# 共生菌药物的前景

 **基于上面这些生物医学研究的发现，业内不少公司正在积极推进利用人体共生菌治疗各种人类疾病。**

比如我们刚刚提到的Rebiotix公司、SERES公司、Finch公司和Vedanta公司，都分别有针对自闭症、炎症性肠炎、肿瘤免疫治疗等方面的药物开发布局。技术路线大同小异，仍然是基于粪菌移植：将健康人的肠道共生菌提纯植入患者体内，期待它帮助患者重建共生菌的平衡，改善疾病症状。

但我们必须看到，上面提到的粪菌移植的问题仍然是存在的：

粪菌移植的活性成分来自健康人的肠道，而每个健康人肠道中的细菌类别、数量千差万别，很难确定哪种组合才有最好的临床收益。药物的整个制备和移植过程也存在很多不可控的因素，比如，我们很难确认到底有多少移植的细菌能够成功定植在患者肠道中并成功繁殖。

更为严重的问题是，即便是健康人的肠道共生菌，也总是或多或少地携带一些可能的致病菌，只是在大多数时候，它们的数量和活性有限，并不会导致疾病而已。而在粪菌移植操作中，这些致病菌也会被无差别地提纯和转移到患者体内，引发的后果可能是不可控的。

2019年，一位血液疾病患者在接受粪菌移植治疗后死亡，事后研究者们发现，是因为粪菌中带有一种罕见的耐药性大肠杆菌 [31]。此后，美国药监局对粪菌移植治疗的监管进一步加码，要求医疗机构和医药公司筛查粪菌样本中可能存在的致病菌和耐药菌。但毕竟这种筛查是有针对性的，无法穷尽所有可能的危险因素。比如，在新冠流行期间，粪菌样本是否会传播新冠病毒就成了新的风险点，也让美国药监局暂停了几个粪菌移植的临床研究 [32]。

可想而知， **只要我们仍然在使用粪菌移植这种相对比较粗糙的手段，就无法彻底回避它的几个核心问题，比如有效成分变化幅度大、生产程序难以严格控制、致病菌和耐药菌的传播等。**

当然，这些问题也有一个明显的解题思路。 *如果我们能够进一步挖掘出具体哪种共生菌可以治疗疾病，甚至哪种共生菌生产的哪个化学物质可以治疗疾病，就可以越过粪菌移植本身的问题，更精准地利用细菌这种“活药物”来治疗疾病。*

在这个大方向上，我认为，有三个精细程度层层递进的技术路线可供参考：

 **第一个路线是找到治疗疾病的单一菌种。**

这其实是我们熟知的一种思路。目前市面上所有的益生菌产品，本质上都是单一菌种制剂或几个身份清晰的菌种的混合物，这些菌种都是载入各国食品目录的 [33]。当然，一般意义上的益生菌，比如最广为熟知的乳酸菌，实际上在改善健康、治疗疾病方面并没有可靠证据，也不被各国监管机构允许做相关宣传。

但在肠道菌群药物的未来开发中，这一思路却值得借鉴。比如我们刚刚提到的脆弱拟杆菌，是否可以单独做成药物用于自闭症患者的治疗，本身就是值得关注的一个方向[34]。

 **第二个路线则是提纯或合成特定菌种分泌的有效化学物质，用于治疗疾病。**

比如，如果我们认为脆弱拟杆菌可能用于治疗人类疾病，那么与其使用数量难以控制、随时能在繁殖过程中发生变异的活细菌，还不如分析这种细菌到底是如何影响人体健康的，提取其中单一的有效成分、开发成药物。

在针对脆弱拟杆菌的研究中，研究者就发现，它表面的多糖分子（PSA, Polysaccharide A）能替代细菌，起到治疗炎症性肠炎 [35]、预防肠癌 [36] 等作用。相比直接使用脆弱拟杆菌，使用这种多糖分子可能就是更安全、更可控的手段。美国也有不少初创公司在试图挖掘肠道菌群的小分子代谢物，用于包括代谢疾病、精神疾病、免疫系统疾病的药物开发[37]。

 **第三个路线则是反其道而行之，对细菌进行遗传改造。**

这个路线的核心是利用共生菌能够在人体中长期存活和繁殖的特征，把它们变成持续释放药物的载体。

就在2022年，Aurealis公司完成了一项一期临床试验：将乳酸菌进行改造，让它持续生产和释放三种能促进伤口愈合的蛋白质，然后将这种细菌敷在糖尿病患者溃烂的足部伤口，有效地促进了伤口愈合 [38]。2021年，另一项研究发现，经过改造的大肠杆菌植入人体后，可以有效地治疗一种遗传疾病——苯丙酮尿症 [39]。

在我看来， *Rebyota这款药物的获批固然激动人心，但粪菌移植作为疾病治疗手段，可能只有短期的临床价值。在更长的时间尺度上，人们应该会朝着更精准的方向利用肠道菌群治疗疾病。*

当然，这些方向的探索注定不会轻而易举，考虑到人体共生菌是一个由上千种细菌、上百万亿个细菌形成的复杂生态系统，我们是否能够从中找到单一的成分用于疾病治疗，其实还要打一个大大的问号。但人类医学的发展从来都是沿着从粗糙到精准的路线进化，我们没有理由怀疑共生菌会是一个例外。

## 划重点

1. 和传统药物最大的差别可能是，粪菌移植是一类“活”药物，药物的有效成分来自多达几百上千种活的细菌。

2. 人体共生菌作为一个整体，在人体健康和疾病中起到的作用非常深刻和广泛。

2. 粪菌移植作为疾病治疗手段可能只有短期临床价值。未来，人们应该会朝着更精准的方向利用肠道菌群治疗疾病。

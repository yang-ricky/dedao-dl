# 第五十期：如何用干细胞逆转衰老？

你好，我是王立铭。2023年4月6日，第50期《巡山报告》又和你见面了。

2022年，中国人口出现负增长。和人口负增长同步到来的，是人口老龄化程度的持续加深。2022年的数据表明，中国60岁以上的人口比例已经达到18.9%，进入深度老龄化区间。

在这个背景下，中国社会经济生活的重心将难以避免地逐步向生育端和老年端倾斜。对生物医学领域而言，和生育相关的医疗产品，以及和衰老相关的医疗产品，都将出现长期需求。

这期《巡山报告》，我们就重点谈谈利用干细胞技术逆转衰老的前沿进展。

# 衰老的生物学本质

衰老和死亡是几乎所有多细胞生物个体必然的命运，人类当然也不例外。

 *从进化生物学的角度看，衰老并不是生物学机制精密调控下循序发展的结果，相反，它更可能是生育阶段结束后，各种生物学机制逐渐失效和失调的过程。* 如果要打个比方的话，衰老更像是一件精美的玻璃器皿逐渐磨损、裂缝、破碎的过程，而不是一辆赛车咆哮着冲向悬崖的过程。

因为在长期繁衍之后，生物个体的基因组中会积累大量的有害变异，客观上表现出的就是生物的各种生理功能逐渐衰退、最终走向死亡。

请注意，这个结论不仅在基础理论层面说明了衰老的本质，实际上还对人类抵抗甚至逆转衰老提供了重要的思想工具。

既然衰老不是生物体精密调控的结果，那就不太可能找到一个神奇的衰老“开关”，通过扳动它，就能延缓、停滞甚至逆转人体的全方位衰老。相反，研究者需要仔细探寻在衰老过程中逐渐失调和失效的生物学过程，设计方法重新激活和理顺它们，从而起到延缓甚至逆转衰老的作用 [1]。

不过必须得说，这个说法虽然在逻辑上能自圆其说，但现实应用起来却存在不少困难。特别是，如果人们试图从单个维度理解衰老过程，并借此对抗衰老的话，失败几乎是可以预见的。

其中，两个著名的例子就是端粒缩短和活性氧自由基增多。

端粒是染色体DNA末端的重复序列，起到保护染色体DNA完整性的作用，但它会随着细胞的不断分裂复制而持续丢失和缩短，从而影响染色体的稳定性。

活性氧自由基则是一系列化学物质，包括氧离子、过氧化氢等，它们的化学结构中含有未配对的活跃电子，非常容易和周围的物质发生氧化反应。在衰老的细胞内，线粒体功能的失调会产生大量的活性氧自由基，破坏细胞内很多物质的结构和功能。

在不同物种的衰老过程中，人们都能观察到端粒的缩短和活性氧自由基的升高。

按照刚刚讨论的逻辑，如果能找到办法阻止端粒的缩短，或者降低活性氧自由基的水平，就应该能够延缓甚至逆转衰老。但在现实中，研究者却无法观察到显著且一致的抗衰老效果。

比如，虽然在小鼠中，激活能够延长端粒的端粒酶，确实能延缓衰老 [2]；但在人类中，端粒较长的反而可能有更高的癌症发生率 [3]。而且如果我们做一个跨越物种的比较更会看到，端粒长的动物可能反而寿命短，端粒短的动物可能反而寿命长，小鼠和人类就是如此。

再比如，过量表达能够消除活性氧自由基的蛋白SOD1，根本无法延缓小鼠的衰老 [4]，在线虫中甚至发现了相反的结果：删除能够消除活性氧自由基的SOD基因，反而能延长线虫的寿命 [5]。

至于这些失败背后的原因，我推测仍然和衰老的复杂性有关。既然衰老伴随着许多生物学过程的逐渐失效和失调，那么我们就很难通过改变其中的一个过程，在个体身上观测到立竿见影的抗衰老效果。例如活性氧自由基，它除了和衰老过程相关，更是细胞内重要的生物信号分子，简单粗暴地操纵一下，有害无益也是可以理解的。

# 表观遗传学时钟

既然单点突破的思路有局限，那么相应的， **人们也可以通过一个更系统的思路来理解衰老，那就是细胞遗传物质的持续变化，特别是表观遗传学变化。**

我们知道，遗传信息的载体是DNA，DNA序列的变化，我们一般归入遗传学的研究范畴。而所谓表观遗传学，指的是在不改变DNA序列的条件下，对基因表达水平、细胞内生物学过程，乃至整个生物体特征的改变。

比如，一个受精卵持续分裂，形成一个由几十万亿个细胞构成的人体。这些细胞的DNA来自同一个受精卵，差别极其微小，但它们却承载了许多差异巨大的生物学功能。

一对基因组DNA完全相同的同卵双胞胎，刚出生的时候，从相貌到行为几乎没有差别，但伴随着成长发育和人生起伏，两者的差异会越来越明显。再比如，同一个人类个体，在幼年和老年时期，基因组DNA序列没有发生什么变化，生理机能却存在天壤之别。

上面说的这些差异，只能来自DNA序列之外，也就是表观遗传学的范畴。

表观遗传的物质基础有很多类型。比如，基因组DNA的化学修饰，特别是DNA碱基的甲基化，就能够在不改变DNA序列的条件下，改变基因表达的强度；负责染色体组装的组蛋白化学修饰，也有类似的功效。

而且，表观遗传还有着非常广泛的内涵，比如RNA分子的化学修饰，非编码RNA、蛋白质翻译后的修饰，细胞器的数量和形态，都是表观遗传可以发挥作用的舞台。

 **很多人相信，衰老过程并不会改变细胞和生物个体的基因组DNA序列，但可以通过改变其表观遗传特征，破坏基因的正常表达和蛋白质分子的正常功能，从而引起生物学过程的混乱。**

早在2011年，研究者就进行了这样的尝试：观察人类的衰老是否和人体表观遗传水平的变化有直接关系。他们发现，基因组DNA上几十个位置的甲基化都和年龄相关，而其中区区三个位点的甲基化修饰水平就可以很好地预测人类个体的年龄 [6] 。

研究者随后又发现，如果在全基因组尺度上研究DNA甲基化和年龄的关系，还能进一步提高对年龄的预测准确度 [7]。

因此，研究者提出了所谓“表观遗传时钟(epigenetic clock)”的概念。他们认为，人类基因组DNA甲基化水平的持续变化，反映了衰老的持续发生，这不仅仅体现为生理年龄的增加，还体现为健康状况的持续衰退，多种慢性疾病的逐渐发生，以及死亡率的持续提高 [8][9]。

2019年，研究者们甚至利用表观遗传时钟在9位健康的受试者中做了一个小规模临床研究，引起了全球主要媒体的关注。9位受试者接受了生长激素的注射，用于唤醒他们衰老的免疫系统。1年后，检测这9位受试者的表观遗传时钟，发现他们平均而言比同龄人“年轻”了2岁半 [10]。

而更重要的问题是，在衰老过程中，表观遗传学时钟是如何计时的？如果我们假设，表观遗传学修饰的逐渐失控是衰老的幕后驱动者，那么到底是什么因素引起了它的失控呢？

2023年1月，哈佛大学医学院David Sinclair实验室的一项研究为这个问题提供了一个重要的解释视角。他们认为，DNA分子的随机断裂破坏了表观遗传修饰的稳定性，从而导致了生物的衰老 [11]。

在这项研究中，研究者们利用转基因技术，在小鼠体内表达一个来自多头绒泡菌的核酸内切酶，它可以在小鼠基因组的20个特定位置上切割双链DNA，制造DNA断口。这种特殊的断口能够被及时修复，基本不会引起DNA序列的变化，却能显著改变小鼠的表观遗传学特征，相当于拨快了小鼠的表观遗传学时钟。

结果发现，这大大加快了小鼠各种生理指标的衰老，比如体重下降、毛发变灰白、代谢率降低、驼背、骨密度降低、肌肉量和收缩能力下降、运动机能下降、认知功能受损等。

这项研究首次说明，扰乱生物的表观遗传学修饰就足以引发快速的衰老。

值得注意的是，尽管这项研究中的DNA断点是人为诱导而来的，但生物的生活环境中有很多因素足以引发类似的DNA断点，比如射线辐射。

这样一来，我们也许可以用这样一种全新的视角来理解衰老：

 **在生物体成长发育、繁衍生息的过程中，细胞中的DNA持续遭受内外危险因素的侵犯，出现各种DNA断点。这些断点大部分都能被快速和准确地修补，并不影响DNA序列的准确性。但不幸的是，在断点产生和修复的过程中，生物体的表观遗传修饰会出现混乱，而这些混乱的积累将把生物体带入不可逆转的衰老和死亡过程中。**

# 干细胞逆转衰老

那么，有没有什么办法整体逆转生物体的表观遗传学时钟，实现对衰老的阻击，甚至是逆转呢？

 *有一个看起来很有希望的思路，就是干细胞技术，特别是人工诱导干细胞技术。它提供了一种将衰老的细胞重新拉回年轻态、重置衰老时钟的可能性。*

“干细胞”这个概念早已是万众瞩目。究其本质，干细胞表示的其实是细胞的分化潜能，也就是，一枚细胞到底能产生多少种功能各异的后代。

人类这样的多细胞生物，从小到大发育成熟的过程是靠一枚受精卵细胞持续不断的分裂实现的。这个细胞一变二、二变四、四变八，最后形成几十万亿个的过程，就伴随着“潜能换功能”的变化。

受精卵是所谓的全能干细胞，能够分裂产生人体所有类型的细胞。但在持续分裂的过程里，它的后代会逐步失去这种无所不能的分化潜能，只能执行单一的生物学功能，比如成为皮肤的一部分，成为大脑神经系统的一部分等。

这个过程有点像巨石从山头滚下的过程：在山顶的时候，巨石拥有无限种滚落的路径选择；但在滚下的过程中，它的速度越来越快，可供选择的路径越来越少；直到跌落地面，在地面忠实地执行被赋予的生物学使命。

长久以来，人们默认这个石头滚落的过程是绝对的单行道、完全不可逆。

直到2006年，日本科学家山中伸弥在《细胞》杂志发表论文 [12]，证明在已经完成分化的细胞当中，人为的表达四个基因，分别是Oct4、Sox2、Klf4和c-Myc，简称OSKM，或者叫“山中因子”，这些细胞就能重新变成具备各种分化潜能的干细胞状态。

这项研究引爆了整个生命科学领域，很快就有人尝试在临床试验中用这样的方法治疗疾病。道理很简单，原本可能需要做异体器官移植的患者，借用这项技术，可以直接用自己的身体细胞，加上山中因子，重新诱导回干细胞状态，再定向分化成需要的器官，进行移植就可以了。

举一个具体的例子：

美国国家健康研究院在2019年启动了一项200人规模的临床试验，试图用这个办法治疗老年黄斑变性。他们把患者的身体细胞取出，重新推上山顶，变回具备分化潜能的干细胞，然后再定向分化成视网膜细胞，注射到患者眼球中，替代那些过早死亡的视网膜细胞，挽救患者的视力 [13]。目前在全世界范围内，有近百项人工诱导干细胞的临床试验正在进行中，靶向包括大脑、血液、呼吸、内分泌等多种人体系统 [14]。

回到衰老的问题。如果直接把山中因子放到衰老动物的细胞里去，会有什么效果呢？

还真有人试过这个方法，结果很危险：如此操作后，小鼠全身多个器官会长肿瘤 [15][16]。原因也不难理解：山中因子的作用是把已经完成分化的细胞重新推上山顶，具备分化潜能。如此一来，这些被重新赋予分化潜能的细胞就有可能随心所欲地进行不受控制的分裂和分化，这本来就是癌细胞的典型特征。

这条抗衰老的路子就被彻底堵死了吗？可能也不是。

研究者们又尝试了一个新思路：既然持续供应山中因子会引发癌症，那能不能只是短暂的给身体细胞提供一点山中因子的刺激，部分的逆转这些细胞的命运呢？

2016年，美国索尔克研究所的Juan Carlos Izpisua Belmonte实验室尝试了这个思路，取得了巨大成功 [17]。他们发现，短暂地启动四个山中因子几天到十几天时间，不会导致肿瘤发生，反而会逆转小鼠身体细胞和多个器官的衰老，大大延长早衰症小鼠的寿命，从大约18周延长到24周。

当然，这项很漂亮的研究有一个硬伤：它是在早衰症小鼠上做的验证，正常小鼠有差不多2年的寿命，而早衰症小鼠携带一个罕见的遗传变异，未老先衰，普遍只能活三四个月。能让早衰症老鼠多活几周，和让正常衰老的动物重返青春，可能是性质完全不同的两件事情。

在2022年3月7日，同一个实验室的科学家在《自然-衰老》杂志上发表了一篇新的论文，对这个遗留问题做了进一步的解答 [18]。

这一次，他们改在正常小鼠体内表达四个山中因子。估计是受实验条件的影响，他们并没有观察这些小鼠的寿命是否得到了显著延长——可能是因为要完成对小鼠寿命的观察，至少需要等两三年时间，太久了。

但他们使用了一系列别的方法，来间接确认小鼠的衰老是否得到了逆转，比如身体细胞的基因表达和染色体DNA的状态是不是被显著改善，小鼠伤口愈合的能力是不是得到了增强，小鼠的整体代谢状况是不是也有改善。这些都是衰老中常见的问题。

除此之外，这项研究还有一个比较重要的发现：他们测试了长达7-10个月的山中因子刺激，发现在逆转衰老的同时，没有带来任何肿瘤风险。

这大大增强了人们对这项技术安全性的信心。毕竟在人体进行四个基因的遗传操作是相当冒险的举动，时间窗口也很难把握得特别精准，要是几个月的操作都安全，我们就更可以放心的在人体中测试几天这种时间尺度的抗衰老操作了。

2023年David Sinclair实验室的研究同样测试了山中因子，准确的说，测试了山中四因子当中的三个——OSK——的抗衰老效果。他们发现，先在小鼠中产生DNA断点、加速衰老，再人工诱导OSK三因子的表达，确实能够回拨表观遗传学时钟，逆转小鼠的衰老过程。

为什么在动物体内表达山中因子，人工诱导干细胞状态，能够逆转衰老呢？原因大概还要从表观遗传学时钟的角度去理解——

我们知道，细胞分裂的过程中，DNA序列并没有发生变化，是一系列表观遗传的变化为细胞赋予了不同的分化路线和具体功能。山中因子既然能将已经分化完成的细胞重新编程、成为具备分化潜能的干细胞，当然就需要“擦除”在分化过程中形成的一系列表观遗传变化，比如DNA甲基化修饰、组蛋白化学修饰等等，这一点已经得到了充分的证明 [19]。

而既然衰老过程和表观遗传修饰的混乱直接相关，山中因子当然就能擦除这些混乱信息，将衰老细胞的表观遗传时钟拨回到年轻状态了。

这样一来，我们就得到了一种全新的理解和对抗衰老的思路：

 *衰老的本质，可能是在生物体的生命周期中，表观遗传修饰逐步走向混乱的过程；*

 *这种表观遗传修饰的混乱，可能源于频繁的DNA断裂和修复过程；*

 *人工诱导干细胞技术，比如OSKM山中四因子，可以擦除细胞分化和衰老过程中出现的表观遗传修饰印记；*

 *这种表观遗传印记的擦除，能够产生回拨表观遗传时钟、逆转衰老的作用。*

当然，基于衰老过程的复杂性，我们大概很难通过一种方法就一劳永逸地实现对衰老的逆转。上面这种基于表观遗传学变化和干细胞技术的抗衰老思路，目前也仍然停留在细胞和小鼠模型当中，尚未在更大型的动物中得到检验，更不要说人类了。

而且我们更要注意，基于衰老过程的复杂性，表观遗传学变化是不是就能完全解释衰老，以及干细胞技术是否就仅仅是通过改变表观遗传学时钟来实现抗衰老的，可能都还不能给出一个非黑即白的回答。

但同样值得注意的是，这条逆转衰老的技术路线，除了能整体性的逆转衰老之外，还能选择性的在重点器官里使用。

2020年，David Sinclair实验室的另一项研究还证明，只在小鼠的视网膜里表达山中三因子OSK，就能显著的逆转视网膜细胞的衰老，提升视网膜的再生和修复能力 [20]。

于是，一个相对客观的期待是：人工诱导干细胞技术，也许有望在不久的将来用于治疗特定器官的早衰型病变。在确认安全和有效之后，我们可能才可以进一步畅想它出现在逆转人体衰老的战场。

## 划重点

一种全新的理解和对抗衰老的思路：
1.衰老的本质，可能是在生物体的生命周期中，表观遗传修饰逐步走向混乱的过程；
2.表观遗传修饰的混乱，可能源于频繁的DNA断裂和修复过程；
3.人工诱导干细胞技术，比如OSKM山中四因子，可以擦除细胞分化和衰老过程中出现的表观遗传修饰印记；
4.表观遗传印记的擦除，能够产生回拨表观遗传时钟、逆转衰老的作用。

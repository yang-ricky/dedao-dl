# 金融的思维方法论：金融与人生

我是香帅。今天给你串讲《香帅的北大金融学课》最后一个模块，金融思维方法论。

这个模块我觉得已经有一点脱离了金融学的本身，而我是想说，我们能不能够从整个金融的逻辑方法论里面，提出对自己的人生特别有用的一些思考方式和模式。我给你讲六个我认为特别重要的金融思维。

# 预期思维

 **第一个叫预期思维。** 我们都知道“预”就是未来，金融它本质上就是一个关于未来的学问，它绝对不是关于过去的。所以我们会看到，比如说股票市场上永远说现在的股价反映的是什么？不是你的历史业绩，是我们市场对于未来业绩的预期。当然，这个预期里面包含着你过去干得好不好，包含着过去的业绩。你过去干得好，我对你未来的预期可能会更好，但是这中间不能打等号。

比如说2023年的4月份，A股市场就炒很多小票，就是关于文旅的很多小票价格就涨得很高。为什么？大家预期2023年的五一节会是一个消费狂热复苏，大家拼命出去旅行的这么一个时间，所以预期这些公司的业绩会很好。你看，这就是预期。

那么好了，我们先不讨论这个预期对不对，但是我们把这几个思维方式放到生活中我们就会去思考，什么是好公司、什么是好股票？就是你对这个公司的未来预期非常的乐观，它会高增长，而且非常稳定，它不会七上八下，因为这个公司特别得稳定。

好了，同样的逻辑，我们把这个事放到自己身上。因为周边总是有小朋友跟我抱怨说，老师，我怀才不遇，我为什么不被大家认可？我说那就是因为你没有给市场一个足够好的预期，那你就要想一想，怎么去改善市场对你的预期，让市场觉得你是一个特别高增长的？你要做哪些事情让市场对你形成这种高增长的预期？这需要你特别的稳定，不是说今天好明天就不好，就叫稳定的预期。

所以，当我们在谈论资产的时候，我觉得其实也在谈论自己。 **有时候我在想，我们说一个好股票的时候，这种思维的框架，完全可以放到企业和个人的成长上面，就是要让市场对自己形成乐观的和长期稳定的一个预期。** 这就是预期思维，这个思维放到生活中，我觉得真的是很有用的，不要着迷于过去，更不要着眼于当下，而要放眼于未来。这就是预期思维。

# 贴现思维

第二个思维也是特别重要，叫贴现思维。现就是现在，贴就是折扣的意思。所以， **贴现你基本上可以理解为，未来价值和现在价值中间的一个折算。** 简单的数学就告诉我们了，贴现率越高，你现在的价格是不是折扣就越高，对于企业来讲也是，你要让自己这个折现率更低一点。同样对于人力资本也是一样，要让自己折现率更低一点。怎么让自己折现率更低一点？第一个就是高增长，第二个就是低折旧。

在这个课程里，我当时举了一个特别有意思的例子，比如说你们会发现好医生、好教授都是越老越值钱。因为医生、教授的知识结构，他们的技能是随着时间累积，然后甚至会有一个阈值，突破了一个阈值以后，增长还会更加快，这就是增速。好多比如说体力这种东西，增速慢慢是呈一个下降的趋势，所以我们就要找到这样的人生技能。

还有一个就是低折旧。什么东西是折旧很高的？就像我老是说女人的美貌就像真丝一样，洗了水以后慢慢就失去了光华。但什么东西折旧很慢呢？知识，然后就是你的学问，这些东西都是折旧很慢的。

产品也是一样，什么东西折旧很快？电子产品折旧很快。什么东西折旧很慢？茅台酒，折旧很慢。

所以你看，投资和人生真的特别像，当我们说要找那种低折旧、高增长的产品、赛道和企业的时候，完全可以放到自己身上，我们要找低折旧、高增长的技能。 **我有时候一直在讲贴现思维它对企业投资适用，但是你把自己看作一个金融资产，对你自己的个人成长也是完全适用的。** 尤其在这种人工智能冲击这么大的时代，这种高增长、低折旧特别特别的重要。

# 复利思维

第三个叫复利思维。复利这个词你们听得很多，它其实本质上就是一种计算利息的方法。就比如说今天我存100块钱进去，明年是110，然后明年我计算利息的时候是以110作为基数，那就变成了再10%，不就是121吗？就比你每年10%增长得要快。然后大家就会发现时间越长，这个复利就像滚雪球一样越滚越大。所以很多人会认为，复利是鸡汤，在我们生活中也会有这样的现象，比如说减肥、健身这些事情都有很强的复利在里面。

但是在这个课程里面，我跟你讲了在我们用复利思维的时候，你会发现为什么会有人批评复利是鸡汤？是因为他们没有讲初始阈值的问题。当时我在课程里给你举了两个例子，就是说如果两个基数差别很小，比如说一个是1，一个是1.01，进行同时复利的话，经过很多年以后，这中间的差别也没有那么大。

那什么就会复利差别大呢？如果一个是1，一个是1.2，那个复利的差别，过十年、二十年以后差别就会非常非常大。我在那篇课程里面就讲到了。

所以大家一定要重视这个初始阈值，就是到一定的值以后，包括财富增长、能力增长什么时候复利有用？初始阈值要足够大，那怎么才能够实现那个初始的阈值呢？就是初始的速度要大。

所以，当时我写完这篇课程以后，我就跟团队的小朋友们说，我说你们现在拿十万块钱来投复利怎么投？说心里话，我觉得都没有那么大的意义。我说你现在要干的是什么呢？你拼命花两三年让自己的这个初始的速度绝对的大，然后保证你的年收入从10万跳到30万、50万、100万，到这里以后，再开始用复利的思维进行投资，这样你的财富才会越卷越开。同样，财富这个词完全可以换成能力。

所以就是那种鸡汤，比如说一万小时定律，我真的觉得不是鸡汤。你在一件事情上没有花够足够的时间，没有突破掉一定的阈值的时候，你是感受不到复利的作用的。所以一件事情、一种思维方式是鸡汤还是真正有用的人生准则，真的看我们自己怎么想和怎么看。

# 风险思维

第四个是风险思维。金融不就是技术和工具专门来处理风险这件事情的吗？所以风险思维也是金融里面一个非常核心的思维。

在前面课程的串讲里面，我给你们讲过一个故事，我说有很多那种做金融交易的人，他们都特别强调一个底线思维。你看，就是最激进的威廉·江恩和利弗莫尔，这都是做股票投机很厉害的高手，但是他们一个强调不能够all in，要金字塔原则，慢慢一点一点地进。另外一个强调要止损，亏损到达一定程度，哪怕是砍断手脚，也一定要及时离场。这就是风险控制。所以你看，真正的高手是非常非常讲究风险底线思维的。

在课程里我们有两篇文章，我当时举了一个例子，我说金融投资跟我们打网球特别像，你会发现在高手的对决里面，最后决定他们胜负的是什么？不是你打出的制胜球，谁的unforced error ，是叫非受迫性失误更少，谁就赢得那个大战的胜利。

这跟投资是一模一样的道理，你在场上站得最久，你就是赢家，因为大部分人都“死”了，这个逻辑其实特别有用。

比如说我们在做投资的时候，我经常跟我们团队小朋友说，其实真正的高手在做投资的时候是要经常空仓的。就是你大部分时候应该是保持空仓的，所以我说那种每天都在看盘的人，大概率是要亏钱的，为什么？这种人其实是没有风险底线思维的，什么叫顺势而为？就是抓住一个好的机会，然后猛然一击，但是严格地控制自己的风险底线。记住这句话， **我不一定要做最对的决策，但是我要尽量少做错的决策，这样你的胜率其实是最高的。**

风险思维其实放在创业上、放在人生上都是一样。我不一定要做整个的创业群体里面最冒尖的那一个，但是我少犯错误，坚持到最后一定是跑出来的那一个。人生同样的道理，所以风险思维也是对我们的投资和对我们的人生都是非常非常重要的。

# 杠杆思维

另外一个就是杠杆思维。老是有人说要降杠杆，我说你要把杠杆降到零的话，就不需要金融了。你去想，你买房的时候是不是就用了杠杆？你付30%的首付，意味着你用了几倍的杠杆？用了3.3倍的杠杆，对不对？那么你的房子涨一块钱，你就涨了三块钱，你的房价涨了10%，你就获得了33%的收益。这是不是就是如果我们碰到好的机会、好的资产的话，杠杆就帮你放大你的财富，这也就是为什么我们团队好几个小朋友毕业两三年，他们就用比较低的首付买了房，财富快速积累起来了，就是用了杠杆，所以不用杠杆就不用做投资了。

但是，杠杆率控制在一个什么样的水平，用什么样的杠杆，这是有很多讲究的，这也是金融投资里面，我们一直在讨论的一个事情。

所以在杠杆原则里面，它有两点，第一个肯定是要借力，你不借力就不需要用杠杆了。但是，什么情况底下，你可以用杠杆？我给了另外一个词叫硬核，什么叫硬核呢？比如说现在我特别具有投资能力，这是我的硬核技能。那么我就可以多使用杠杆，它就能够放大我的财富。但是假设我这个判断是错的，我其实是个二愣子，我根本就不会投资，我放大杠杆，结果导致是什么？本来市场亏10%，我用五倍杠杆，我亏了50%，我赔得差不多血本无归了。

同样的道理，当我们讨论金融衍生品是魔鬼还是天使的时候，它的底层资产好，它就是天使，它就放大你的收益。它底层资产差，那就是毒药，就毒翻一大片。你看2008年的时候，金融危机里面不就是那种次级贷款，也就是有毒的资产加了很高，100多倍的杠杆以后，出现了全球性的金融危机。所以这也就是为什么， **诺奖得主罗伯特·希勒说，好的金融意味着好的社会，坏的金融会带来一个坏的社会。**

所以杠杆思维，也是金融思维里面非常重要的一个思维。我们要学会拿捏尺度。一方面你要在自己硬核的能力、硬核的资产、硬核的项目上碰到这种机会，人生中可能没有那么多，一定要加杠杆，这样子你才能够鲤鱼跳龙门。

但是，千万不要错估，在自己没有把握、不是很硬核的时候，不要去借力，一定要有安全意识，这个时候你就把风险思维加到你的思考框架里面。所以杠杆思维一个方面要借力，但是借力的时候要对自身的能力有一个正确的评估。

# 信用思维

最后就是信用思维。把这个放到最后，因为它不仅仅是金融世界的核心关键词，可以说信用也是人类社会的一个关键词。因为我们在整个社会上，我们做人做事到最后是什么？不就是一个立信的过程吗？我们老是说，我在团队里怎么样，我在银行的眼里是什么样子，我在父母家人眼里是什么样子。你去想一下，我们做的一切事情都是为了建立自己的信用体系，对不对？就是要使我这个人在别人眼里很值钱，而且很靠谱。

一个人具有信用思维的话，他其实就会不断地在思考一件事情，就是我能做些什么，能够快速地建立这个信用，而且我怎么能够快速地使用这个信用。而在使用这个信用的过程中间，你就会快速地运用到杠杆思维和风险思维。

所以我就说信用思维，它本质上是我们所有金融思维的一个底层的逻辑。其实一个人就可以把自己当作一个金融资产来对待。你去想，我们金融资产的目标是什么？降低未来的不确定性，使得未来的现金流增长很快，而且使得这个增长很稳定，在这个过程中，不断地去使用各种金融工具、金融技术来实现这个目标。

那你去想，一个人成长很快，然后人又很靠谱、很稳定，他在社会上不可能不是一个人才，不可能不是一个优秀的人力资本。 **当你把自己的人生当作一个金融资产去经营的时候，就很自然的会运用这些金融思维。**

所以，到根本上我们会发现，投资和人生有很多相通的地方，而金融就是在这个过程中不断地为我们赋能。 **掌握了这些金融学的基本概念，金融学的这些思维和逻辑以后，能够更好地帮助我们应对未来的不确定性。**

# 金融术与道：提升3种金融核心能力

我是香帅。今天给你串讲的是《香帅的北大金融学课》第八模块的内容，金融术与道。这个模块很轻，它一共只有6讲内容，当时我的想法就是用几根线把课程的内容全部串讲起来。

这几根线是什么呢？

# 金融的底层逻辑

 **第一条线索就是金融的底层逻辑。** 300多讲的课程里面，我们一直都在强调金融是什么？金融就是技术，就是工具。它就是帮助我们穿越时间，利用和控制未来的不确定性，替自己生活赋能的这么一套工具和技术。 **所以它的底层逻辑就是三个：时间、杠杆和风险。**

当我们谈到金融的时候，你就要牢牢地记住这三条。

第一，它是关于时间的。

第二，它是关于风险的。我经常在媒体上听到说，我们要消除金融风险的时候。这就意味着他真的没有理解金融。因为金融的核心就是风险，没有风险就不需要金融了。所以，当有人开玩笑地说要消灭金融风险，那就是消灭金融。当我们去判断一个市面上的说法对不对的时候，你就要从这个底层的逻辑出手。

第三，就是杠杆。金融是关于杠杆的学问，就是你怎么把这个杠杆用好，什么样的杠杆是最能够替人赋能，但是又不是产生风险的。我们都知道，杠杆用得好是人生的借力，你可以飞跃一大步。杠杆如果没有用好，一下子就把你的人生像一个多米诺骨牌一样推倒。所以，金融的本质就是怎么用好杠杆，好的金融就是好的杠杆。

 **时间、杠杆和风险，这是金融的三大底层逻辑，理解金融一定要从这三个要素入手。**

# 金融能力

第二条线索，就是金融能力。好多人问，我作为一个非金融专业的学生，学金融课干什么？其实我就说金融课它不是课，金融本质上是一种能力。前面我说了，它是技术，它是工具，它是为你的生活赋能的。而现代社会本质上就是金融社会，所以你缺乏这种能力，缺乏这种思考框架，你就会被这个时代碾压。

哪几种金融能力很重要呢？我们都知道，现代社会是由企业构成的。 **所以你一定要有理解企业决策的能力。** 你要知道，这个企业运行大概是什么样子的。这样就不仅是你在买股票、投资、投项目的时候有用。对于一个生活在企业里面的人，这也是非常重要的。你知道你工作的这家企业到底是怎么想的、怎么做的，这样对我们自己是很有帮助的。所以，第一种金融能力是理解企业决策的能力。

 **第二种能力叫作，金融时事能力。** 这是从课程上线的第一天，我就开始强调的能力。你会发现，所有的金融市场、金融工具和金融产品，都离不开它所在的时代背景。 **所以，如果我们不理解自己所处的这个时代，你就不能够具有真正的金融能力。**

不说别的，2017年全球的金融市场和2023年全球的金融市场，实际上已经有了很大的不同。如果你不能够理解2017年是一个全球化的态势，2023年是一个岛链化的态势，你就会对整个市场发生很多的错判。

 **第三种能力，个人资产配置能力。** 我觉得是对我们个人来讲最最重要的。你有了理解企业决策的能力，有了金融时事能力以后，就可以演化出一种新的能力。我把它叫作个人资产配置能力。

有好多人经常问，老师，要不要买股票，要不要买债券，要不要买黄金，要不要买比特币，我买房子要配置多少？其实这些问题的背后，都是资产配置能力。那你怎么样才能具有这种正确的资产配置能力呢？就是基于前两种能力，如果你能够深刻地理解，你所在的时代发生了什么变化，你理解这个时代的企业在发生什么样的变化，这个时代的资产在发生什么样的逻辑变化，你怎么可能做不好自己的资产配置决策呢？

所以整个来讲，我觉得在这么一个金融时代，所有的人都需要了解一些金融知识，然后掌握这种金融能力，才能更好地为自己和家庭赋能，更好地在一个不确定的大时代过上更确定的生活。

# 金融和金融学

第三条线索，就是大家不要把金融和金融学给搞混淆掉。我们虽然叫《香帅的北大金融学课》，但是其实这是一门讲金融的学问。你在商学院里学到的是什么？是一门关于金融学的学问。这个分野很重要的，现代金融学是一门非常年轻的学科。大概在20世纪50年代，马科维茨的投资组合理论才开始建造，就是我们讲的资产定价、公司决策、风险测度，这些非常量化、非常数学的内容，属于现代金融学。

但是，其实我们在这个课程里，你听到的更多是关于金融的内容。金融是一门非常古老的学科，从原始社会的集体主义到古代社会的婚姻制度，还有宗亲家族的关系，其实本质上都是金融关系。所以我说，金融是人类社会关系的数字化表达。金融是不断地用新工具、新技术去降低人类社会生存的不确定性，替人类社会赋能，这是一门古老的学问。

所以， **《香帅的北大金融学课》整个体系，我们是基于现代的金融学理论，但是真正想给你传达和解释的是，我们怎么样利用好金融这个技术和工具。** 你在商学院里的学习目标是学好金融这个学科。但是这个课程不一样， **它是希望你理解金融这个古老学科的逻辑，然后更好地为个体和家庭生活赋能。** 所以，这是从学问和学科的角度来理解。

# 金融视角

最后一条线索，我希望带大家从金融的视角来理解，整个人类社会的动态演化。因为我从讲这门课的第一天开始，我就一直强调， **金融是离哲学最近的学科，是跟人类文明离得最近的学科，是最柴米油盐酱醋茶的学科。** 所以你会发现，整个人类社会的演进和金融的发展是密切相连的。

在这个课程里，我给你总结了一个非常简单的公式，去理解整个人类增长的过程。在这个公式里，你就可以看到金融在其中起到了一个怎样决定性的作用。

我希望学习这个课程之后，你就真的能够像我们课程的slogan说的：站在高处，重新理解财富。因为如果你不是站在柴米油盐酱醋茶的维度，而是站在一个人类文明的角度，来重新看待我们生活中所有的金融现象，你就会发现，当你的视角不一样以后，你的视野也会完全不一样。

## 划重点

3. 金融是离哲学最近的学科，是跟人类文明离得最近的学科，是最柴米油盐酱醋茶的学科。

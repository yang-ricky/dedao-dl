# 131丨你理解的“有效市场”是对的吗？

你好，欢迎来到香帅的北大金融学课。

这一周，我们要讲一个特别有意思的事情，叫做有效市场理论。可能有同学会说，这个理论我已经很熟悉了。

那我给你讲一个故事，上世纪，美国有一家报纸做了一个实验，让大猩猩去投掷飞镖，它投中哪只就选哪只，用这个方法选了5只股票出来，组成了一个组合。

然后这个报纸的编辑们又从市场上选了8个明星分析师，让他们精心挑选了5只股票，也组成组合。一段时间以后，拿这个明星分析师和大猩猩的股票组合去对比，结果发现，大猩猩选的股票比这8个明星分析师选的要好。

这个故事想说明的就是，市场是非常非常有效的，所有的信息都已经反映在了价格中，所以所有想战胜市场的努力都是徒劳的。有效市场理论的热爱者特别喜欢使用这个故事，来证明有效市场的可靠性。

可是他们不知道一件事情，有效市场理论之父，也就是尤金·法玛，他在芝加哥上课的时候有一个著名的场景，就是前面一堂课，他用很多确凿的证据来证明市场是有效的，价格对信息的反映是特别充分的。当所有的学生都已经完全折服，相信了市场是有效的时候，他的最后一页讲义上突然出现：我所讲的一切都是错的。而最后这一句话，其实才是有效市场理论的核心。

这是什么意思呢？就是市场是不能够机械地判断有效还是无效的，如果我们把有效市场当做一个机械的概念来理解，那一定是错了。

那要怎么理解有效市场理论呢？首先我们要理解市场有效的三种模式，以及它的推论。

## 一、三种“市场有效”

第一种就叫做 *强有效市场（strong efficient market）* 。在这个市场里边，定价是完美无缺的，反映了关于企业、证券的所有信息，所以任何人的私有信息，都是没用的，都不能够战胜市场。

你看，如果市场是强有效的话，任何私有信息都不能够赚钱，那么就意味着内幕消息也不会赚钱。那市场上的一切监管，应该都是多余的了。

而这跟我们在现实生活中看到的情形是明显不吻合的。你会发现，这半个世纪以来，全球的金融监管其实是越来越严格的， *而且每次放松监管就会伴随着一次金融危机的出现* 。

在19世纪到20世纪初的时候，美国的自由市场的思潮特别兴盛 ，对监管是天生地警惕，甚至反感。整个股市非常地自由，杠杆率爱多高就多高，内幕交易、炒作比比皆是。很多人在股市上就玩的是那种空手套白狼的游戏，而这个趋势到大萧条之后才彻底地被改变，然后华尔街进入了一个强监管时代，很多人认为那个时代的美国市场其实是无效的。

即使在美国这样的市场上，大家也普遍认为，“强有效市场”是做不到的，所以它只是一个理论上的概念，现实生活里面并不存在。

第二种模式，叫 *半强有效市场（semi strong efficient market）* 。在半强有效市场中，内幕交易是可以赚钱的，所以金融监管也是需要的。但是，由于市场的价格已经反映了企业的所有基本面的信息，企业的所有公开信息都已经被包含在价格中了。

它的推论就是，市场是很难战胜的，所以一般人不可能从中间赚钱，投资者们不要贪心，获得一个大盘的平均收益就够了。

70年代的时候，半强有效市场的思潮是最浓的，当时学术圈有很多研究，都指出美国就是一个半强有效的市场，所以那个时候推出了很多指数型的基金，因为大家认为主动型的投资策略是不赚钱的。有很多证据也表明，美国的主动型基金表现很一般，扣掉手续费以后，没有替投资者赚什么钱，还不如买指数基金。

好了，那你可能会觉得，是不是金融市场就是一个半强有效的呢？其实也不一定。因为当时市场上也出现了很多反例。

比如说一个公司如果公布自己的收入，这显然属于公开信息，那么这个公开信息会迅速地进入股价中间，就不应该有很长时间的超额收益。但是现实情况是什么呢？在发布这种盈利的公告之后，如果这个公司的盈利超预期，它的股价的上涨会持续很久；如果盈利低于预期，股价的下跌也会持续很久，时间能长达三四个月。

这种现象被有效市场理论的拥护者们称为“异象”，也就是说异常现象。

但是，很多人发现，像这样的异常现象在市场上经常出现，使它显得一点都不异常了。所以很多人都提出来，半强有效市场其实在现实生活中也是不存在的。

那么紧接着就有了第三种模式，叫做 *弱有效市场（weak efficient market）* ，这种模式又比半强有效市场软化了一点点。

它也认为，某些公开信息可能没有被纳入到市场的价格里面，所以你还是有盈利机会，也是能够获得超额收益的。那这样就可以解释刚才我说的那些异象了。

但是在这个有效市场的模式底下，无论如何交易信息也是要进入市场价格的。这意味着什么呢？今天的股价和明天的股价是独立的，换句话说，股价就应该是随机漫步，历史规律是没有效的。

那么根据这个理论，可以推论出，股票操作中的技术分析是完完全全没有用处的。因为 *技术分析就是根据过去的规律来推断未来的股价走势* 。

但这个观点也受到了一定的质疑。你看，历史上有很多著名的股市高手，比如威廉·江恩，他就是技术分析的集大成者，而且赚取了极其丰厚的利润。因此很多人，特别是量化交易的高手们认为，这种弱有效市场也是不存在的。

## 二、市场是一个动态变化的过程

你会发现，从“强有效”到“半强有效”，到“弱有效”，有效市场理论也不是一成不变的，它一直在修改着自己的理论框架，试图对市场上的各种现象有更好的解释力。但是，围绕着市场有效的悖论也从来没有停止过。

为什么对这个理论会产生这么大的悖论呢？其实，是因为 *我们对有效市场的理解太过静态和太过狭隘* 。

市场上有一拨人，他们特别像程咬金，老是希望用三板斧就包打天下，用一个名词二分法就来解释整个世界。

比如说到有效市场的时候，市场要么是有效，要么就是无效，要么市场是永远不可战胜的，要么市场是可以战胜的。二者中间没有任何中间地带，好像都要分一个是非曲直出来。

其实，真正懂金融学的人谈论有效市场的时候，谈论的是 *哪些种类的信息会被怎样地反映在价格里。* 而我们都知道，信息是流动的、是动态的。现在的信息、过去的信息，今天的信息和未来的信息都是不一样的。这就意味着市场的价格也会是一个动态变化的过程。

这告诉我们什么呢？其实 *有效市场应该是一个动态概念，所谓的有效和无效是相对的* 。

比如说，如果市场有效，那么价格就应该反映价值。但你是不是考虑到价值就是动态变化的呢？

拿现实情况来讲，现在市场上有很多创业企业，它们的成长周期越来越短，不要说每年了，每个月，甚至每天的经营情况都可能发生巨大的变化，那就意味着它的价值可能在短期内都会发生巨大的变化。而这个变化是动态的、是连续的。所以企业的价值是动态变化的。而这些信息反馈到价格中也会是动态变化的。

所以，如果你谈到有效市场，谈论的只是一个静态的截面的话，那么这种理解就是缘木求鱼了。

那你要怎么正确理解这个有效市场理论呢？

* 第一，市场是动态的，所以有效市场也是一个动态的概念。

* 第二，作为一个理论，有效市场理论是有一系列的前提和推论的。而我们要做的，并不是简单地去记住这些名词，记住这些推论，而是要* **去看它的这些前提是不是发生了变化，哪些前提条件被违背了，哪些市场条件被改变了，然后去进行沙盘推演，看怎么找到在市场上挣更多钱的机会** * 。

也就是说有效市场理论是我们思考和行动的一个基准，而绝对不能够作为一个静态的、死板的概念来理解。

所以，理解有效市场，你应该学会的是 *敬畏市场，但不要迷信市场* 。你理解有效市场，但不要滥用有效市场理论。

## 划重点

1.市场有效理论有三种模式：强有效、半强有效和弱有效市场。
2.有效市场是一个学术上的理论，它主要讨论的是哪些种类的信息会被怎样地反映在价格里面。但是信息是流动的，所以市场必然是一个动态变化的过程，也正因为此，有效市场应该是一个动态概念，所谓有效和无效是相对的。
3.学会敬畏市场，但不迷信市场。

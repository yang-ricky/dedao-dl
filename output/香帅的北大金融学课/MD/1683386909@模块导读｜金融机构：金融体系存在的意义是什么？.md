# 金融机构：金融体系存在的意义是什么？

我是香帅。这里是《香帅的北大金融学课》第二模块的串讲，金融机构。

我们已经知道了金融活动要完成资金集聚、风险共担，然后还要计算资金的时间价值这些功能。所以肯定你需要很多的工具，尤其当我们人类社会变得越来越复杂之后，这种工具也会越来越复杂。这个时候，其实就需要社会分工，就是要求专门性的机构来替我们做这些专业的事情。而这些专业机构其实就是金融机构。

我们生活中非常熟悉的银行、基金、投资银行，都是属于金融机构。金融机构非常重要，因为它在整个社会体系中扮演着很重要的角色，所以这个模块的内容也比较多，一共有70讲。我们先从最常见的金融机构——银行开始说起。

# 银行

银行真的是太普通了，发工资、取工资、房贷、买理财产品，反正柴米油盐酱醋茶都跟银行有密切关系。但这其实不是银行体系的全部，在整个银行体系里面有一个大BOSS，我们平时看不见它的存在，它也没有什么储蓄所，但是它无处不在，而且跟我们的钱袋子密切相关。

我给你举个很有意思的例子，2023年的3月份，中国的股票市场、债券市场突然就雄起，为什么呢？就是因为这个大BOSS——央行，降低了存款准备金率。这样一来，市场上钱就多了，钱多了不就是钱的供给多了吗？如果需求不变的话，钱的价格是不是就下来了，利率就会往下走。利率下来大家就会有点期待，经济会慢慢变好，这种投资消费的欲望就会起来，就刺激了股票和债券的价格突然狂飙。

很多人会觉得央行、银行体系很神秘，其实我跟你讲，一点都不神秘。我一直有一个比喻，我说央行像什么呢？我们把整个社会比作一个大水库，然后这种资金不就是水吗？央行就是水库的管理员，他手里拿着什么呢？就拿着那个水闸的开关，如果他觉得这个时候田里太旱，他就把水龙头拧大一点，放一点水过去。他要是觉得市面上水太多了，这个田会涝掉，说不定他就会把这个闸门关小一点，甚至还要拿一个水管把水抽出来。要是万一哪个地方突然说起火了，需要水来灭火，那就干什么？那得赶紧找一辆消防车灌上水，然后开到那边去给它灭火。

比如说2023年3月份，美国金融市场上有个特别大的危机，硅谷银行突然倒掉了。市场上一片恐慌，出现了流动性的危机。大伙都忙着争先恐后往外逃，这时候美联储也就是美国的央行干了什么事呢？马上给它放水，而且是大量放水，就不能让这个火势蔓延下去。而且我们会发现，美联储做的还是行动非常迅即，一下子就把硅谷银行那种危机要蔓延的局势给卡断了。

怎么做的呢？首先就是原来要加息，加50个BP的，现在不加了，少加一点。更重要的是大家当时为什么恐慌？因为我把钱存在硅谷银行里，我担心这个银行倒了，我的钱血本无归对不对？美联储就说了，你不用担心，我来给你兜底，你们在里面存了多少钱，最后银行倒了，我来付。

从央行的角度看，美联储这样做一点毛病都没有，为什么？他能够印钞票，所以如果资金是水的话，央行是什么？央行可不是水的搬运工，是水的造物者。那谁是水的搬运工呢？那就是我们平时天天看到的这种银行，建设银行、工商银行、北京银行、上海银行，还有华夏银行、招商银行，这些都是商业银行。

商业银行在整个中国的金融体系中，占着绝对举足轻重的地位。有多重要呢？我举个例子，大概是在2022年年底，整个全国金融体系总资产大概是400多万亿，其中银行体系占了差不多91%，就是绝大部分。在这个91%中间，其中五大行又占了差不多36%，这意味着什么？ **这意味着银行体系对于中国的金融稳定、对于中国的经济发展具有举足轻重的地位。**

这一点就和美国的金融市场不一样，为什么呢？在美国的整个金融体系里面，商业银行体系大概就占不到30%的比重。他们主要靠什么呢？靠直接融资市场。比如说股票、债券，所以我们老是说中国是一个银行体系为核心的这么一个国家。我们会看到，中美金融体系是很不一样的。我们一个是以商业银行体系为主导，美国是资本市场为主导。所以你就会看到，各个国家的政策也不一样，比如说我们国家的很多政策你就必须要考虑银行体系的稳定与否。同样，美国的政策要考虑什么？美国一定要考虑华尔街的利益所在，为什么？那是美国的立国之本。

这也就回到了我们在上一个模块里面讲到的“橘生淮南则为橘，生于淮北则为枳”，不同的金融市场，它在不同的历史环境、制度背景下，它会有很不一样的表现。

理解了商业银行对我们国家的重要性以后，商业银行到底对我们的经济有什么作用呢？四个字，叫信用生成。比如说央行发行了货币，但是这些货币发出来以后，你要转化成信贷，这个中间靠谁？靠商业银行。中国大概有4000多家商业银行，层次还是很鲜明的，在顶层就是五大行，中、农、工、建、交，现在还加上一个邮储银行，也号称六大国有银行。跟他们的名字一样，工商银行就是为工商业服务的，建设银行为建设服务的，交通银行为交通服务的。你一听这名字就知道，这都是妥妥的国家队，实力雄厚，分支众多，不在话下。

还有一波就是股份制银行，比如说大家都很熟悉的，在股价上表现最好的招商银行，然后还有华夏银行，然后还有浦东发展银行。这些股份制银行一般也是实力比较雄厚，而且比较有特色的。比如说招商银行就是零售业务做得特别好，民生银行就是小微企业的贷款做得特别好，因为他们从纯资金实力来说，没有办法跟国有大行比，所以就一定要发展这种特色业务，然后做差异化竞争。

再紧接着下来一层叫城商行，北京银行、上海银行、南京银行、长沙银行。你一听就知道了，它是跟城市发展密切相关的银行。城商行有一个很大的特色是什么呢？因为它是地方性的银行，所以城商行的质量会跟这个地区的经济发展密切相连。北京银行、上海银行它比很多大的股份制银行还要牛，为什么？因为北京、上海的经济太好了。如果是一些经济欠发达地区的城商行，可能实力就没有那么雄厚，毕竟城商行是为城市经济服务的。

另外还有农商行，还有村镇银行，这就是更小一点的基层的商业银行，实力肯定跟前面几个不能够比。但是它会跟某个地区的经济发展或者某个特色的业务联系得比较紧。举个例子，比如说西部区域有一家农商行，做什么呢？因为西部区域有很多农业的项目，它就专门做农业的信贷也做得非常好。

这4000多家银行，我们把它们想象成一个什么呢？一直说资金是水，央行是一个巨大的水闸，中间这些大大小小的国有行、股份制银行、城商行、农商行像什么？就像这个巨大农田里面，纵横交错连通起来的水利设施。所以，如果这个水利设施修得好、效率高，就意味着央行的水放下来，对整个农田的灌溉效率就会非常高。

但是如果这个沟渠修得不好，这个水利设施年久失修了，那就会导致什么？就是放水的效率会非常低，旱的时候旱死，涝的时候涝死。这个就正是现在中国的金融体系面临的一个很大的问题。因为我们的银行体系比较多，层级也比较多，这个链条就很长，沟渠中间也不太通畅，所以整个资金流下来的时候，你会感觉效率不高，链条上的摩擦成本很高。

这里我举个例子大家可能更有体感一点，2023年市场上一直在讨论要不要降息的问题。有很多人说不要降息了，为什么？因为像市场上长期的利率LPR已经到不足4了，如果我们说这种沟渠都是通畅的，央行给的是一个4左右的利率的话，到企业手里应该多少？我们加一点点过程中间的成本，大家都在链条上有一点钱挣，企业拿到5、6应该差不多了，对不对？但实际的情况是什么呢？我们调研了很多民营企业，即使是有抵押物的民营企业，最后拿到的资金成本也在8甚至到8.5以上，就是整整比你开始时候的利率翻了一倍。这就是沟渠，这就是这种水利设施沟渠效率不高的一个集中性的体现。

所以，在课程里你也会听到，我一直在每一讲里面都会讲到， **中国需要提高金融体系的效率，它的核心就是要提高商业银行的经营和运行效率。**

实际上商业银行充当的是我们整个社会信用扩张的机制，到底哪些银行在这个机制里面会作用更大，平时我们买的那些银行理财产品到底是什么？还有各个大大小小的银行到底是怎么赚钱的？我相信，你都可以在课程里找到答案。

# 投资银行

说了商业银行，我们紧接着要说第二种很重要的金融机构了，叫投资银行。

其实投资银行就是我们国内说的券商，比如说中信证券、中金证券，还有国外大家耳熟能详的高盛、摩根士丹利，这都是著名的投资银行。投资银行是干什么的？他们和商业银行就非常不一样了。商业银行体系我们把它叫作间接融资，我这里有资金需求，我到银行去贷款，你有存款的需求，你把钱存到银行里，我们中间隔了一个第三方，这就是间接融资体系。

如果说我现在是企业，我想融资，然后你正好愿意投资，我们就到股票市场上、债券市场上，我发个股票、发个债券，你就直接买我的股票、买我的债券，这不就是直接融资市场吗？

投资银行在这里面干什么？就是在第二个市场上，帮我们来找到买家和卖家。投资银行找到哪些人想投，哪些人想融资，所以投资银行干的活就是在直接融资市场帮企业融资。那些平时听上去特别高大上的IPO、敲钟、兼并收购，实际上都是投资银行，也就是券商干的活。

因为他们这种生意大了，动辄是数亿甚至数十亿、数百亿，所以投行一直被称为全世界离钱最近的一个行业。所以你们去看，不管是美国的投行也好、中国的投行也好，一直都是各大著名院校金融系最顶尖学生最向往的这么一个就职的岗位。

我在北大光华教书的时候，我就跟同学们说，我知道你们都想去投行，可能会觉得投行这事特别金光闪闪。但是，如果回到底层逻辑的话，你就会发现其实没有那么了不起，说白了，它就是资金市场的媒婆。比如企业要上市融资，那是什么？那不就是待嫁的姑娘吗？然后你作为媒婆，作为中间人，你去把这姑娘的家底搞清，人漂不漂亮，然后把这个男方，就是要投资的这个人的家底也弄弄清楚，然后跟你来做更好的匹配。比如说双方嫁妆要多少、彩礼要多少，一般大家都知道，姑娘都是三分人才、七分打扮，对不对？所以投行要替这个姑娘进行包装。比如说给你路演，然后要联合承销，他自己的能力不够，还要联合一群媒婆把你嫁得更好，但是在这个中间也收取特别特别丰厚的佣金。所以，投资银行的本质上就是一个媒婆的生意。

兼并收购已经是现有的企业了，我想收购你，或者我想兼并你，那像什么呢？我说像二婚市场，就是把企业拆了合、合了拆。虽然说逻辑很简单，但是你想这是资金市场的媒婆，这绝对是全世界离钱最近的地方，所以投行用一个词形容，就是有钱。

关于投行，这里面有一个点特别要注意，你看投行因为离钱最近，然后它又集中了全世界最聪明的这些大脑，做的是最有钱的买卖，所以投行一直是金融创新的尖端部队。比如说20世纪五六十年代起来的这种零售经济、垃圾债券、量化交易，还有后来在2008年给全世界惹下大祸的资产证券化，这些金融创新模式都来自投行。

另外还有一点要注意的就是2008年之后，其实全球已经没有独立投行这么一个说法了。现在不管是高盛也好，大摩也好，他们都是金融控股公司。这意味着什么？这意味着，20世纪30年代一直到80、90年代，商业银行和投资银行叫分业经营的。到了20世纪90年代以后，这个间隔已经被打破了，现在大家都是金融控股公司，实施的都是一个混业经营的状态。

所以我们会看到，顶尖的投行一直在蜕变的过程中间，所以我们也不要就把投行理解成一个它就只做IPO，只做兼并收购。这样就会疑惑，怎么投行又出了一个量化基金部呢？投行的业务就一直在不停地动态变化之中。

举个特别简单的例子，比如说这几年随着科技金融、人工智能的兴起，高盛投了大量的钱在人工智能和科技金融上面，可能过几年或者过十多年以后，我们看到的高盛也好、大摩也好，包括我们自己国内的投行也好，可能已经变成了科技金融领域的领头羊。所以，真正一个好的生态，它一定是在不断的、动态持续的演进当中的。

# 基金

讲完了银行还有投行之后，还有一类金融机构就是基金，这是我们生活中特别常见的一种金融机构。

什么叫基金呢？其实就是把大家的钱集聚在一块，然后由专业人士来进行专门打理的一种资产管理的方法，就是资金集聚专业理财。所以很明显， **分散化、专业化就是基金的两大优势。**

基金有很多分类，比如说私募和公募基金，主权财富基金、对冲基金、量化基金，还有主动型基金、被动型基金，这些概念化的名词，你都可以在课程里找到答案。

我觉得大家可能更关心的是怎么选一支好基金。当我们谈到选基金的时候，其实选基金就是金融学里面很重要的话题了，你要选基金的话，你就首先得理解基金的收益来源是什么？一种就是随大流跟整个市场差不多的一个收益，这个我们把它叫作β收益。还有一种就是超过大盘的收益，这个叫α收益。

如果理解这种收益有难度的话，你就不妨想一想自己的工资，你的工资是公司的整个平均水平，那么你就是β收入。你比公司的平均收入要高很多，或者低很多，这个就叫α收入。它可能是正，也可能是负的。你选基金的时候就要注意了，比如说被动型指数，它挣的就是β收益，你就不要再期望一个更高的α收益。主动型基金就更考验人，就考验这个基金经理是不是有择时、选股的能力。比如说易方达的基金经理张坤，他旗下有好多只基金，这些就都是主动型的基金。

那什么样的主动型基金我们会觉得好呢？就是能够持续创造那种正的α收益的基金，我们就觉得那是好基金。其实真正考验我们眼力的是选那种主动型的基金。

 **但是选主动型基金，其实到最后是选基金经理。** 不说别的，大家都知道巴菲特，几十年创造了平均20%多的这么一个回报率。这已经是20世纪最厉害的一个投资大师。为什么每个人都对巴菲特这么耳熟能详？就因为这样的例子实在是太少了，因为实际上在市场上能够长红的基金经理真的是非常有限的。

就比如说2020年，美国有一个特别网红的基金经理叫“木头姐”，当时她投了好多数字货币，也投了特斯拉。所以2020年~2021年的时候，她的业绩都特别好。当时很多人都会认为说，“木头姐”可能是数字时代的巴菲特。结果呢？到了2022年“木头姐”亏得一塌糊涂，所以这就再次说明了挑选好基金、挑选真正能够长红的基金经理，真的是件非常难的事情。

最后稍微总结一下，金融机构这个模块内容很丰富，70讲的内容，但你真正要理解的核心是什么呢？就是金融机构它本质上就是金钱的搬运工，做的就是资金的更优配置的工作。

金融体系存在的意义是什么呢？金融体系它提供了一个整个社会信用收缩和扩张的机制。所以，我们不妨把金融市场看作整个现代社会的一个软性的基础设施。这跟我们看城市很像，当我们判断一个城市好不好的时候，你首先看什么？首先看的就是这个城市的基础设施好不好，尤其是路修得好不好。对于金融市场来说，金融机构就是这种路、桥，一个有效的金融机构，能够很大程度上提高整个市场的运行效率。

## 划重点

1. 金融体系存在的意义在于，它提供了一个整个社会信用收缩和扩张的机制。
2. 银行体系对于中国的金融稳定、中国的经济发展，具有举足轻重的地位。
3. 中国需要提高金融体系的效率，它的核心就是要提高商业银行的经营和运行效率。
4. 投资银行的本质就是一个媒婆的生意，主要在直接融资市场帮企业融资。

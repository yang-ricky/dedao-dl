# 1.柏拉图为什么要上街找苏格拉底？

柏拉图说过这么一句话，“最重要的是能够活在与苏格拉底同一个时代，能够与他认识。”

他觉得自己能够认识苏格拉底，是他一生最大的幸运。

为什么这么说呢？在认识苏格拉底之前，他原本是要去参加雅典年度的悲剧作品竞赛的他希望争取到桂冠诗人的头衔。但在前往剧场的路上，他听到了苏格拉底和一群人的对话。就是我们在上个模块《哲学源自惊奇》里讲到的名场面。

柏拉图听完这场对话之后，内心大为震撼，立刻下定决心，要转向哲学。因为哲学是爱智慧，人生所有的问题到最后都要归结为能发用理性做一个清楚而彻底的思考，以便掌握真理。

从此，他不再前往剧场，也不去参加悲剧竞赛了，而是转身回家把所有的文艺作品一把火烧掉。从此以后，每天只做一件事，就是上街寻找苏格拉底。

# 2.柏拉图向苏格拉底学到了什么？

那柏拉图向苏格拉底学到了什么呢？

用一句话来说，柏拉图学到了要用真诚而开放的态度追求真理，把每一个人天赋的理性，能够思考的理性充分发挥出来。具体来说，有三点。

第一点是学到了原则，就是用理性的态度，而不是神话的态度探讨真理。

第二点是学到了题材，就是找到了人生的意义这个关键问题。

第三点是学到了方法，包括反诘法、归纳法和辩证法。

苏格拉底赴死之后，柏拉图非常难过，他说“老师死了，我们都成了无父的孤儿”。他把苏格拉底当成了精神生命的父亲。

美国哲学家爱默生说，柏拉图就是哲学，哲学就是柏拉图。为什么这么说呢？柏拉图的哲学究竟厉害在哪里呢？

# 3.为什么柏拉图说大家都在洞穴中？

如果你跟着傅佩荣老师一起进入柏拉图的世界，就会发现，柏拉图对于人性有深刻的洞察。最典型的就是他的洞穴比喻。

柏拉图先描述了一个场景，说在一个洞穴里，人类都被限制了手脚，待在黑漆漆的山洞里，出不去。他们只能看到面前的洞墙，上面有一些影像。时间久了，就以为那就是全部真实的世界。

过了很久以后，有个人慢慢挣脱了绳子，一回头发现，原来先前墙上的影像，只不过是身后火炬的光，投射的影子。那些影像并不是什么真实世界，而是倒影罢了。

他不甘心就这么待在洞里，找到了洞口，于是就爬到了洞外，看到了光天化日下的世界。因为不能适应强光照射，他眼睛刺得难受。过了好久，才适应了一丁点。

他一想到还有那么多伙伴困在洞里，不知道外面的世界，就说回去把伙伴们都叫出来。可是伙伴们却认为他不诚实，把他毒打一顿，甚至杀害了。

之所以带你详细回顾了洞穴理论，是因为，它的警示意义和哲学意义都太重要了。

所谓警示意义就是，寻求真理是需要付出惨痛的代价的。比如柏拉图的老师苏格拉底。

所谓哲学意义就是，柏拉图希望通过洞穴比喻，对当时流行的辩士学派提出批判。辨士学派主张，把知识当作知觉。

而在柏拉图看来，这样的认识，给人们带来的就是洞穴上的倒影，并不是什么真理。在柏拉图看来，感官所掌握的现象世界是不可靠的，我们要继续探求真实的东西，提升到感官以及感官的对象之上。

# 4.柏拉图对人性的思考，和孟子的性善论有什么不一样？

更进一步，我们还可以从柏拉图洞穴比喻的背后，看出柏拉图对人性的思考。其实，哲学家们对哲学的思考，最终都会落到他们对人的思考上。

而在柏拉图这里，他的看法是，没有人是心甘情愿是行善的，没有人是以行善为乐的。他认为，一个人行善，是为了行善而行善，或者是为了某种外在的名声，或是有形的报酬而行善。这个认识非常深刻。

所以在晚年的时候，柏拉图特别强调法律的重要。毕竟人性是需要被约束的。

你会发现，这和我们非常熟悉的孟子的性善论很不一样。

# 5.柏拉图是如何思考教育的？

再来看柏拉图对于教育的思考。

柏拉图的教育观念，依托于他的人性论。正因为人性并不完美，所以必须要接受教育。注意这里，其实柏拉图是把知识直接等同于德性的。随着你深入学习这门西方哲学课，你会发现，这会是后来哲学家们讨论的一个经典命题。

在柏拉图这里，你记住两句话：

第一句，受教育就是让你得到真正的知识，了解人生的幸福何在。

第二句，幸福的人生，不能在财富、名声，各种装饰品里面寻找，一定要转向内在，在自己的心灵中寻找。

你可能会误以为，这和市面上流行的心灵鸡汤讲的差不多。其实柏拉图这么讲，是基于他对于普遍意义上的人的命运的思考。柏拉图是古希腊人。古希腊是城邦的社会，个人生命与城邦的存在跟发展有密切的关系。他是在整个国家、整个社会意义给出了自己的解决方案。

# 6.柏拉图是如何设计他心目中的理想国的？

前面我们复习了柏拉图对于人性的思考，接下来我们再来看柏拉图对于现实政治的思考。你学习完第二模块，想必已经发现了，柏拉图对于理想国的设计，在某种程度上，可以说是后来各种国家组织形式的源头。

回顾柏拉图心目中的理想国，你只要记住三句话就好了：

第一句是柏拉图的核心观念，他认为，政治组织是基于人与人的互相需要。每个人的天赋、才干不同，所以大家要做专业的分工，这样效率更高。

第二句就是他对于理想城邦的划分了，他认为应该分三个阶级：统治者、卫士和一般市民。城邦对这三种不同的阶级，各有什么要求呢？那就是四枢德：明智、勇敢、节制和正义。

最后一句，柏拉图还总结了五种可能的政体，分别是：贵族政体、名誉政体、寡头政体、民主政体以及暴君政体。

你掌握了这些，就理解了柏拉图对个人与国家关系的思考了。

# 7.柏拉图是如何讨论灵魂的？

复习了柏拉图最基础，也是最核心的哲学思想后，接下来，我们再深入回顾一下柏拉图另外几个重要的哲学思考。首先就是很多同学在课程留言里讨论的灵魂观。

我相信只要是学习过这门西方哲学课的同学，一定是多多少少就提前思考过这个问题的。恭喜你，思考这个问题，意味着你早就进入了哲学里的最核心的命题“终极”。这其实是我们每个人，终其一生都无法回避的问题，至于什么时候碰到，那只是早晚而已。

两千多年前的柏拉图，又是如何思考这个问题的呢？我们一起回顾下。

你记住这句话，柏拉图主张灵魂不死。

柏拉图的灵魂观是受到我们以前介绍过毕达哥拉斯学派的影响，就是把灵魂当作生命的原理，它是不死的，而身体是灵魂的监狱。有句话你可能早就听说了，就是柏拉图说的“哲学就是练习死亡”。

那柏拉图是如何证明灵魂不死的呢？你记住这五句话：

第一句，灵魂是有轮回的。现在活着的灵魂，是过去已死之人的灵魂。因为所有事物都是从它的反面产生出来的。

第二句，知识的普遍性需要灵魂不死。这是从柏拉图的“知识回忆说”里推导而来的。柏拉图提倡人的知识具有普遍性，注意这个前提很重要。因为如果你不是在前生已经知道的话，那知识就没有普遍性了。

第三句，灵魂不死还是由柏拉图的理型论推导出来的。理型就是理性所掌握的原始模型。理型在某种程度上是所有知识的本体，想要掌握理型，灵魂必须十分接近它才成。既然理型具有普遍性，那灵魂也必须不死。

第四句，柏拉图认为，人的身体的任何动作都需要灵魂发号施令，所以灵魂本身是一个自动的原理，它必须不死。

第五句，内在的一种恶的因素并没有使灵魂死亡，那是因为灵魂在根本上就是不死的。

如果你仔细去思辨柏拉图的论证，相信你一定可以找到不少逻辑上的漏洞。我在同学们的课程留言中，也看到了不少精彩的分析。

那为什么我们还要学习柏拉图的灵魂观呢？这就要说到柏拉图理论的提出背景了。古代的希腊人，他们并不关心天堂或地狱，他们关心的是人现在的生命，在现在的生命里灵魂是主动的、完整的，可以融合可知之物与自然之物。只有灵魂可以领悟理型，引发和谐而有韵律的活动，以及引发善的生命。这才是柏拉图灵魂观的意义。

# 8.什么是柏拉图所说的“理型”？

刚刚复习了柏拉图的灵魂观，你也一起回顾了柏拉图非常有名的一个哲学概念“理型”。

先回忆一下柏拉图“理型论”的思想传承。在《哲学源自惊奇》模块，我们讲到，巴门尼德让柏拉图知道了思想的重要。而老师苏格拉底致力于探索人生的真善美，决定了柏拉图的“理型论”的大致的方向。柏拉图的理型论，其实是要用理性走上真理之路。

柏拉图是这么论证他的重要哲学思想“理型”的。

他认为， 人类认识世界的时候基本上有两个方法：一个是靠感官知觉，一个是靠我们的理性认知的能力。但是，靠感觉的话，我们面对的是许多变化生灭的个体，这是不可靠的。

只有靠人的理智，我们才能掌握到认识对象的形式，知道这样东西之所以归类为这样东西，它的本质到底在什么地方。换句话说，只有理智可以掌握到真正的知识，理智的对象就是那个永远存在的理型。

所以柏拉图说了，我们是发现而不是发明理型。另外，柏拉图谈到理型的时候，他真正关怀的理型按照先后的顺序，排在前面的才真的重要。第一种是价值方面，就是在伦理学与美学方面的理型；第二种是数学方面的理型；第三种是概括的理型。

你掌握了这些，基本上就掌握了理型论的核心观点，也借助柏拉图了解了人类对于普遍知识的探索。

# 9.到底如何理解“柏拉图的恋爱”？

关于柏拉图，我们最后还要再回顾下非常有名的“柏拉图的恋爱”。我相信你在学习这门哲学课以前，就经常听人说起过。

既然已经学过了哲学，我们就不能还只是停留在一知半解的层面。那我们到底要如何理解“柏拉图的恋爱”呢？

柏拉图认为，当我们爱一个人的时候呢，我们不能只爱他的身体，因为人的身体会衰老，会生病，最后还会结束。那我们要爱什么呢？要爱他的理型，简单来说，他的灵魂。

注意，这和柏拉图对“美”的认识是分开的。柏拉图认为美在本性上是永恒存在的，不生不灭，不增不减。你看，在柏拉图这里，美跟真、跟善已经合一了。所以，爱美就是爱智慧，爱那个永恒不变的可贵的东西。

所以一句话，所谓柏拉图的恋爱，就是哲学家的爱。它不只爱一个人的灵魂，还要爱一个人的理型。

# 10.亚里士多德为什么认为老师柏拉图的学说有问题？

你一定非常熟悉亚里士多德那句名言“吾爱吾师，吾更爱真理”。

没错，亚里士多德对他的老师柏拉图非常的推崇。他高度评价自己的老师柏拉图，说我们老师用他的言行表现向人间证明了一点，就是凡是有德行的人才有真正的快乐。

那为什么亚里士多德又要说“吾更爱真理”呢？

因为他们两个人在建构哲学系统的方向上不一样。虽说最后两个人殊途同归，都是想确立人生的最高目标，也想把生命的意义当做爱智者的核心的关怀。

但亚里士多德不同意柏拉图的理型论。

他说，不是像老师柏拉图说的那样，我们见到的个体都是分享了一个完美的原始模型。他说这都是老师想象出来的，其实并没有这回事。

亚里士多德分野，导致了西方哲学后来的二分法。西方哲学认为，一个人的心不属于亚里士多德，则一定属于柏拉图，好像只有在两者之间做个选择。

如果你选择柏拉图的话，代表你重视理性，重视理想，偏重人的心灵，向往永恒的这一面。

如果你选择亚里士多德的话，代表你重视经验的世界，想要对于变化有所认识，知道这个世界还是有一个发展的方向，但不能忽略具体的状况。

这也是为什么这么模块，傅老师要命名为《哲学双重系统》。

# 11.为什么说亚里士多德是帝王师？

首先当然是因为亚里士多德教过著名的马其顿国王亚历山大大帝。亚历山大大帝承认亚里士多德是他心灵上的父亲，对他非常的尊敬。

所以亚历山大大帝后来在很多地方都帮助了老师亚里士多德。

你知道的，研究学问需要庞大的资金与人力。亚历山大大帝给亚里士多德大量的资金，给他上千名奴隶，让他使用。大帝远征的时候还特别派一支特遣队，专门把找到最新奇的植物与动物的标本往后送，送到亚里士多德的研究基地。

后来亚里士多德果然发展出各种当时难以想象的学问，还开创了在后代影响重大的各种学科。

所以自然地，亚里士多德在政治上有很多的思考。你重点记住他的两个观点：

第一点是，人不能脱离政治，因为人需要过群体的生活。

第二点是，人虽然不能脱离政治，但不要对政治寄望太多。

这也是为什么亚里士多德要建构自己的学院，从事教育工作。等你进入到课程后面的模块，就会发现，亚里士多德对后来西方的影响非常大，可以说几乎没有哪个学科可以绕开他。

# 12.什么是合乎逻辑的思考？

和柏拉图不一样，亚里士多德肯定的是经验世界。

那如果我们肯定经验世界的话，就要有正确的思考方法，才能够掌握经验世界是怎么回事。这个正确的思考方法就是逻辑。

学习哲学，这个知识点一定要多次回顾，反复复习。

“逻辑”这个词，我们以前谈过，它来自于逻各斯。逻各斯是古希腊哲学家赫拉克利特所强调的，认为宇宙里面有一个主导一切变化的规律。这个词变成逻辑，成为一门学问，所讲的正是正确思考的方法以及它的规则。如果你要驳斥别人的诡辩或是谬论，你本身思考的规则要非常的明确，这样才能够说服别人。

这也是对你日常生活和工作帮助非常大的思维方法。

掌握好逻辑的思考方法，你重点记住这三点：

第一点，我们先定义我们所使用的概念。尤其是在跟别人讨论问题的时候，要形成自己的肌肉记忆。如果别人说的话不清楚，就要请他定义他所使用的概念，这样双方可以进行沟通以及有效的讨论，再进一步得到共同的结论。

第二点，就有范畴的思维。我们在思考任何事情发生的时候，就要问它是不是能够涵盖所有相关的范畴。范畴涵盖得越多，就把事情看得越完整、越仔细。这些范畴用得越完整，对一个人或一件事的了解就越透彻。

第三点，就是推论。我们进行推论的时候，可以使用三段论法，从大前提、小前提再推论到新的结论。

# 13.如何了解复杂的万物？

我们都知道，亚里士多德的博学几乎可以说是前无古人，后无来者。那亚里士多德是如何了解这么多复杂的万物的呢？

这就要说到亚里士多德认识事物的方法“四因说”了，也就是：质料因、形式因、动力因和目的因。亚里士多德认为，我们理解万物，从这四个原因入手，就可以了。

质料因说的是材料。任何东西总要有它的材料，要是没有材料，我们就不能分别这个事物和那个事物，这很好理解。

形式因说的是，我们在具体了解到某一种东西的时候，如何判断它就是它，而不是其它。那个决定“它”就是“它”的东西是形式。

还是说这个东西，那它是怎么造成的呢？可能是吸引力，也可能是排斥力，又或者是其它动力，这就是动力因。

目的因是亚里士多德自己的创建。他认为，任何东西的存在，它一定有一个目的。当然有的时候，这个目的也会比较夸张。不要觉得亚里士多德这个说法离我们很远，实际上，你回忆一下，我们人类可以说天生就是会用目的论来思考问题。

这四因，就是亚里士多德所综合的古希腊时代对于变化的世界的万物所做的了解。

当然，如果你觉得还是复杂了，那我们还可以用质料和形式作为事物根本的范畴，把四因简化为二因，用来说明宇宙万物。

# 14.事物是如何变化的？

我们前面说过亚里士多德，他以“质料”与“形式”这两种原因说明一样东西的存在，它基本的结构是怎么回事。

除此之外，亚里士多德还用“潜能”与“实现”这两个词来说明一样东西变化的过程。我们可以从两层论证来把握。

第一层是，凡是变化都是由潜能走向实现的过程。

打比方来说，几块木头不会自己变成桌子，它需要有人作为动力因，把它做成一张饭桌或书桌，这就是从潜能到实现的说法。

第二层是，潜能因为“缺乏”走向实现，由实现才可以理解潜能。

这里的“缺乏”，不是一般的字，它是哲学上的术语。所谓“缺乏”就是这个潜能本身它缺乏实现，它自然就会往“实现”去发展，这叫做缺乏。

我们知道了从潜能到实现的架构，就知道了实现优于潜能。好比说，我们有个目标去实现，那现在就会朝目标去努力，培养自己更好的条件。

最后再复习一组具有思辨性的概念：“时间上的先后”和“逻辑上的先后”。你可以把它作为自己思维上的训练。

所谓逻辑上的先后所说的是如何理解一样东西；而时间上的先后，纯粹是就过去—现在—未来的直线发展而已。

如果没有逻辑上的先后的了解，我们只是在时间的变化过程里面，根本不知道自己要走向何方。这组概念很抽象，你理解到这个层面就可以了。

# 15.如何理解上帝和哲学的关系？

我们中国人学习西方哲学，最大的难题之一就是如何理解上帝。我们来回顾一下亚里士多德是如何讨论的。提醒你注意，复习这个知识点，会帮助你更好地理解后面中世纪的哲学思想。

首先一点，亚里士多德这里的上帝，和宗教没有关系。上帝是怎么来的呢？是亚里士多德推导出来的。

前面复习过，亚里士多德强调变化的世界。那如果世界充满变化的话，我们就要问了，变化的力量从何而来，变化有什么目的？亚里士多德的答案是，上帝。这就是他从这个地方推出来所谓的“哲学家的上帝”。

在亚里士多德这里，上帝是思想的主体，也是思想的客体，他就是思想的本身，所以称他为“思想之思想”，这是西方对于亚里士多德的“上帝”很明确的认识。

那上帝和我们人有什么关系呢？

第一点，这个上帝不会回报我们对他的爱，这就和宗教中的上帝就完全不同。一般人以为说，我如果信仰一个宗教，我爱里面的神和佛，它也会照顾我，也会爱我。而亚里士多德的“上帝”完全无动于衷，不受人所影响。 你想想看，如果你爱上帝，上帝要回报你，他不是在行动吗？

还有第二点，人类在任何情况下，更不能说我们爱上帝。换句话说，上帝对这个世界没有任何监督、管理、安排的愿望，也没有任何可能的行动。

所以这样的上帝已经完美了，他已经是完美的形式和实现，不需要回应任何东西了。

# 16.到底什么是形而上学？

理解了什么是哲学家的上帝，接下来我们要复习一个很重要的哲学概念形而上学。

必须要说，学习了哲学，一定要弄清楚形而上学到底是什么意思。我们来重点复习一下。

首先我们要理解它的中文意思。

“形而上”三个字来自于我们中国的传统经典《易经》。《易经》的<系辞传>里面特别提到“形而上者谓之道，形而下者谓之器”。

所谓“形而下”，就是落在形体里面、有形可见的东西就称为器物。像这是一张桌子，桌子就是形而下的、有形可见的器。

对应地，“形而上”就是我们看不到的、所谓的桌子的原理是什么。如果没有桌子的原理，我们就造不出来桌子。对照着“形而下”，你就能知道什么是“形而下”了。

再来说形而上学这个字的英文meta-physics，就是指放在自然学后面的那门学问。亚里士多德认为，研究无形可见、永不变化的本体。到这里，你一定要牢牢记住了。

那如何了解形而上学？

当然是非常复杂非常深奥的。好在傅老师给我们做了三层抽象。

第一层叫做物理抽象，就是把一个事物的物理特性、个别的东西去掉。你学习了物理学，就比较清楚了。

第二层是数学抽象，就是把事物所有的形状都可以抽出来，然后加以运算。它有几个角，每个角多少度，三内角的和是多少，这都是数学抽象的结果。

第三层就是形而上的抽象了。亚里士多德谈到形而上学的时候，说“把存在之物当作存在之物来看”，意思是，不要看它是不是苹果，是不是圆形的东西，不要看那些，而只看它是不是存在。

学了形而上学之后，我们对许多事情就要有客观的心态，要学会去探寻根源，而不要只问我学这个有什么用，立刻就要找它的用处。

实际上，你可能有这样的体验，很多东西在最后才发现，它是契合于一样东西的根本、一样东西的本体的。

很多人说，西方的哲学看起来可能是无用之学，但实际上，这种无用之学与人生要追求最后的幸福、追求理性最高的满足是有直接关系的。

# 17.如何理解“知”与“行”？

这就要说到了亚里士多德的“知行观念”了。

我们在谈亚里士多德的形而上学的时候，知道了是我们的理性想要求真，想要知道最后的永恒不变的一个本体。而人生除了求真之外，还要行善，还要审美。

亚里士多德和大部分的古希腊人不一样，他特别强调中庸的路线，也就是用理性来掌握情绪和欲望。他说，我们在任何抉择的时候都不要过，也不要不及，而是要找到适中的、适当的方式，然后不断地往上提升，追求卓越，这样才能够真正地修炼自己。

那体现在“知”与“行”上，他认为杰出的“知”与杰出的“行”不太一样。他本人不反对我们拥有某些必要的生活条件，包括基本生活需要的财力，包括房子住所这些，还包括一些好的朋友。所以他认为哲学家不能够离世而独立，而是要把这些都放在追求德行的考虑里面。

# 18.学哲学的目的究竟是什么？

从亚里士多德这里，你会发现，我们完全可以学习跟亚里士多德一样进行日常的生活、学习、工作、事业、成家立业等等，在社会上有各种相关的活动。

但与此同时，我们不能忽略学习的重要，因为学习的目的就是要让理性的潜能充分发挥。

我们还可以从亚里士多德这里得到的启发是，如果我们要追求人生的幸福，那就应该把我们学到的观点，和我们本身的思想系统保持一致。

当然，我们一般人不一定有什么完整的思想系统，但至少要有个出发点，就是从"人是有理性的动物"出发，用理性来调节我们的感情和我们的意志。

人生观的可贵不在于谁说得对，而在于我们是否能形成一个系统，用言行来证明自己的观点有它的价值。我们在不同的年龄、不同的生活阶段，可能会有不同的人生观，这也是所谓的形而上学的自然的倾向，每一个人都有，我们不必勉强自己立刻接受这几位大哲学家的观点，因为那需要我们自己的心得与体会。

很多同学在课程留言里都提到了这个问题，相信你复习到这里，能够找到属于你的答案。

最后，让我们用亚里士多德的幸福理论，为《哲学双重系统》这个模块的复习作为结尾。

亚里士多德认为，许多事情我们努力奋斗也不见得会有结果，但人生最高的幸福在于观想，只有观想能够让我们进入真正的悠闲。

有了这样的观想，我们就会知道，哲学不牵涉到我们在现实世界上成功失败的问题，而只牵涉到我们自己本身有没有适当的处世态度。

最后，祝你学习愉快。我们下个模块的复习再见。

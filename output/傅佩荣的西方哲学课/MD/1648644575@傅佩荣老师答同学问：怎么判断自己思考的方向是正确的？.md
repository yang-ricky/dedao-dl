# 傅佩荣老师答同学问：怎么判断自己思考的方向是正确的？

你好，欢迎来到《傅佩荣的西方哲学课》。

收到了你学习哲学课过程中遇到的问题，我集中为你做一次解答，这次总共有五个问题。

# 01 如何确认存在是否存在，“我”是不是真实的？

我们把两个同学提的问题，合在了一起。

第一位是@乌麦天同学，他是在课程里提出来的。另一位是@猛火同学，他在直播课的时候就提出来了。

首先@乌麦天同学说，他以前听过我讲老庄的课。我在讲到庄周梦蝶的时候，说，庄子说“我”跟蝴蝶一定有所不同。庄子说的是“真实”，那么“道”就是究竟的真实。

> “昔者庄周梦为胡蝶，栩栩然胡蝶也，自喻适志与！不知周也。俄然觉，则蘧蘧然周也。不知周之梦为胡蝶与，胡蝶之梦为周与？周与胡蝶，则必有分矣。此之谓物化。”（战国·庄周《庄子·齐物论》）

同时他也听我提到孔子说的，不要在空的市场上找马，这里说的是“真诚”。

> “彼已尽矣，而女求之以为有，是求马于唐肆也。”（西晋·郭象注：“唐肆，非停马处也。”）（战国·庄周《庄子·田子方》）

事实上庄子说“他跟蝴蝶有所不同”，以及孔子说“不要在空的市场上找马”，这两段资料都来自于庄子，他就综合一下孔子说的真诚，就是老庄所说的真实。

第二方面，他还听到我谈“遭遇”这个词，遭遇就是碰触到的真实。他也有过“遭遇”的经验。有一次看到一株小蓬草，一刹那被它的美惊呆了，非常感动。他就问， **一个人遭遇到美好的时候，那份激动莫名的美妙情绪，是不是就是真实的呢？**

好，综合起来两句话。第一，庄子说，我跟蝴蝶一定有所不同。这是“真实”的吗？第二，孔子说不要在空的市场上找马，这是指“真诚”吗？

很大的问题，也是很好的问题。我简单这样回答。

 **首先哲学上是没有什么标准答案的，也就是要注意三点。**

 **第一，哲学没有对错，但还是有高低之分。** 怎么分呢？看你在思考的时候，是不是比较完整，也比较根本。越完整地注意到生命的每一方面的，越根本的注意到生命最根源的问题的，那就是比较高的。

 **其次，在思考方法上可以得到一些借镜。**

 **还有第三点，除了思考，还有每一个人的生活体验。** 生活体验是开放的，所以哲学的问题永远是开放的，等待你自己去探讨，去觉悟，去体会。

所以针对@乌麦天同学的问题，我们可以这样说。

首先庄子为什么说我跟蝴蝶一定有区分呢？是因为人跟蝴蝶都属于万物之一，万物都是“气”的变化，所以是相通的。但是人是万物里面最特别的，因为它有可能悟道，所以它跟其他的万物都不一样。

 **“道”是究竟的真实，人是相对的真实。你只有找到道的时候，才能像一滴水回到大海一样，完全融入道里面。找到究竟真实，可以安顿自己。**

那么孔子说在空的市场上找马是什么意思呢？是说， **每一个人都在特定的时空里面，有一定的历史背景跟社会现实，那你就要突破这种特别的条件所构成的现象，这样才能看出一个人的本质。**

换句话说，孔子的言行表现在特定的状况之下，那是真的孔子吗？可能时过境迁之后，你发现，好像不是这么一回事。

他整个谈话的背景，是孔子提醒颜渊说，你不要就我的现象来把握我的本质，从现象把握本质这个本来是正确的途径。人不可能脱离现象的世界。

但是你要特别注意到， **本质毕竟跟现象不完全一样，本质可以不受任何时空的限制。**

所以在这里面我们做个结论。 **道家所说的真实，儒家所说的真诚是相通的。** 为什么？因为以儒家来说，人只能够做到真诚，但是人的真诚也就是人的真实，这一点在道家里面是相通的。

那么请问我的存在是否存在呢？我是不是真实的我呢？

当你在思考这个问题的时候，有所觉悟，答案就是肯定的啦，你说哪一个我是真实的我？那当然是我在这一刹那，没有任何利害的考虑。那么我的真实，就可以面对@乌麦天同学所说的那株小草。

 **就等于是没有利害考虑的时候，我把自己完全开放，这时候才可能直接“遭遇”。在道里面，这些“遭遇”跟万物就可以相通，甚至合二为一。**

另外@猛火同学他说，我有时候觉得自己喜欢运动，但是又不知道这种喜欢是不是就是真的喜欢？这种状态已经很长一段时间了，头都想破了。

换个方式说，就是我所认为的“我”是不是真的“我”？怎么去判断我当下的状态，是我最本我真我的意思，而不是自我欺骗？我怎么去缓解我的这种自我怀疑呢？

我们继续再来回答。那么猛火同学的问题，最关键的就是“自我欺骗”这个词。

首先，当你有自己的利害考虑。比如说运动可以健身，我是为了健身才去运动。

或者第二方面，我从别人的角度来做判断，比如别人提醒我我受别人影响，但是并没有自觉。我去运动健身可以得到别人的肯定，结果让我以为自己喜欢这种运动。

这两种都要排除， **第一个不要有厉害的考虑，第二个减少别人的影响。这两点去掉之后我就可能真正属于我自己。**

所以这是存在主义里面一个很好的话题。像法国的萨特，就特别提醒人，大多数时候都在自我欺骗的状态里面。那么你要设法回到自己，也就是说要属于自己。

 *所谓的真诚，就是我属于我自己，只对我自己负责，不去考虑任何利害关系，也不去考虑别人的观点如何。所以你真的喜欢运动的时候，在里面你可以自得其乐，就不用太担心了，那就是你真正的自我的意识，那跟自我欺骗就谈不上关系。*

所以这两方面，你有一种了解之后，就不会有太大的问题。

这是对于第一点的我们所做的回答。谢谢。

# 02 如果没有别人，我这个人还会不会存在呢？

这是 @Helios 同学的留言。他的意思是，一个人他不懂得动物学，看到熊猫就说，这是动物。懂一点动物学的人会说，这是熊科动物，或者是哺乳动物。这么看来，熊猫其实也是动物的“质料”。

所以有没有绝对的形式呢？

他这里提到质料跟形式这一组概念。

接着他继续问，那么人的形式是什么？他说他觉得是别人的印象。一个人是人，是因为我们会说，能思考就是人，会生活的就是人。

可如果要认识到“一个人是张三”，就各有各的说法了。有的人觉得他是八荣八耻里面八荣占据最多的那个人，或者说他是从小就无耻龌龊的那个人。

但这些“形式”，都是张三的侧面，可能张三自己都不知道。

所以 **人的形式来自于另外一个人的印象，而这个印象可能停留在某个时间下的某个空间。那么如果没有别人，我这个人还会不会存在？**

这是很好的问题。但这里有两组概念，先要设法厘清。

第一组就是质料与形式。 **任何东西的存在，有质料跟形式。质料就是问，有东西吗？形式就是问，是什么东西？**

譬如你看到一个东西过来了，知道它不是虚无。再一看是熊猫，熊猫就是“形式”。形式，就是可以让你分辨这是什么东西的条件，称为形式。质料就是让你分辨有东西而不是虚无，那就是质料。所以任何东西的存在都要形式跟质料配合。

第二组概念是什么呢？就是本质跟存在。谈到本质，它跟形式有关，就是分辨它是一种什么样的东西，它的本质是什么。所以本质是我们认识的对象，比如今天并没有恐龙真正的存在，但是你看过《侏罗纪公园》就知道恐龙的本质是什么，它之所以为它，有什么样的条件，这叫做本质。

存在的话，就要问在具体的时空里面，它真的有占据一个存在的位置吗？

那我们接着就要问，人的形式是什么？  Helios同学说是他人的印象。

这个说法是有问题的。他举张三做例子，说他是好人或者说他是坏人，但是，那都是其他人的印象。这个印象得自于特定的时空里面所出现的“张三”。

问题来了，如果没有别人的话，这世界上只有我一个人，那么我这个人还会不会存在呢？这是非常具有哲学性的问题。

我们可以这样说， **一个人的形式，既不是这个他说的“会生火的”或者“别人的印象”，而是什么？而是理性。** 因为人跟其他动物的差别，在于人有理性可以思考，所以他说“能思考的就是人”这是对的。所以每个人都一样，人有理性。

那么形式确定的话，同一个人可能有胖有瘦的可能，这是质料。

一个人的形式是什么？是别人的印象吗？当然不是，因为你说别人的印象那是谁呢？这别人谁？是一个人还是很多人？

因为所有人看到的张三，都是在特定情况下，他的言行表现，而这个特定情况随时可能改变。

所以你只看别人的印象，那是常常在变化中的。

所以为什么西方近代笛卡尔会说我思故我在。换句话说，我的存在，不可能由“我思”之外的其他人，或其他的力量来决定。

“我思”是什么意思呢？就是我要问，“我”真的存在吗？当我这样问的时候，“我”必须存在，我才能问这个问题。这就是“我思故我在”。

所以，“我思故我在”的我，是一个能够思想的“我”。能够思考的我，并不是天下人对我的印象。 *所以天下人对我的印象，跟我的本质没有必然关系。* 这是为什么我们会说知己难求？因为很难找到知己。甚至有时候我都不清楚自己是谁，因为人是会变的。

 **但是我也可能在某一种特定情况下显示真正的我，那就是关键的时刻，比如生死关头。**

所以一个人他有可能觉，有可能悟，觉悟的时候，那个本我，真正的“我”就出现了。

所以你先不要着急。所以在这里我们要强调的是， **我这个人我的存在跟其他人没有直接的关系，但是我是一个人，不能脱离人类的世界。而在跟其他人互动的过程中，可能有某些片刻，我真正显示了自己的根本的终极的关怀。** 比如在生死关头，那么在某些状况，我会觉得要说出内心里面真正的感受，那一刹那，我就是属于我自己，我才是真正的存在。

所以这个问题是可以做这样的思考。

# 03 怎么判断自己思考的方向是正确的？

提出问题的是@雪地里仰望繁星这位同学。他说，所有的问题归根结底都是哲学问题，只是哲学化的程度有高有低，每天思考的越多，越是好事。

这时候他想到一个问题， **就是怎么判断自己思考的方向是正确的？思考的方向有没有绝对正确的呢？** 或者很多观点都是相对正确的。

这问题非常好。

所有的问题在思考的过程里面都只有相对的正确性，也就是所谓人间只有相对真理，因为一切都在变化之中。你所观察的对象，人事物，甚至在观察的，你自己在思考的你，都在变化中去。到最后你能把握的只有认知的主体，也就是自我。

那么我之外的一切都可能是认知的对象。

 *所以在这个时候，你只要知道，自己的思考，从方向到它内涵，都是相对的，这样你就更接近正确，更接近真实。*

如果你以为自己思考的方向是绝对正确的，这一来反而变成相对的。你无法证明你是绝对正确的。

所以我们在人间跟别人互动的时候，要怎么去掌握到所谓的思想方向是否正确呢？

 **第一个，不要执着，先不要认为自己一定是对的。** 因为你只能看到你所看的这一面，甚至你会去选择你想要看的这一面，而不可能看到说一个人或一件事情的全貌。

所以 **第二点就是要宽容，要尊重跟你不同意见的人。** 你的想法对同一件事情是如此，你不能够阻止别人有不同的想法。

 **第三点，就可以用庄子的一句话就是用心若镜，使用你的心思要像镜子一样。** 镜子它只照见来照的东西，它不会欢迎，也不会欢送。什么东西来的，就照见它的真相。没有个人主观上喜不喜欢，愿不愿意的问题。这样就可以欣赏一切。

> “无为名尸，无为谋府，无为事任，无为知主。体尽无穷，而游无朕。尽其所受乎天而无见得，亦虚而已！至人之用心若镜，不将不逆，应而不藏，故能胜物而不伤。”（庄周《庄子·内篇·应帝王》）

所以道家的思想，我们常常强调，他对于善恶不太愿意多谈，因为善恶是人间所定的某些标准，所做的相对判断。道家要强调，从真实到审美，只要存在就属于真实，而绝对的真实是道。

所以的一切都在道里面，道也是无所不在的。

你如果体会到一切都在道里面，掌握到真实，就可以进而欣赏一切，也就是产生一种审美的感受。

所以你怎么判断自己思考的方向是正确的呢？ *你要先知道自己不可能绝对的正确，只有相对的正确，就容易尊重其他人的想法。* 这个问题可以做这样的思考。

# 04 为什么东西方思想中总会出现一些共通的概念？

这是@奕（yì）麈（chén）富同学在上一次直播课所提出的问题。他说他有些问题百思不得其解。西方在启蒙运动之后已经出现了很多哲学家，而科学革命也已经开始了，但怎么当时就没有推翻像“死后的审判”，“天堂”，“地狱”这些神学的观念呢？同时这么凑巧，东西方都有这样共通的说法，比如东方宗教里面也谈到的地域，轮回等等，难道这些都是出于人的天性吗？

这问题很好。而答案就是他所说的是人类天性有这些问题，所以在宗教里面就会提出某些回答，他们的回答。无法得到客观上的证实，但是并不因而就没有什么意义。

 *说到意义这两个字，这是重点。人有理性，有理性就要求理解，否则就谈不上意义了。你能够合理说明一些现象，这些现象就有意义。*

那么人生呢？人有自由可以选择可以行动，所以就有应该不应该的问题。 **先不要问应该不应该怎么来的。你有自由可以选择，就要对后果负责，所以就有善跟恶的结果，善要有赏，恶要有罚，这才是合理的正义。**

否则的话，为什么你有自由呢？有自由之后让你自由行动，而对于行动的后果又没有什么适当的报应，那这个自由不是很难理解吗？

所以理解人的自由，就要知道，最后一定要有善恶的报应。

所以在这里，可以归结为三点：第一个，人到最后一定会死亡，生命会有结束，结束之后一定要有审判。对于你这一生所谓的善恶的行为，有一种适当的报应。那审判的时候就分两种可能，第一种就是西方所谓的上帝，他全知全能全善，所以可以有一个公平的报应。

第二种就是东方宗教，谈到轮回，三世因果，六道轮回，把格局拉得更开。但是哪个对，哪个错？都是教义，只有信不信的问题，没有对错的问题。

那么第三点，谈到赏罚这样的观念，是把人间的一种状态去投射，去比拟死后的一种情况，所以这就是对这个问题所做的回答。

那么请问前面所说的神学的观念，这些是人类天性吗？

没错，人类天生有思考的能力，就希望能够去理解人生所有的问题。但是理解不可能的，就需要有宗教信仰来加以说明。

所以这样去了解的话，就知道是怎么回事了。而宗教只有信不信的问题，没有什么对错真假的问题。

# 05 儒学没有追问天道，没有追问“我”，那要怎么样保证道德的绝对律令呢？

这是@梅同学问的。他说他学了康德以后，有个问题想不通。然后他提到论语里面，曾子说过，“吾日三省吾身，为人谋而不忠乎？与朋友交而不信乎？传不习乎？”

他说，表面上看每句话都对，要一个人自律，要有真心，要有诚信，要学而实习之等等。

但是试问我们自己，并不是每个人都能做到的，甚至可以说是很难做到的。

他觉得问题就出在“人依靠什么自律”这个事情上。

那么他听我谈过康德，说是靠上帝的绝对命令来自律。

 **那么儒家是靠什么自律呢？靠天道？但儒家没有追问什么是天道？或靠自己？可儒家好像也没有追问“我”是谁。**

这样一来，曾子这说法是不是就大有问题呢？

这整个是一个很重要，也不容易说清楚的问题。

我们先看西方康德来说，康德认为人要自律，这自律来自于上帝的绝对命令。“律”这个字，就有命令的意思。命令的来源是上帝，它就具有普遍性，跟不谈条件的无条件的这种要求。

 **那么命令的内容是什么呢？就是“我应该”这三个字，也就是我应该，我有良知，我就有适当的义务要去做这些事，是无条件的，这是康德一面。**

那儒家这一面有吗？他特别提到，好像儒家没有追问什么是天道，也没有追问什么是自我。

事实上，你如果深入思考，都是有的。

 **第一个，儒家基本上以孔子来说，他不谈天道，他谈天命。** 因为在古代讲天道，通常是指天地运行的规律，特别是指对人有一种吉凶的报应。

所以古代一句话非常的经典，就是赏善罚恶，这叫天道。但是孔子讲的是，“五十而知天命”。

同时君子要敬畏的首先就是天命。天命就是自觉有一种使命，来源是天，也就是“我”是一个人，“我”有这样的人性，人性向善，所以这就是对天命的一种体认。

 **第二个，儒家有没有问自我呢？自我，在孟子来说就是天给我的良知，所以这也是儒家所要问的。** 而良知具体体现在，对自己的真诚，跟对别人的感通上面，孟子就把它归结为“心的四端”。

作为良知的具体的内容，你只要真诚，它就会发出作用。

所以这一来的话，所谓的儒家对于天命，对于良知都有一定的说法，从这个地方下手，更容易了解。

 *至于曾子所说的三句话，是强调人的社会性、我跟别人之间的关系，而不是说他要谈一套哲学，来思考道德的绝对命令的来源。这样去思考就不至于说把这两个问题这个混在一起了，各有各的做法。*

很好，以上五个问题是同学们提出来的，也感谢每一位提问题的同学，谢谢。我是傅佩荣，再见。

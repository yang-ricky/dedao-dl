# 《简斯维尔》4：为什么对失业者的职业培训不成功？

我是顾衡，欢迎来到《顾衡好书榜》。

上一讲我们介绍了，通用汽车在简斯维尔的工厂关闭后，威斯康星州各级政府开出了史无前例的优惠条件，争取通用新上的平价小汽车索尼克的生产线落户简斯维尔，却在竞争中败给了密歇根。然后民主党人举行了十万人规模的游行示威，想让共和党州长沃克下台。

为了阻止沃克签署的《预算平衡法案》生效，威斯康星州政坛竟然上演了史无前例的，16名民主党州参议员集体逃到州外的丑闻。

但是民主党的种种努力最终都没有取得效果，举债搞基建和扩招公务员的路子就被堵死了。如此一来，政府还能做的事情就只剩下一件：为失业人员提供再就业培训。

# 再就业培训

再就业培训这件事情很古怪，就是通用汽车生产线上的工人，在生产线上拧了几十年螺丝之后，一旦失业了，反倒可以走进大学的校门，并且学费、书费和交通费由政府支付。这个政策，是怎么出台的呢？原因有两个：

 **一是，把一部分失业人口引到校园里去，这是缓解就业压力起效最快的办法。**

 **二是，在传统社会，一个人读了法律专业后就真的会去当律师，学了财务后就真的会去当会计。** 也就是说，他所受的教育和今后所从事的职业是强相关的。而且一旦选择了一个职业，就会踏踏实实干一辈子，很少跳槽。

但是今天，这种强相关性却已经消失了，学校里学的东西和工作中所需要的技能二者之间没啥关系。而且工作也变得非常不稳定，跨行业跳槽成了职场常态。总之，相比于传统社会，今天的劳动者从各个层面各个角度来看，稳定性和可预期性都丧失了。

如此一来，一个劳动者在他一生的生涯中，会经历多个过渡阶段。他找到的下一个工作，很可能与他的教育背景没关系，和以往的工作经验也没啥关系。在这样的过渡期，政府认为有责任出手相帮。就像奥巴马说的：我的政府会像关注龙卷风和飓风受灾地区一样，关注汽车行业的40万失业者。

2008年，奥巴马政府为10万失业工人提供了再就业培训。职业技术培训这个东西很贵，平均每个学员要花8000~1万美元。这么着，10万失业工人就需要8亿~10亿美元。联邦政府，也就是全美纳税人，掏了5.74亿美元，缺口由州、县政府补齐。简斯维尔作为重灾区，当然也得到了一大笔费用。

失业工人由政府买单进大学读书，简斯维尔恰好有这么个大学，就是黑鹰技术学院。这是一个两年制的大学，比社区大学的含金量要高一些。拿技术证书需要学习一年，而要想拿到一张副学士的文凭则需要学习两年。工厂关闭之后，不用说，2009年，黑鹰技术学院注册学生暴增了54％，学院为此新增了88门课程。

那么，花了这么多钱，效果如何呢？当然是一塌糊涂。

根据艾米·戈德斯坦的统计，就读黑鹰技术学院的2000名失业者，到2011年只有1/3找到了稳定的工作，而没去黑鹰接受培训的，却有一半找到了工作。而且去黑鹰接受培训的失业者不仅找到稳定工作的人少，即使是找到了稳定工作，他们的收入也远远低于没接受培训的。

2007年，也就是金融危机发生前，工厂关停前，简斯维尔的一个劳动者如果去黑鹰技术学院接受过培训，接受完培训平均季收入是7294美元；没接受过培训的季平均收入为7239美元，三个月加起来就差50多美元，二者相差无几。

但是到了2011年，接受过黑鹰培训的失业者，新工作的季平均收入只有3348美元，而没接受过黑鹰培训的，季平均收入为6210美元，将近前者的两倍。更好笑的是，我不读一年，我读两年，花大工夫拿副学士文凭，这样的人收入下降得反而最厉害，连3348美元都赚不到。

# 培训失败的原因

前面一堆数字，有点儿乱，咱们捋一捋：这些数据的意思是， **去黑鹰接受培训，找工作反而更难，即使找到了工作，平均工资也只有不接受培训的人的一半。而且，学习成绩越好，毕业后找工作越难，收入也越低。**

为什么会出现这样的结果呢？有这么三个原因：

 **首先，** 为失业者提供职业技能培训也不能算错，但是如果让弗里德曼来做这个事情，他一定会给失业者发比如说一万块钱的教育券，让他自己选择去哪个学校，去读什么专业。但是现在是政府直接拨款给学校，黑鹰技术学院教什么、教成什么样子，都有钱拿。这种做法是不负责的，失业的人得不到真正需要的培训，反而浪费了教育资源，也浪费了失业者再次进入劳动市场的时间。

面对潮水般涌来的新学生，黑鹰技术学院的师资和设备明显不足。比方说，最受欢迎的专业是电网线路检修。为什么这个专业最吃香呢？因为简斯维尔的联合能源公司，是个公共事业公司，是公务员，有编制，比较稳定。而当时联合能源公司的工人平均年龄已经55岁了，学校就跟学生们说，这家公司近期肯定会招工，因为临近退休的人很多。呼啦，被同学们戏称为爬电线杆的专业就报了一大堆人。那最受欢迎的专业的师资是什么样呢？只是一个老师，就是以前爬过电线杆的线路检修工人。

 **其次，** 通用汽车的工人们早就被惯坏了，骄傲自满，固步自封。学校的老师惊奇地发现，很多人竟然不会用电脑，连开机都不会。一大半学生只会在纸上写作业，学校强令从今天起不许写在纸上，必须用Word文档后，竟然有很多人因为这个就退学了。不过事后来看，退学对他们还真不是坏事。

 **第三，** 现在的职业培训，都会煞有介事地先给学生做个测试，明尼苏达多项人格测定啊、九宫格什么的，整得神乎其神的，但其实这个、那个的心理学量表并不比算命更准。比方说39岁的马特填完入学表格后，学校立即对他做了个测试，结果显示，他适合的工作是数据库开发员、足病医生、护士和园艺师。我实在是看不出来这四项工作之间有什么共性可言。

为了能找到体制内工作，马特最终这四项工作都没选，还是选择的爬电线杆专业。但是没想到，因为经济大环境不好，那帮吃皇粮的55岁工人决定延迟退休，黑鹰技术学院最热门专业的学员们，一年的时间，只学会了爬电线杆和换保险丝，这个技能能找到什么工作呢？

我们就当电线杆专业这事儿是个乌龙，不算数。那我们把眼光放宏观些来看，学校向学生推荐的“容易找到工作的专业”，靠谱吗？肯定不靠谱，肯定是拍脑袋瞎胡扯。学校是教书的，它上哪儿知道什么专业容易找到工作呢？所以不奇怪的是，2007年以前你去黑鹰读书，如果你选一个“毕业后容易找到工作”的专业，找到工作的比例是81％。但是你随便找个专业的话，找到工作的比例却高达89％。

工厂关闭后，如果你听学校的，去读一个“容易找到工作的专业”，毕业后找到工作的概率是62％。如果你不听学校的，按自己的兴趣来，找到工作的概率是61％。可见学校向学生推荐专业这事儿，完全是乱弹琴。

# 小结

好，到这儿我们小结一下：

 **简斯维尔是美国制造业空心化的一个缩影，而制造业空心化的根本原因是工会这个毒瘤，通过集体议价逼迫资方支付远高于市场价格的薪酬和福利。企业规模越大，工会对资方胁迫的能力就越强，这就是美国锈带衰落的根本原因。**

上世纪90年代，在新一轮经济全球化的浪潮下，技术和资本实现了自由流动，钢铁之都匹兹堡，还有汽车之都底特律和费城的凋零，也就成了必然的结局。面对这样的困境，共和党和民主党开出的药方是不一样的。

好，艾米·戈德斯坦的名著《简斯维尔：一个美国故事》就介绍完了。这本书很好读，充满了让人感动的细节。主角虽然是美国一个十八线的小城，从中却折射出美国的政治生态，和2018年以来中美贸易战爆发背后的深层次原因，强烈建议你亲自读一读。

最后我还要补充几句：2008年工厂关停后，简斯维尔的各个社会团体向陷入窘境的一个个家庭伸出了援手，商人和企业家们也纷纷捐款，并自发成立了一个名为“罗克县5.0战略”的组织，全力推动简斯维尔的多元化转型。

2020年，简斯维尔作为一个曾经的汽车城，在工厂关停后，人口反而比2008年有所增长。各行各业的新企业纷纷落户简斯维尔，为这里的人们带来了更多的工作机会。在特朗普任期内，简斯维尔的失业率跌到了4％，是50年来的最低值。奥巴马政府八年没有做到的事情，简斯维尔人靠自救，却做到了。这就是社区的韧性、市场的规则和人们对家乡的爱，所展现出的力量。

当然去年又传出消息，美国一家汽车公司打算在锈带开一家新工厂的时候，工会领袖又去了，又要去集体谈薪资，好消息是这家工厂并没有选择简斯维尔，简斯维尔已经把那块地卖掉了，来实现产业多元化的转型。

好，让我们祝简斯维尔这个以前的小汽车城好运吧。

我是顾衡，感谢你的收听，咱们下期见！

## 划重点

1.政府主导的再就业培训不是解决失业问题的良方,而是可能带来更多问题,如资源浪费和培训与就业需求不匹配。
2.工会通过集体议价抬高薪酬福利不是保护工人权益,而是制造业空心化和锈带衰落的根本原因之一。
3.简斯维尔的转型成功不是依靠政府干预,而是社区自救、市场规律和居民对家乡的热爱共同作用的结果。

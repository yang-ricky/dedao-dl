# 203 官腔：权力社会，所有的人都是弱势群体

上一讲我们讲了武松具有非凡的侦查能力，不到半天的时间，他就证明了他的怀疑：哥哥武大是被害死的；他还把基本的案情搞清楚了：潘金莲、西门庆和王婆都是共犯。

那么面对基本清楚的案情，武松选择了什么方式来复仇呢？

武松此时毕竟是县公安局的刑警大队长，体制里面的人呢，所以，他要公事公办。

武松便把何九叔、郓哥一直带到了县厅上。对知县说：“小人亲兄武大被西门庆与嫂通奸，下毒药谋杀性命。这两个便是证见，要相公做主则个。”

知县马上就与县吏商议。可是县吏是与西门庆有关系的，他说：“这件事难以理问。”

其实知县也同样和西门庆有关系，于是对武松说道：“武松，你也是个本县都头，不省得法度？自古道：‘捉奸见双，捉贼见赃，杀人见伤。’你那哥哥的尸首又没了，你又不曾捉得他奸。如今只凭这两个言语，便问他杀人公事，莫非忒偏向么？你不可造次……”

武松觉得自己是公家人，所以要公事公办；

县令也觉得武松是公家人，所以要懂事，要有觉悟。

也就是说，不要给公家找麻烦了。这个案子，立案的依据不足，不能立案。

武松怀里去取出两块酥黑的骨头，十两银子，一张纸，告道：“覆告相公，这个须不是小人捏合出来的。”

这下总有依据了吧？

知县看了，没话说了，道：“你且起来，待我从长商议。可行时，便与你拿问。”

何九叔、郓哥都被武松留在房子里。有证人，有证物，剩下的，那就是拘捕嫌疑人了，至少要立案侦查。

但是，没想到，第二天一早，知县却回出了骨殖和银子来，说道：“这件事不明白，难以对理。圣人云：‘经目之事犹恐未真，背后之言，岂能全信？’不可一时造次。”

县令这么一说，县吏马上帮腔道：“都头，但凡人命之事，须要尸、伤、病、物、踪五件俱全，方可推问得。”

你看这个县令和这个县吏啊，为什么他们推三阻四，就是不愿意立案呢？原来，当日西门庆得知武松要告他，早已经派人来县里给他们送银两了。

武松可能会觉得奇怪，我可是县步兵的都头，县公安局刑警大队的大队长啊！我也是打虎英雄，戴着大红花接受群众欢呼的英雄模范啊！我还刚刚帮知县办过一件私密的家事，也算得上是知县的心腹人啊！

所以，我可不是一般的普通百姓啊！

我们在讲武松打虎之前的心理活动的时候，特别提到武松的自视不凡。他见到树皮上关于景阳冈上有老虎的警示，自言自语地说：“这是酒家诡诈，惊吓那一等客人，便去那厮家里宿歇。我却怕甚么鸟！”

你看，即使在那时，武松自己也就是一个下层人，但是他对一般人是满含轻蔑的，对自己则是充满自我优越感的。

现在，作为都头，作为一个打虎英雄，作为一个县令的心腹人，他当然更有高人一等的感觉。

但是，没想到，作为县公安局的刑警大队长，他连为自己的哥哥被谋杀要求立案都被拒绝。

所以，你记住， **不要以为自己不是弱势群体，在权力的社会里面，所有的人都是弱势群体。区别只在于时势。**

好，我们来分析一下知县和这县吏所说的话，你会发现，他们是在打官腔。而且，这是两种最有代表性的官腔，政治官腔和专业官腔。可以成为我们官腔研究的典型案例啊，我们不妨来看一看啊。

知县的官腔，我把它称之为政治官腔。

政治官腔的使用者一般是有一定权力和地位的官僚。这种官腔的用途在于，确立自己的政治优势，显示自己的政治正确。

你看这个知县，他不仅口称“法度”，甚至抬出圣人的话来证明自己的政治正确。虽然他所说的什么圣人之言完全是他的捏造，但是，你记住了， **政治官腔最喜欢的就是利用圣人的言论，来说套话、官话，这是政治官腔的基本特征。**

好，我们再看看县吏的官腔。县吏的官腔叫专业官腔。

 **专业官腔的使用者，一般都是一些什么人呢？都是一些为权势者或者是某些利益集团所御用的学者和专家。**

 **这种官腔的用途在于确立他们自己的专业知识优势。** 你看这个狱吏，他说出了一大堆的专业术语，什么尸、伤、病、物、踪，令人摸不着头脑。既然摸不着头脑，那你也就无法反驳他，他马上就占有了有利的专业知识的优势。

简单地说，官腔既是一种说话方式，更是一种话语权。有话语权的人才有资格打官腔。

像我们上面分析的，县令掌握的是政治话语权。而县吏掌握的呢，是技术话语权。

老百姓怎么样？没有话语权。只能忍气吞声。否则，那你就是不懂事，甚至是寻衅滋事。不懂事有时候叫不明真相，闹事那就叫危害稳定，这都是可怕的罪名。

但是，武松哪里会被官腔吓住呢？他倒不是不怕官腔，或者有什么办法对付官腔。他是根本没有耐心来听官腔。

 **对武松来说，第一，不管你什么政治正确，也不管你什么圣贤之言，我只要基本良知。**

金圣叹在知县的“圣人言”三个字下面批了这么一句话：“三字骗得进士，骗不得武二”。为什么骗不得武二呢？因为武二没那么多想法，他只要一个基本的常识，那就是杀人偿命，欠债还钱。

第二，武松也不懂得什么专业术语，他也只要常识。他只知道，哥哥被人谋害了，弟弟必须为他讨还公道。

没错啊，对付官腔的最好武器就是良知和常识，不跟着官腔绕弯弯。你跟着官腔绕，那一定会把你绕得一点道理都没有，一点脾气都没有，一点头脑都没有。

所以， **对付官腔的最好办法是什么？就是随你官腔说千道万，我就坚持一点：良知和常识。**

最简单的思路有时候恰恰是最正确的，最不被别人的官腔愚弄和牵着鼻子走的思路啊。武松就是这样的个性，这样的思路。

所以，面对知县的官腔，武松几乎一点也不听，也不会给知县找什么麻烦，他马上就打了退堂鼓。

武松道：“既然相公不准所告，且却又理会。”你看，他毫不纠缠。

但是，却又理会，是怎么个“理会”法呢？很简单，往下看你就知道了。他的理会之法就是杀嫂，杀西门庆，血溅鸳鸯楼，反上二龙山，啸聚梁山泊。

这就是官腔的后果啊。

好的，这一讲我的题目是：官腔。并且主要分析了两种官腔：政治官腔和专业官腔。官员一般打政治官腔，专家一般会打专业官腔，虽然立论的依据不同，却都是官腔。更重要的是，两种官腔的目的都是堵死你利益诉求、讨还公道的道路，使你无法可想。

有一句特别有意思的话是这么说的：“你没有办法，他有法办你。”说的就是这个意思了，武松为什么没有办法？因为他的办法都被官腔堵住了。县令、县吏有什么办法来办武松呢？有官腔啊！

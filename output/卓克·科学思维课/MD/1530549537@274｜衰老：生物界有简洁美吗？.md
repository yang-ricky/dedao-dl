# 274｜衰老：生物界有简洁美吗？

## 概念27：衰老

衰老，是每个人都会经历的，但它并不是每个生物都会经历的，有些动物就没有明显的老化，甚至越老越强。了解衰老，是科学地树立世界观必要的环节，它可以让我们更坦然地面对生命的最后阶段。

这节课，我们要讲真核生物和细菌的对比，从中我们要反思的是，什么是简洁，什么是美？

因为简洁的美，这样的表述经常出现。尤其是人们表达对一个事物的价值判断上。

但，起码我们细致观察生物界，就会发现，“简洁的美”这样的词汇，在科学体系里已经狭窄到没有容身之地了。

## 真核细胞与细菌

之前的课里我们说了，主流科学界对细胞中普遍存在线粒体的猜测，那就是有一个不会利用氧气的某种大块头的细菌吞噬了一种会利用氧气的小块头的细菌。而且小细菌还有幸没有被消化掉，这样，它们达成了共生的细胞之后，也就是现在的真核细胞。

说起来轻松，但实际发生这样的转变，需要迈过很多大的步骤。因为单细胞的细菌跟真核细胞差别实在太大了。

这种巨大的差异，最明显就体现在体型上，按体积来算，真核细胞大约是细菌的好几万倍。

那么细菌对比真核细胞算不算是简洁呢？

> 那现在你可以在心中下一个判断。

我们接着说，如果按照结构来看，真核细胞有细胞核，遗传物质被保护在里面，而且遗传物质特别多。

可是反观，细菌那边没有核，遗传物质直接暴露在里边，量也非常少。此外，细菌都有细胞壁。你可以把细胞壁这种东西理解成昆虫的外壳那样的硬邦邦的，维持一个很稳固的结构那样一个东西。即便细菌肚子里的液体都被吸干了，它这个外壳还是可以维持大致的形状，这是细菌。

但是真核细胞就没有这么一个硬壳，它只有一个油脂一样的膜，真核细胞是随着外界的压力改变外形的。

当然，区别还是有很多，我们刚才只列了几个不需要太多基础知识铺垫的区别。

> 那么听了这么多，也请你现在下一个判断，真核细胞跟细菌哪个是简洁的，哪个是美的呢？

## 遗传物质上的差异

为了提供更多的细节，我们着重对比真核细胞里的遗传物质跟细菌里的遗传物质，看它们的多少。

真核细胞里的遗传物质要比细菌里的多太多了，衡量的标准是用一个专业词，叫做 c 值。指的是一个细胞中单倍体的 DNA 含量，单位叫皮克，就是一万亿分之一克，这是一个质量单位。

那么假如我们的目的是比较一下不同生物 DNA 数据量上的多少，假如大家全都是双倍体，那我们只要衡量遗传物质的质量有多少克，就可以对比它们的数据了。

但是生物界是有很多样式的，甚至有很多奇葩的，它们主要就是各种的植物，比如说小麦，虽然以六倍体为主，但是也有三倍体、四倍体、八倍体，还有其它植物甚至有十几倍体的情况。

这个时候，我们要对比这种植物遗传物质的数据量多少的时候，不能简单地称量它的质量了，因为它们也许很沉，但是沉是因为拷贝的数量太多，不是数据量的多导致的，所以要衡量数据量多少，得衡量单个拷贝上的数据量才可以，这就是 c 值的来源。

如果我们通俗地比喻一下，就是：

> 一个书商他有3万多册的图书库存，而一个读书人可能只有5千册的藏书。
> 
> 
> 
> 但我们不能说书商拥有的信息远远多于那个读书人。
> 
> 
> 
> 因为书商是搞批发的，这3万册书可能都是《物种的起源》，都等着批发给下游的渠道呢。而那个读书人的5千册书可是基本没有重样的。

知道 c 值的含义，我们来对比。

* 第一个是酵母菌，酵母菌是最简单的真核生物。那对比细菌来说，它的 c 值（0.018pg）是细菌的3倍。

* 那我们再找一个比酵母菌稍微复杂一些的，最简单的多细胞生物，叫秀丽隐杆线虫，它的 c 值这下成了细菌的13倍。

* 我们再找一个更复杂的，就是我们人类自己，他的 c 值（3.1pg）是细菌的500多倍了。

* 那我们再看看其它的哺乳类动物、鸟类，一查它们的 c 值也大约跟人是差不多的。

那到此为止，这个规律很好地体现出来了，越复杂的生物，c 值越高。但是刚刚这些数据只是我精心挑选出来的，貌似让你听上去符合我们心意的数据。

比如娃娃鱼这种活化石一样的生物，它的 c 值是人类的20倍。你觉得这20倍虽然多了点，但是这毕竟是一个活化石，经历了无数的雪雨风霜，也许遗传物质里代表着历史遗留的问题就是很多，毕竟它也长成了一条大蜥蜴的样子，不简单了。多20倍也就多20倍吧。

好，我们再来看，只有半毫米大小的无恒变形虫（Polychaos dubium），它的 c 值竟然是人类的200倍。

而且不只是娃娃鱼跟变形虫这两个特例，在爬行动物中，跟原生动物，还有植物里，有大量 c 值比人类还高的物种。

这会儿规律就打破了，我们不禁要问，这些真核生物为什么要死抓住这么庞大的数据，而不把这些数据派上用场呢？

在生命的复制过程中，要维持这么大量的数据，在拷贝的过程中保证极高的正确率，这可不是一件容易的事，它是需要一套复杂的机制来保证极高的正确率的。所以一定要耗费额外的能量。

可费了那么大事，比人的 c 值都大了200倍，可最终呈现出来的只是一个阿米巴虫的样子，那我们实在不禁要问，大自然简洁的美体现在哪？

简洁如细菌那样不好吗？

生存需要表达多少蛋白质，就尽量节省地使用多少遗传物质，细菌也不用单弄出一个细胞核出来，也不用众多的细胞器。只要有一个比较坚固的细胞壁那个外壳就可以了。这才是节省能源的简洁美啊。

但是，放眼看看当今的地球，细菌还是30亿年前的那个细菌，它们的生活方式跟身体结构基本没有变，可是真核生物已经展现出天差地别的样式了。

甚至有一部分真核生物都在思考，到底是细菌拥有简洁的美，还是我们真核生物拥有简洁的美呢？

## 课程小结

这节课，我们通过细菌和真核细胞的对比能得到什么科学上的思考呢？

那就是，简洁美到底算什么层面的评价呢？

我想，起码是强烈依附于文化偏好的一种评价，简洁美跟一切同类的形容词，也都是在人类出现了语言之后才有了意义。这类的词，实在不能用来形容此前出现的事物跟规律。所以也可以说这类词的适用范围是如此的狭窄。

那什么样的表达适用范围能广一些呢？

当然就是科学方式的表达。

就拿这节课的 c 值来说，酵母菌是细菌的3倍，人是酵母菌的4倍，变形虫那么原始，却又是人的200倍，对此我们不妄加感叹，这些可测且可验证的内容起到的效果是，吸引我们继续探索更深层的原因。

既然生物上是这样，那其实任何的学科，像物理、数学、化学都是这样的。

 **我们无法对科学细节中哪部分更美，哪部分更丑，哪部分更简洁，哪部分更冗余，哪部分更先进，哪部分更落后，作出什么恰当的形容跟描述，因为这些文学词汇的核心概念，都比科学描述的那个对象，出现得晚得多得多。 **

当我们站在科学证据面前，所能使用的形容词也许只有惊叹和困惑了。

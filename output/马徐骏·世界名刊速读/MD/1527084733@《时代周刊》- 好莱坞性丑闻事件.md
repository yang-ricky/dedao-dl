## 知识清单：

 **| 今日关键词**

 *casting-couch [ ˈkæstɪŋ kautʃ ] culture ：* 试镜椅文化。casting-couch ，演员去面试角色的时候坐的沙发。为了得到自己想要的角色，有时候演员就不得不满足一下有权势决定他们命运的人提出的要求。后来这个casting-couch culture 就被引申为以性为交易获得角色的行为，这在美国影视界非常普遍。也有人把这个词翻译为“潜规则”。

 *tacit [ˈtæsɪt ] code ：* 潜规则。tacit ，是“不言而喻、心照不宣”的意思，code 是“代码、规范”的意思，合起来，就是“心照不宣的规则”，即tacit code ，潜规则。

 **| 身败名裂的哈维·温斯坦**

今天的内容来自于《时代周刊》的封面文章《身败名裂的哈维·温斯坦》，The downfall of Harvey Weinstein。讲的是最近轰动美国的一起好莱坞性丑闻事件。

本期《时代周刊》的View （观点）栏目请了四位不同行业和领域，并与这一事件有关联的人发表对于这次事件的看法。View 栏目是《时代周刊》一个非常有特色的专栏，针对美国国内的热点问题发表观点，经常会有很多独到和犀利的看法。

 **| 权势熏天的哈维·温斯坦**

哈维·温斯坦是好莱坞的金牌制作人，在好莱坞几乎无人不知、无人不晓。他发掘了很多优秀的演员和导演，比如拍出《低俗小说》的昆汀·塔伦蒂诺。哈维·温斯坦发行和制作的电影里，光是奥斯卡奖的提名就拿了300多个，而真正拿下奥斯卡各个奖项的，有70多个，很多获奖作品咱们中国人也比较熟悉，比如《莎翁情史》《国王的演讲》《英国病人》《危险关系》《心灵捕手》等等，包括很多的华语片，像《无间道》《英雄》《一代宗师》等能去北美发行上映，也都是哈维·温斯坦一手操办的。

所以这个人在好莱坞的地位是大神级的，就连美国前总统奥巴马的女儿，都曾经在他的公司里当实习生。

但这次《时代周刊》的封面上，用了三个以P开头的单词来形容哈维·温斯坦，producer （制片人）、predator （捕食者）、pariah （贱民）。意思是，哈维·温斯坦利用自己金牌制片人producer 的权力和名声，在过去30多年的时间里，像捕食者predator 一样，不停地潜规则女明星和自己公司的女性员工，在被媒体曝光后，终于成了人人唾弃的贱民pariah 。

 **| 性骚扰与潜规则**

 *随着越来越多的女性走出家庭，进入职场，人们对于职场性骚扰的关注度明显提高了，据《时代周刊》数据统计，美国有40%的女性，都曾经在工作场合遭受性骚扰或者性侵犯，而70%到90%的女性受害者，不愿意声张。*

哈维·温斯坦事件在美国的持续发酵，实际上并非一起孤立事件，类似的事件可以说是层出不穷。去年《福克斯》新闻网的女主播格雷琴·卡尔森指控自己的老板性骚扰；甚至连美国现任总统特朗普都因为爆出侮辱女性的言论被媒体群殴过。

casting-couch culture ，“试镜椅文化”是非常有代表性的一种行业潜规则。有权势的大佬们，利用自己的权力，对处于弱势地位的演员予取予求。哈维·温斯坦就是此中老手，在过去的30年里面，他骚扰过的女明星，知名的就有30多个，比如说《莎翁情史》的女主角，也是漫威电影系列里面钢铁侠的女朋友，小辣椒格温妮丝·帕特洛，还有非常有名的《古墓丽影》的女主角安吉丽娜·朱莉。

其实在此之前，已经有不少案件被曝出来了，但是都被哈维·温斯坦的律师花钱摆平，或者被业界大佬给压下去了。但这次不一样，被《纽约时报》曝出来之后，不但哈维·温斯坦被公司开除，米拉麦卡斯公司还有四位男性董事也辞职了，但是美国媒体现在对这一事件依然是穷追猛打，不断地挖掘猛料，鼓励其他的受害者站出来说话。

 **|《时代周刊》的观点**

《时代周刊》此次邀请了四位不同领域的人士来谈自己的看法。其中两位是职场性骚扰的直接受害者。得过奥斯卡奖的美国知名女演员米拉·索维诺和福克斯新闻网的前女主播格雷琴·卡尔森，还有一位是女作家兼女律师吉尔·菲利波维奇，最后一位是《时代周刊》的影评人史蒂芬尼·柴科瑞克。

米拉·索维诺是哈维·温斯坦事件的受害者之一，1995年的时候，哈维·温斯坦就试图对她进行性侵犯，后来之所以放过了她，是因为米拉·索维诺跟大导演昆丁·塔伦蒂诺谈恋爱了。

索维诺说，现在的美国，强势男性在工作场合会对女性进行性骚扰，甚至是强暴。哪怕是像她和格雷琴·卡尔森这样成功的职业女性，也免不了遇到职场性骚扰，更别说普通人了，但是受害者却往往不敢站出来说话，因为害怕之后的闲言碎语和打击报复。

女主播格雷琴·卡尔森被自己的老板性骚扰的事件，去年在美国大选之前爆出来，也是引起了全民关注。卡尔森说现在事情过去15个月了，但她收到了好多女性的咨询，不止是从事媒体和娱乐业的女性，像一些快餐店的员工、单亲妈妈，甚至是普通的白领女职员，都向她诉说自己遭遇性骚扰的问题，但是她们苦于不知道怎么处理。所以，卡尔森为此专门建立了一个机构，为这些女性提供帮助。

女作家兼律师吉尔·菲利波维奇认为某些人之所以敢肆无忌惮、明目张胆地进行性骚扰和性侵犯，就是因为他们手里掌握着权力。权力是他们的手段，是他们的动机，也是替他们掩盖的方法。而且，像哈维·温斯坦这样的人，做这些事，他自己都不觉得有什么不对，因为西方对于女性权利的重视，是上个世纪末才开始的，1991年的时候，sexual harassment ，“性骚扰”这个名词，才第一次因为一起诉讼案，正式进入了大众视野。

那为什么哈维·温斯坦横行霸道了30年，现在却翻船了呢？

一是随着更多的女性进入职场工作，女性的经济地位和社会地位都在不断提高，所以很多女性不甘于忍气吞声，更敢于站出来揭露事实，社会给予她们的同情也更多。

二是信息传播速度越来越快，使得媒体更需要猛料来吸引眼球。从上个世纪末到现在，美国发生过很多起知名的性骚扰案，一起比一起反响大，去年格雷琴·卡尔森案件，在美国引起的关注甚至不亚于美国大选。所以现在的新闻机构，都把这样的事件当做头等要闻来重视，一旦爆出来了，就穷追猛打，不但会收获大量的关注，媒体的名声也会更响。

这就给了这类事件持续曝光和发酵的空间，公众也因此会更关注。于是，大佬们想要凭借自己的关系网络洗白，变得更加困难。有名望的人避之唯恐不及，更别说出来维护了。像哈维·温斯坦事件爆出来之后，奥巴马立刻就站出来跟他划清界限，免得被一起拖下水。

在这起事件被爆出来之后，美国社交网络上很多人说，知道了奥斯卡这些知名影片的背后居然有这么肮脏的交易和黑幕，从此“累觉不爱”了。对此，《时代周刊》的影评人史蒂芬尼·柴科瑞克说，这些电影本身是很优秀的，而且是很多文艺工作者一起努力的成果，如果因此另眼相看的话，其实是对受害者的另一种伤害。我们应当把作品和制作者分开对待。顶着压力和困难拍出优秀作品的人，特别是女性，反而更值得我们尊敬。

其实哈维·温斯坦事件至少可以给我们两点启示。 *首先，高速的信息交流和传递，使得原来看似牢不可破的东西，正在不断地松动和改变；其次，通过不停地努力和推动，哪怕再困难的目标，也会逐渐变得可行，甚至会产生多米诺骨牌一样的效应。*

就像文章中说的，要扳倒最初的几个巨人是非常困难的，哪怕是现在，也依然不容易，但是每当我们成功一次，下一个巨人的陨落，就会变得更快一些。

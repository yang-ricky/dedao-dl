## 知识清单：

 **| 今日关键词**

 *procrastination [prəʊˌkræstɪ'neɪʃn]：* 拖延。我们说的“拖延症”就是这个词。在口语当中，“推迟什么事情”也可以说 put off 。比如“今天能做的，就不要拖到明天”，“今日事，今日毕”，英文表达就是，Don't put off today's work till tomorrow 。

 *Combat [ˈkɒmbæt ]：* 战斗。“与拖延症战斗“，就是 combat procrastination 。

 **| 如何克服拖延症**

今天的内容来自《今日心理学》的《如何克服拖延症》，How To Stop Procrastinating On Self-Change。作者比尔·科诺斯是一位教育学博士，也是很高产的作家，写过超过20本书，其中就有一本是专门讲克服拖延症的。

今天这篇文章，讲的就是如何克服拖延症。作者科诺斯对自己的方法非常自信，他在文章中，说自己的方法是“绝对不会失败的方法”，No failure way 。他不仅讲了方法论，还讲了在实际操作过程中有可能出现的思维陷阱。比如说，我们想做什么事儿，拖延的时候总会有一个“但是”帮我们找理由，这就是一个思维陷阱。

 **| 战拖行动篇：想清楚，快动手**

 *“想清楚，快动手”是克服拖延症的六字箴言。*

“想清楚”，就是找出你最渴望实现的目标。

作者在文章中说，其实大家都差不多，有很多的目标想实现。比如说想读更多的书、学习更多的知识、写出漂亮的文章、减肥、健身、变得更自信、不再害怕别人的拒绝，等等。但也许我们拖延的原因之一就是我们想要得太多了，没办法一次全部完成。不知道你发现没有，大家一直在喊“战拖战拖，战胜拖延症”。可是，拖延的到底是什么事情呢？这么多事情，它们的重要程度是相等的吗？肯定不是，那么在实现这些目标的时候是不是也要有一个优先级呢？

所以，第一步就是，在那么多想要做的事情中，找到一个对你来说最重要，最希望能够实现的目标。你要想清楚，为什么你特别想实现这个目标？如果实现了，它能为你带来哪些好处？是能够让你获得有价值的提高，还是能够帮你解决一个很重大的问题？把这些优点通通写下来。

然后，将这个目标具体化，让它可衡量、可管理。这个咱们得具体解释一下，比如说你想减肥，你的目标如果写成——“要瘦”，这就不是一个可衡量、可管理的目标。怎么改是对的呢？比如说，可以改成每周瘦一斤。这就是一个可衡量、可管理的目标了。

好，通过这一步， *你找到了自己最不想拖延、最想实现的目标；同时，你也知道实现了这个目标，自己会有哪些收获，获得了动力。最后，通过具体化这个目标，有了努力的方向。*

请把这三点：你的目标、它能带来的好处、具体努力的方向，都用笔写下来。在写完了之后，你甚至可以拍一张照片发给自己，这就是你要做出改变的证据。

下一步就特别重要了，就是快动手。

重点就是“快、快、快”，别想那么多，先做起来再说。目标也设置好了，从其中最简单的步骤开始。只要开始做了，坚持下去的可能性就高了很多。

文章给了这样一个例子，主人公叫Tom。Tom觉得自己在和别人对话时，很容易动摇，很难坚持自己的观点，导致别人也不是很尊重他。Tom不喜欢这种状态，所以他的目标就是——在必要的时候站出来，勇敢地表达、捍卫自己的观点。如果是直接让他在工作会议上站出来，坚持自己的观点，那对他来说挑战就太大了。

做出改变，最重要的就是开始。他可以先找一些相关的书来看一看，然后，找出书中他认为有效的方法去尝试一下，这就容易了很多。很多时候，我们制定了非常详细的目标，也知道这能带给我们什么，可就是迟迟不开始，也许就是自己把自己给吓住了。从最简单的步骤入手，做起来，你就走在成功的路上了。

但是很多时候，即使我们想得非常好，还是没法开始行动。总是给自己找一些理由拖延，比如说“我还没准备好”，那怎么办呢？

 **| 战拖心理篇：Yes / but思维陷阱**

Yes / but思维的表现形式是这样的，你会在心里想：“Yes，战胜拖延、改变自我很重要，but，我不做是有理由的。”

比如，“Yes，我知道我应该多运动，but，我还没准备好”，再比如，“Yes，我知道我应该努力学英语，but，我还没准备好。”作者说，你会发现“我还没准备好”这句话几乎是一个万能理由，拖延任何事情，都可以拿这句话来作为理由。它背后的思想就是——不是我不想做，而是我还没准备好。这让我们在心理上得到了极大的解脱，也不那么愧疚了。but 后面还有可能接各种五花八门的理由，比如我身体不舒服、我衣服还没洗、我先歇一会儿、我找个搭档或者伙伴一起做比较好，反正就是让自己在拖延时不那么难受。

那么，怎么克服 Yes / but 思维呢？

作者在文章当中说，拖延，从表面上来看，也许只是推迟做什么事情的时间，但内里，是你内心对于失败、挑战和本该面临解决问题的逃避。有些时候，你自己都没有意识到这些问题。不过，它们有可能就藏在你的but 里。

当你开始拖延的时候，你要有意识地去查看一下自己有没有Yes / but思维。如果有，你就把这些but通通写下来，一条一条去看，也许你就能发现这背后真正的问题。

再拿刚才的Tom举例，他不是希望自己能够勇敢地表达、捍卫自己的观点吗？在这个过程中，就可能出现 Yes / but 思维。Tom脑海中可能有个声音说：“Yes，是的，我是需要勇于表达自己的观点，坚持自己的立场，但是，这对我来说压力太大了。”这个时候Tom就要深入地去看but 这句话，挖掘背后的问题。比如说，到底是觉得哪儿压力大？是担心如果坚持自己的观点，看起来会很傻吗？如果担心的是这个，接下来，Tom就有非常具体的问题要解决了。首先，自己坚持表达观点的样子，真的傻吗？Tom可以去问问自己的朋友、同事。他有可能会发现，根本不是这么回事儿，别人都觉得很棒。即使是真的傻，Tom也找到了自己真正要解决的问题了。

 *Yes / but思维陷阱在于，but后面的句子给了我们足够的理由不去愧疚，推迟我们想实现的各种美好的目标。但通过今天的内容，我们也知道，我们可以通过分析发现隐藏在其中真正需要解决的问题，着手开始改变。*

好，总结一下：我们的生活中，确实有很多的目标值得我们去实现。也许有的时候，就是因为想做的事情太多，我们反而迷失在其中。也许是时候静下心来，拿出你的笔和纸，写下你所有想实现的目标和愿望，然后找出其中对你最重要的那件事情，开始做起。当然，也许会出现很多 Yes / but 的句子，写下这些想法，找出真正的问题，一项一项地去解决。正如文章当中所说，改变很难，拖延很容易。绝大多数的改变都没有它们看上去的那么难。

给你留一道思考题：“按照今天的内容，找出你最想克服拖延，想实现的目标”。欢迎在留言区写下你的目标，作为开始改变的见证。

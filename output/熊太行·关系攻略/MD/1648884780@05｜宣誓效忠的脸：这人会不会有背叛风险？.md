# 05｜宣誓效忠的脸：这人会不会有背叛风险？

你好，我是熊太行，欢迎来到《关系攻略》。

今天我们来说一说，宣誓效忠的脸。

一个优秀的人在职场上，应该会有那些春风得意的时候。比如，刚刚被提升，或者在全公司同事面前被表彰，又或者得到某个重要领导的赏识。 这个时候，多半会有宣誓效忠的脸飘然而至。

如果你刚入职场就崭露头角，他们会说：“你真的太棒了，少年才俊。”

如果你正值壮年，受到重用，他们会说：“您真的太优秀了，真不愧是本单位的中流砥柱。”

如果你厚积薄发，大器晚成，他们会说：“真是太佩服您了，姜还是老的辣。”

无论你是初出茅庐，还是老树发新枝，这种脸都会笑嘻嘻地迎上来。马季先生的相声里就曾经塑造过一个这样的角色，别管你怎么谦虚，他都能把你捧上天。

捧完你之后，接下来他就会说明自己的来意：

“以后还得请您多关照，有需要用我的地方，随时吩咐。”

这就是宣誓效忠的脸，你不掌权、不走红，就永远看不见这张脸，你开始飞黄腾达的时候，这张脸就会立刻来到你的跟前。

有同学跟我说：“熊师傅，这种人应该怎么对待才好呢？如果我把这样的人直接推开，撵走，对方是小人还好，但如果他是真心支持我，那可怎么办呢？”

自古以来，区分别人的忠诚是真是假，都是权力场上的难点。一个人嘴上说效忠你，但实际上可能并不打算真的为你效力。

那么，怎样才能知道，宣誓效忠的脸背后的真实想法呢？这张脸的主人有没有可能背叛你？如何应对这样的脸才得体？

这一讲我就给你说说，怎么应对职场上那些宣誓效忠的脸。

# 1.宣誓效忠的脸背后有三种人

关于忠诚，我在以前的课程里专门谈过，我把文稿的链接放在文字里，你可以找来复习一下（链接：《忠诚攻略：你一定对职场上的忠诚有许多误会》）。

在职场上，最重要的关系就是你和你领导的关系。

这个领导说的是直接领导，你最好不要越过直接领导去向大领导表示忠诚，包括不分管你的副职高层领导。

因为职场上的忠诚有排他性，随便宣誓效忠很危险。所以， **真正的忠诚，只应该存在于直接的上下级关系里。**

明白了这一点，如何辨别效忠的脸就很简单了。

宣誓效忠的脸背后有三种人：

第一种人是 **谄媚的人** 。这种人不是你的下属，仅仅是因为级别比你低，就对你摆出一张效忠的脸。

为什么说他们是谄媚的人？

这不是我的论断，而是孔子的判断。孔子曾经说过：“非其鬼而祭之,谄也。”

这里的鬼，说的是祖先，一个人要祭祀别人家的祖宗，那一定是想巴结、谄媚活着的人。

职场上的效忠宣言也是类似，你不是他的领导，他来巴结你、谄媚你，多半是为了日后让你违背规章制度，给他行点方便，或者希望调入你的部门，弄个一官半职。

第二种人是 **效力的人** 。这类人跟你在同一个部门里，他们不一定是你的下属，但是在你的保护之下，这些人的宣誓效忠，有祝贺和重申忠诚的意味。

第三种人是 **叵测的人** 。这类人比较特殊，他们可能是你之前的对手，是竞争当中的失败者，他们的宣誓效忠最难判断。

一方面他们可能确实接受了你受提拔的事实，也表达了自己愿意合作的态度。但是另一方面，他们可能在你地位动摇的时候，做出对你不利的事情。

给你举个例子，电视剧《雍正王朝》里，八王爷就在雍正继位后，对政权把握不稳的时候，突然召集了八大贝勒进京议政，想要动摇皇权。 好在十三王爷及时赶到，才化解了这场危机。

你看，争夺皇位，性命攸关，尚且有人愿意冒险。在职场上安全地放一支冷箭，当然更是大有人在。

效忠的话很顺耳，很好听，但宣誓效忠的脸背后， *只有效力的人是好人，谄媚的人和叵测的人，你都应该小心提防。*

那么，在具体的社交场合里，你应该如何对待这三种人呢？

# 2.对三种人的区别策略

先说如何 **对待谄媚的人** 。

对这类人 **你要做到礼貌、客气，同时保持距离，观察他下一步的行动。**

如果他只是碰巧在这个场合，习惯性地吹捧你一下，那就不用多虑。但如果他之后开始刻意接近你，并给你各种小便利、小好处，那就要谨慎了。

这种人不会出于欣赏你而对你好的，他一定打算从你这里得到点什么。你今天收他一点微不足道的好处，未来都可能十倍、二十倍地偿还。

对谄媚者的收买，可以尽量把对方的礼物、宴请推掉，实在推不掉的，要尽量还人情。

还人情，北京话也叫“还镚子”，今天你吃饭买单，那明天就我来花钱，你要是不答应，我下次就不跟你一起出来吃了，让双方保持互不相欠的状态，这就是“还镚子”，这是用礼貌的方式让对方知难而退。

再说如何 **对效力的人** 。

对这些人 **要尽量保持原状** 。千万不要随便降低对方的职位，或是他的待遇。即使你对这个人不满意，也应该在过渡期结束之后再处理。在你刚刚被提拔、被重用的时候，每个支持者都是宝贵的。

当然，也不要对所有的效忠声明都立刻分配好处，这对那些比较内向，渴望凭本事吃饭的成员不公平。这些人也是你需要拉拢和依靠的，而且你手上可以分配的资源有限，为一个友善的态度就随便给好处，未来办事就困难了。

最后说如何 **对叵测的人** 。

对这类人 **你要特别小心，光听他怎么说是没用的，要尽快给他布置任务。** 要挑那种有点难度，但没那么关键的任务。

如果他真心决定做你的属下，那一定会克服难关，在工作上表现出效率和活力来，如果是虚与委蛇，想要麻痹你，那他一定会表现出懒散、低效，甚至还会抱怨。

用有点难度的任务来试探态度不定的人，是自古以来的妙招。

清兵刚入关的时候，就让明朝投降的将领来打头阵，让他们费力气、伤实力，看他们投降的态度是不是真的。

反观明朝，过去招抚李自成、张献忠这样的起义军，都把这些人当爷供着，给他们的压力很少，结果这些人很快又反叛了。

 **作为被效忠者，你有考验效忠者的特权，用任务考验对方的效忠是不是真的。** 这种考验，就是“交投名状”，这不是欺负或者为难对方，而是确认双方关系的必要环节。

别担心下属可能因此跑路，下属会因为你不公正的待遇离开，但一般不会因为你给了有挑战性的任务就离开。

不过，你真正应该关心的，其实不应该是那些宣誓效忠的脸，这是为什么呢？

# 3.注意力应该放在其他沉默者身上

因为有些下属不会对你宣誓效忠，他们才更需要你的注意。他们不宣誓效忠的理由可能很不一样。

有的人可能不擅长在口头上表达自己的忠诚；有的人则是鄙夷某个谄媚的人，所以刻意回避自己也做同样的事；还有一些人，他们脾气桀骜，对哪个领导都不买账。

这些人不来找你表忠心，选择了保守的策略，选择沉默。如果你简单粗暴地给对方小鞋穿，这是非常不得体的行为。你可能会错失一员得力的干将，还可能还会把一些人推到你的对立面。

对没有宣誓效忠的人，挨个聊聊，听听他们对未来工作的看法，这才是一个新手领导最该做的事。

 **把下属里的业务骨干安排好，协调好团队成员之间的关系，让团队继续稳定输出成绩，再逐渐把不称职的人换掉，这才是新手领导最应该关心的。**

收获别人口头上的支持，虽然有必要，但是不重要。多关注那些沉默者，因为他们才是更大的变数。

最后，你还有一件重要的事情要做。

你既然被提拔了，就要让你的领导知道你对他的感激和敬意，告诉他你是他的友军，准备响应听他的招呼。

你不需要肉麻地宣誓效忠，但你一定要认真地表达你的态度。即使你和领导之间有默契，但是很多话如果你不说，他就可能犯嘀咕，你们的关系就容易出现裂痕。

再强调一下职场上最重要的那个人——他就是你的领导。

# 总结

 **1.宣誓效忠的脸背后有三种人：谄媚的人、效力的人和叵测的人。**

 **2.对谄媚的人你要客气、礼貌地推开；对渴望效力的人你要保障他们的权益；对叵测的人你要进行考验，让他们交出投名状。**

 **3.你真正应该关心的是那些沉默者，他们可能因为耿直或者社恐，不愿意过来对你宣誓效忠，主动跟他们沟通，才能更好地安排工作。**

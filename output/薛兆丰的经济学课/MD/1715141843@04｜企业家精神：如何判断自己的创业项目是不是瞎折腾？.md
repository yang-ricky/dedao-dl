# 企业家精神：如何判断自己的创业项目是不是瞎折腾？

今天我来跟你分享关于《万物经济学》第四集《我们与AI的较量：北京（下）》的经济学体会。

# 创业不是瞎折腾

首先我来跟你回顾一下这一集的内容。

这一集的窗口人物是张紫杰先生。紫杰是一位人工智能的初创者，这些年来，他在许多风口行业的公司工作过，担任的职务也非常重要。但他说，自己总是和各个风口擦肩而过，言下之意是：他并没有因为刚好在风口行业工作，而挣到大钱。终于，他过够了打工人的生活，辞职创业，自己做起了老板。

我在望京SOHO一间景色优美的高层办公室里，见到了紫杰，我坐下来听紫杰介绍他的业务。他正在做的一个小工程，是把AI技术植入到社交平台软件里面，用户就不需要直接打开AI专用的软件，而只需要在他们习惯使用的社交平台软件上，就可以调用AI的服务了。

在我看来，这种改进虽然贴心，但作用并不是很大，而且收费会成为一个障碍。如果不能顺利地收费，这种新的技术就不能持续地得到正反馈，也就很难完善和普及。

几个小时以后，听完紫杰的仔细介绍，我坦率地跟他说：我感觉你手上没活儿，我觉得你手上做的事情，不是一件能让你奋不顾身、废寝忘食、日以继夜地去完成的工作。于是，我和紫杰就去拜访了几位同样从事AI创业的人。我想让他感受到，在创业的过程中，手上有事儿究竟是什么意思。

# 理智评估自己的方向

我们拜访了一家名字叫Deep Music的音乐初创公司，这家公司的CEO刘晓光先生接待了我们。晓光是清华大学化学系毕业的，他从小就喜欢音乐，拿过电子琴表演十级证书。可惜他高考的那年，电子琴证书就不能作为加分项了。他说他选择化学系是因为高中读得太辛苦了，大学想偷懒一下，所以选择了一门他认为最容易读的专业。

在清华读化学期间，他就参加了老师组织的作曲俱乐部，后来他在这个俱乐部的基础上组建了自己的公司。让AI来创作出更多美妙的乐曲，是他当时的梦想。但是，仔细地观察和理智，让他重新评估了这个梦想。

他发现，在过去20年间，市场上可得的音乐作品的数量，从几千首上升到上百万首，但最终能被观众接受和流行的数量，却基本没有变化。也就是说，在过去20年间，需求没有变化，但供给却大大增加了。而AI作曲只会让供给增长得更快，这样来瓜分有限的听众需求。这种新技术的前景是堪忧的。

晓光非常敏锐地发现了这一点，也非常客观地接受了这一点。他调转方向，认为AI作曲的目的，不应该是创作更多好听的、流行的音乐，而是让更多普通人能够享受作曲的快乐。

确定了这个方向以后，晓光开始花很长的时间给流行的歌曲库做相应的音乐标示，让每一首大家耳熟能详的流行曲，都能够轻松地变换不同的编曲、混音和演奏风格，让音乐的外行人也能像内行人一样，轻松地操纵和尝试各种不同的音乐效果。

到现在，晓光的APP已经有2万多活跃用户。他们每天都给晓光提意见、提建议，晓光的产品也就不断地在升级、在迭代。晓光的产品哪怕是再不完美，我也觉得他是属于手上有事的，手上有活的。他的项目是能给他正反馈的，他是知道努力下去是会有什么样的结果的。

从晓光的公司出来，紫杰也说，晓光给他的冲击是最大的。我想，那是因为他们俩的背景最接近，各自手上产品的对比最鲜明。

# 做“手上有事”的创业

我和紫杰还拜访了阿里大文娱首席技术官郑勇先生。郑勇所从事的，是在阿里公司内部的创业，它的产品是数字虚拟影棚。郑勇回忆说，他当年参观过在横店的影视城，发现在那里布景的成本是非常高的，拍摄工作一旦完成，上千万元的布景就得再花上几百万元去拆除、搬运。

他想，如果这些影棚能够做成数字化的，那哪怕初期的成本比较高，后来应用的成本也会大幅下降的。抱着这样的想法，他就一点一点地和同事们开始在企业内部创业。慢慢地，他得到了越来越多的认同、越来越多的资源和越来越多的应用机会。

郑勇给我的启发是，只要找到一个能够形成闭环的创业项目，也就是手上能有事儿，能有一个言之成理的构思，那么哪怕在企业的内部，也是能够调动资源、网罗人才进行创业的。

我和紫杰还拜访了创作AI原声视频节目的汗青先生。汗青特意制作了一段我和亚当·斯密对话的视频，来说明他的工作。在这段视频里，我问亚当·斯密，他为什么没有结婚？亚当·斯密回答说，根据大卫·李嘉图的比较优势原理，他觉得把所有的时间都放在经济学的研究上，更有价值。

汗青创作了不少这种跨时空人物的对话，在网上引起了持续的关注。但是我对汗青有一个疑问。我说，在这段视频里，亚当·斯密的表演并不完全真实，因为它引用了大卫·李嘉图的比较优势原理，而大卫·李嘉图是亚当·斯密之后的人，亚当·斯密去世的时候，李嘉图才19岁，亚当·斯密不可能知道并引用李嘉图的理论。

汗青承认了这个问题，他说他只不过是利用AI来进行一些创作的构思，是在进行一种我们早已熟知的文艺创作。汗青说，AI的作用就像一位主持人，诱导不同的历史人物去发言、去对话而已。我认同了汗青的解释，我觉得这也是一件有趣而且有意义的工作。

在我看来，汗青利用AI的技术创业，也算是一种“手上有事”的创业，它至少能够源源不断地定期推出跨时空人物的对话。在这个过程中，他可以继续寻找新的价值增长点。

一番走访下来，紫杰对他手上的工作有了更清楚的认识。他对我说，他虽然出来创业了，但可能还抱着打工的心态。打工的时候他觉得发工资的那一天来得特别慢，创业以后觉得发工资的那一天来得特别快；打工的时候觉得时间不够用，创业以后觉得时间更不够用了。他说，他决心重新审视他的业务，并邀请我半年之后回来再看看他的进展。

# 经济学知识点回顾

好，我现在再跟你回顾一下，在这一集里咱们遇到的经济学知识点。

 **1、用好“供给”和“需求”**

我听了晓光给我的分享，得到一个重要的启发：他对于基本的经济学原理的领悟，对他创业的方向有着至关重要的影响。在经济学里，没有比“供给”和“需求”这两个概念更常见的了。但越常见、越普遍的概念，就越难把它用好。晓光能够正确地看到，在作曲领域里供给和需求两者之间的关系，从而及时地改变了他创业的方向。这不知道令他减少了多少损失，赢回了多少时间。

 **2、要有用户意识**

晓光给我的另外一个重要启发是，他在开发产品的过程中，非常注重用户的体验和反馈。他在摄影师已经停机的时候，曾经问过紫杰：“你们的产品有用户社群吗？或者更准确地问，你们的用户社群是不是在产品第一天开发的时候就建立的呢？”紫杰的回答是否定的。

我能看出，这个问题对晓光来说，是至关重要的。 *你要在开发产品的第一天就具备用户意识，用户的需求才是你开发产品的指路明灯。* 这种以最终用户为自己生产起点的想法，早在我们《薛兆丰的经济学课》介绍亚当·斯密对市场经济的基本看法的时候，就已经详细解释过了。

他说：一切商业的本源都不是我要什么，而是你要什么，都不是我想挣钱，而是我的产品对你有好处。如果你对这一部分的内容感到陌生，又或者想要得到更多的启发，我建议你一次或者再一次地重温《薛兆丰的经济学课》第6讲《亚当·斯密的人性观》。

 **3、稀缺的企业家精神**

在这一集里，我自己最大的感慨是，对大多数人来说，跟创业比起来，打工其实也挺好的。紫杰说他在不少风口行业工作过，但都没有挣到大钱，我觉得这是有原因的。当然，他受过很好的教育，他很聪明，他也遇到了不少机会。但 *企业里面论功行赏是有一定规律的，最重要的一条规律就是：谁的边际贡献越大，谁得到的回报就应该越多。*

在咱们《薛兆丰的经济学课》里，我们曾经仔细地解释过：团队的特点是，它能够产生的效应，总是要比每一个团队的成员单独所能够产生的效应加起来的和还大。也就是说，团队的特点是1+1>2。

但问题是，团队的效应是无中生有的，是混合的，是无形的，没有办法直接归功到每个成员身上。而企业论功行赏，就必须采用衡量边际贡献的办法，也就是说，我们要尽量地去看，多增加了谁企业就因此新增了多少收入，而这新增的收入就应该归功到这位新增的人身上。

 *在一个创新企业里面，最缺的往往不是专业技能，往往不是聪明才智，往往不是刻苦勤奋，最缺的往往是企业家精神。* 所谓企业家精神，就是那些敏锐的商业嗅觉，果断地能够把人、财、物聚拢在一起。

是他们，把那些分散的宝贵资源聚拢起来的，是他们把高昂的组织成本给降下来的。没有了他们，那些宝贵的资源虽然依然宝贵，但却是一盘散沙，没法聚拢起来，没法有效合作，没法形成能够卖出去的产品。

也正因为是这样，在各个风口当中能够挣大钱的，就只有那些不仅愿意当企业家，而且恰好也做对了的企业家。这一点，解释了紫杰之前虽然在风口行业却并没有挣到大钱的原因。

但问题是企业家精神其实是一种罕见的品质，并不是人人都具有的。

我在节目中说，有些人擅长唱戏，也就是把自己用好；而有些人擅长搭台，也就是把别人用好。两者兼具的人是很少见的，大多数人都只不过是能把自己用好而已，而这也是合理的。

你想想看，如果一个社会，每个人都当老板，每个人都搭台，每个人都有自己的企业，那就只会是一个遍地是小贩的社会。而在一个强大的经济体里面，你肯定会看到不少大企业，那所谓大企业，也就是大多数人都充当打工者的企业。

在节目当中，我也拿自己举例，我说我自己从来就有很清楚的自我认识。我不擅长搭台，我只能唱戏，所以我也不会去做出那些我不想做出的努力。专注于自己的比较优势，知道自己能干什么，也知道自己不能干什么，我觉得两者都是同样重要的。

# 思考题

对于那些勇于充当企业家的人来说，我觉得有一个重要的考量，就是他们手上有没有事儿，有没有可以不断迭代，并形成商业闭环的项目。

但究竟怎么去判断自己手上的创业项目算不算一件事儿，算不算是过程上可迭代的、商业上形成闭环的项目呢？似乎并不存在一个明确的定义。所以我也特别想听听你对这个问题的回答。

你不一定要给出一个明确的定义，你只需要把你的故事和经验分享出来就可以，我们可以通过感觉来体会你的回答，我觉得你只要思考过这个问题，就会对别人有启发。

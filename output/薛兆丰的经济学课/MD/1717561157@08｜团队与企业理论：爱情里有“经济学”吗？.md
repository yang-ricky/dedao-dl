# 团队与企业理论：爱情里有“经济学”吗？

今天我来跟你分享关于《万物经济学》第八集《数字游民的乌托邦：大理（下）》的经济学体会。

# 预期不同怎么办

首先，我来跟你回顾一下这一集的内容。

这一集的窗口人物是阿珍和王越，一对来大理追求理想的恋人。我们选择他俩做窗口人物，是因为他们的故事挺特别的。一般人的刻板印象是，离开大城市去大理生活的人，都是拿定了主意想要追求舒适的人。

但阿珍声明，她来大理从来就不是这么想的，来大理只是她长远生活规划当中的一步，她是奔着挣钱来的。挣了钱以后，她就可以实现她计划中的下一步，再下一步。她可能要开更多的民宿，在不同的城市购置更多的物业。她计划还要读研究生，还要成家，还要生小孩。这就跟很多到大理以后，声称钱不重要的人，形成了一个对照。

有意思的是，阿珍的男朋友王越并不完全是这样想的。王越接受了阿珍的建议，辞去了大城市大厂的工作，拿出了自己全部的积蓄，买下了大理的一家民宿，和阿珍合伙成了民宿的主理人。他本来以为，轻松舒适的生活就从此开始了，但没想到经营民宿再加上一个自媒体账号，竟然是令人日夜操劳的工作。

王越本来想要慢下来享受生活。在装点自己的民宿的时候，想要加上自己喜欢的艺术风格，在慢慢的琢磨和比较当中，享受似乎挺有艺术感的生活。但阿珍本来就没有慢下来的打算。阿珍认为挣钱就是挣钱，花太多的时间来考虑，民宿里面到底应该挂一张什么样的画、铺在院子里的石展到底应该是圆的还是方的，实在没有必要。

所以阿珍和王越之间，虽然他们是恋人，但也因为预期不同产生了对照和碰撞。所以这一集，就是从我参加阿珍和王越的一个朋友聚会开始的。

# 把握自己人生的产权

在闲聊当中，我问到：你们最梦寐以求的生活片刻会是什么样子的？其实我一直很喜欢问别人这个问题，因为从一个人的梦想时刻，是可以看出这个人的很多特点的。

而在那天的聚会里，王越对这个问题的回答是，他很希望哪天他们挣够钱了，他们的民宿可以关门一个礼拜，他能够好好地休息一下。

我说，你们淡季的时候不就可以好好休息了吗，休息一下怎么能成为你的梦想时刻呢？然后他就跟我解释，他一天到晚其实是很忙的，除了经营民宿的各种杂物，心里还要记挂着给自己自媒体账号写剧本、拍片子的事儿，每天都要创作，心是停不下来的。我完全能理解这种被事情推着走，不剩下一点个人空间的感觉。

这忽然让我想到了产权的概念，我们在得到APP《薛兆丰的经济学课》当中，花了很长的篇幅解释产权的概念。因为产权是我们追求幸福生活非常重要的基石。

所谓产权就包括了对资源的使用权、收益分配权和转让权，一个人拥有了这种对资源的自由裁量的空间，他才会有积极性，好好地为未来筹划，心安理得的感觉是非常重要的。给公司工作的时候，哪怕再忙，到了节假日，时间是属于自己的，自己就可以心安理得地使用、消耗甚至浪费。

我当场就跟阿珍和王越分享了一个经济学的小故事：

我说，有一位经济学家发表过一个研究，他发现很多家长到幼儿园接小孩的时候，都会出现迟到的现象，这给老师增加了工作的负担。最后，学校决定用收费的办法，适当地惩罚迟到的家长。你猜结果会怎么样？结果是家长们迟到的时间更长了，原因是只要罚款不至于太过分，家长的迟到反而变得心安理得了。

所以我说，王越应该拥有对自己的时间、空间的产权，他生活当中应该有一部分时间和空间是可以心安理得地使用的。而现在由于民宿和自媒体的业务，两个人可能捆绑得太紧了，没有留下这样的空间。

在这个朋友聚会当中，有一位来大理四年59岁的久哥就提供了一个很好的例子。久哥现在过着流浪的生活，他的子女已经养大成人了，他跟女儿说，我已经尽了义务了，剩下的时间就是我的了。除非你丈夫敢欺负你，别的事情我就不管了，我要好好享受人生了。

我觉得久哥就对自己的人生拥有清晰的产权，而且这并不代表他没有亲情。

# 情侣和合作者&生活和工作

为了让阿珍和王越多一点参考的素材，我跟他们去拜会了另外两对朋友。

第一对，是声音艺术家林冬先生和柏青先生。他们两人有一个非常特殊的共同兴趣，就是收集在不同场景、不同时间点上的声音，这已经成了他们的副业了。他们俩是非常好的兴趣伙伴，他们给自己定下了不同的项目目标，一个一个项目地去完成。他们收集了大量的声音样本，在不同的博物馆展出，也在他们生活的社区使用。

林冬和柏青两人对收集和处理声音的兴趣，虽然小众，但并不孤独。他们虽然只是小规模，但并不是没有大目标。他们两人玩起来虽然很沉迷，但完全不影响他们美满的家庭。

了解了他们俩生活的账单以后，我的感受是，每个人如果都能拿出自己收入的5%、 10%，投资到自己持久的兴趣当中去，不仅是合理的，而且只有这样生活才会变得有意思。

接着，我跟阿珍和王越又拜访了在大理生活了14年的著名女绘本漫画家寂地夫妇。寂地老师成名非常早，在她大学刚毕业的时候，她的绘本漫画销量就已经超过了100万册，但是寂地老师非常内向，大量的时间都是用来独自创作。

当她认识了她的先生圆圆之后，经过偷偷地考察和仔细地思考之后，寂地给圆圆写了一封长长的信，介绍了她自己，说她需要一个助理，她问圆圆能不能替她工作，顺便谈一个恋爱。圆圆只回答了三个字——喜欢的。后来他们就搬到了大理。在大理，他们也仍然是远离社交，而圆圆则不仅照顾寂地的生活，也照顾她的创作。

像寂地和圆圆这样，能在生活和工作上都非常和谐搭配的夫妇，其实是很少见的。在我看来，这主要是圆圆除了照顾寂地以外，没有自己很不一样的个人目标的缘故。

我对阿珍和王越说，如果做不到这一点，情侣或者夫妻之间就没有必要非把工作和生活都拧在一块。大多数的情侣和夫妻其实都是生活在一块，但工作是分开的，这是因为经济学的比较优势原理告诉我们，分开工作收益往往更大。一个感情上好的伴侣，并不一定是工作上好的同事，给对方做出适当的选择，对维系感情来说可能是更好的。

面对阿珍和王越的故事，我想说的只是， *人们在追求浪漫生活的时候，也仍然逃避不了很多涌动的暗流，很多关系的处理，以及很多跟分工和产权相关的经济规律。* 

# 经济学知识点回顾

好，那我现在再跟你回顾一下这一集里咱们遇到的经济学知识点。

 **1、资源的重要性**

首先，好的经济学从来不只看重钱，而是看重所有的资源。一个人的时间、空间、注意力、自由发展的选项，其实都是他的资源。 *每个人都是自己幸福的第一责任人，如果对这些资源不拥有产权，他就没法很好地发展自己，他也就没法享受个人发展所带来的好处。*

一个人的可爱往往是因为他发展了才变得更可爱的，而不是他可爱了就不需要发展了。这一点对所有人，尤其是在亲密关系当中的年轻人来说，我觉得是非常重要的。

 **2、计量经济学中的两种错误**

第二个有点难、但挺有趣的知识点，是计量经济学当中的两种错误。这是我在观察寂地夫妇的生活模式的时候想到的。你看，关于社交，人们有两种极端：

一种是热衷于社交，见了谁都把他当朋友，直到发生了什么事情证明这个人不是自己的朋友为止。这种做法既让自己的朋友数量最大化了，从朋友圈当中得到的好处最大化了，但同时也让遇人不淑所带来的伤害最大化了。

而另外一个极端是尽量避免社交，尽量减少交朋友，直到对方充分地证明是一个值得交的朋友为止。采用这种社交策略的人，朋友数量最小化了，但朋友可能带来的伤害也最小化了。

我们在得到APP《薛兆丰的经济学课》“第154讲｜让子弹飞一会儿”当中，讨论反垄断法的商业模式和竞争政策的时候，就介绍过这两种策略。积极社交的人，为的是努力减少假阴性的错误，但他就更可能犯了假阳性的错误。而努力回避社交的人，他们的目的是想要减少假阳性的错误，但这个过程中他们就容易犯上假阴性的错误。

假阴性和假阳性这两个概念虽然有点绕，但它给我们生活的启发却是清楚的。那就是，无论你走哪个极端，都是各有利弊的，而我相信寂地夫妇所采取的策略，是他们刻意为之，对他们来说是最好的。

 **3、团队和依赖关系**

第三个知识点是关于团队和依赖关系的。在我们的课程当中，我们曾经花费了大量的时间解释团队的概念，以及如何在依赖的关系当中保护自己的策略。我们说人是群居的动物，人与人之间要结成团队，才能创造更多的财富，才能做到1+1>2。

但是在跟别人的合作当中，合作得越紧密，依赖关系就越深，一旦遇到有人不靠谱的事情，自己受到的伤害就会越大。婚姻是一种约束，但它也是一种保障，没有了婚姻，伴侣之间的投入可能就会减少；但如果有了婚姻，而事情的发展又不能如人所愿，有人就会在这依赖关系当中受到更大的伤害。

# 思考题

怎么样才能够结成合适的依赖关系，增强这种依赖关系的绑定，从而促使被绑定的人做出更多的付出，但同时又要防止不靠谱的人和事对自己造成伤害？这种困难我相信你一定遇到过。你的故事是怎样的呢？我很想听到你的分享，这也是咱们经济学当中一个永恒的开放的问题。

# 问题1

> 投资者的情绪偏差，也是基本面的一部分吗？

薛老师：是的，不存在客观的基本面，没有完全客观的不以人的意志为转移的价值。我们说，任何资产的现值都是预期中的未来收入流的折现和。只要你的认识发生了变化，它瞬间就会体现到资源的现值当中， *请注意，我指的“认识的变化”，不是你一个人的认识发生了变化，而是整个社会的认识发生了变化。*

这当中有一个非常关键的问题，就是我们的情绪、误解、留言，以及各种的以讹传讹的坏消息，它们的出现有没有规律？一旦它是有规律的，比方说，大家一到星期五，情绪就低落，这时候，它的股价就偏低。假如这是一个规律，那么，它就会成为我们的新知识。一旦利用了这个知识，人们就会知道周五的股价偏低，因此都会买进，使本来过低的股价回到了基本面，这就抵消了情绪的波动。

市场上，有人专门研究人的情绪，并利用这种情绪，使股价回到基本面。所以，我们说， *情绪如果有规律，那么，这种规律就会被人们识别，并被用来推测新股价，于是，情绪的作用就在股价当中被抵消了，并体现在当前的价格中。*

市场上总有各式各样的误解，但与此同时，也有消除误解的力量，它就是逐利的力量。逐利的力量非常巨大，且经常存在，这使得我们的价格在这两种力量的夹击下，最大程度地逼近于应有的价格。

我们都知道，发现知识是艰难的，而且，有时候是会发生重大偏离的。所以，股价的波动也是巨大的。我们的知识不够，我们的股价就会被误导，但是，逐利的巨大力量能最大程度激发我们追求知识的积极性。

我经常说，最了解天气的人，并不是负责天气预报的人，因为预报天气给你的影响很小，顶多是加减衣服，带不带伞。最能准确预报天气状况的，是炒期货的人，因为他需要知道明年的天气对庄家的影响，这当中有巨大的得失，所以，炒期货的人在追求天气预报准确这件事上的投资是巨大的。

所以，我们常说的“知识是无价的”，是文人的感叹， *从经济学的角度出发，知识是有价的。因为追求不同的知识、追求知识的精确程度，是有巨大代价的。* 在为了获得巨大投入或为了避免巨大损失的情况下，为了使我们的知识更加精确，我们会进行巨大投入。这就是逐利的力量。

# 问题2

> 有稳赚不赔的生意吗？

薛老师：有人说我们投资巴菲特（的股票）肯定能稳赚不赔。我们姑且假设有这回事，（我当然不相信有这回事，它可能会在某段时间内出现，但它不会永久），那么接下来会发生什么事情呢？你的钱会投不进去，而他的股票的现值就会涨得非常高，因为你稳赚。假如你把一枚代币投进糖果机，糖果机给你吐出十块钱，那么这台糖果机就值十块钱，这时候，你花多少钱才能把这台糖果机买回来呢？

在竞争之下，这台糖果机的现价就会逼近十块钱，以至于它的边际利润趋近于零，也就是说，所有糖果机的回报率趋于一致。 *我们讲过，全世界的平均回报率是一张互联网，所有人逐利的动机使得每一个项目的边际回报率趋于相等。*

# 问题3

> 如果预计资源紧缺，是否应该提前囤货？

薛老师：在“稀缺”这个概念上，我们引入了时间维度，这该怎么理解呢？你可以把“今天的我”“明天的我”“后天的我”看作一台台排列开的糖果机，这时，你会怎么使用你有限的资源呢？你会把所有资源投在今天的糖果机里面吗？你会今朝有酒今朝醉，不管后面的生活吗？不会。

我们讲糖果机的例子，哪怕是做公益性质的眼科手术，我们也会不断分摊硬币，而且总会把最后一枚硬币投给能吐出最大糖分糖果的糖果机里。

我们对未来有指望，我们不仅要活今天，还有活明天，活后天，所以，我们有限的资源是要分给今天、明天、后天的，比如，我们不能为了取暖把家具烧掉，也不能为了吃肉把下蛋的母鸡吃掉。

当然，在分配的过程中，因为有不确定性，我们分给今天的资源会略多一点，分给明天、后天的资源会少一点，因为要服从真实利率。但哪怕是这样，我们都要在我们的有生之年，把我们的资源平均分配，这样才能使我们的总收益达到最大。

 *这就是欧文·费雪（Irving Fisher）说的，投资实际上是时间维度的平衡消费。* 你可以把“时间维度”理解成“糖果机”，实际上就是在“今天的我”“明天的我”“后天的我”之间平衡消费，把手中有限的资源分给不同的“我”，从而使总收益最大。

但是，这个问题的前提是，资源稀缺。在这个确定的条件下，我们是不是需要每天省一点？因为你要给未来留一点。那么，怎么才能在今天和未来之间进行均匀的分配呢？除了价格，还有更好的驱动力吗？

东西贵了，咱们就省着点，这是所有人都会服从的，不管是商人囤，还是家庭内部囤，结果都一样，都是省着用。然后，人们会积极寻找这件东西的替代品，从而应付资源的稀缺。

好，我们回过头来说，假如“资源稀缺”是一则假消息，在逐利的动机下，一旦有人发现“资源稀缺”是假的，它就可以用高价抛售原材料，获得巨大利益。这个人可能想隐藏这个消息，但是他的行为已经背叛了他，而一旦这些资源被卖出去，人们就会知道，这种资源其实不稀缺，于是，这件东西就无法维持高价，人们就会放开使用。

# 问题4

> 本科读经济学，硕士想读其他专业，博士再回过头来学经济学，是否可行？

薛老师：你要想想你的终局是什么，不是学得越多越好。我们常在学校里看到这样一种人，他们被称作“读书专业户”：总是读完一门学科，再读一科，读很多科。经济不好的时候，我们会发现，入学率、研究生报考率就高，这其实是一种回避。短暂回避是可行的，但不能长久地回避。

 *你出于什么目的学这几个科目？能解决什么问题？* 我曾经给一个朋友写过推荐信，他说的一句话让我非常感动，他说，薛老师，我硕士毕业后工作这么多年，实在没办法了，我一定要回去读书，因为我积累了太多问题了。他积累问题后再回去读书，和一直在学校读书的人的眼光就会不一样。

# 问题5

> 为什么连货币都没有的原始社会都会存在利率？

薛老师：我的回答是，只要有时间，就有不确定性，就有时间偏好，人们就会掌握现在的、稳定的、实在的东西，比较不喜欢有风险的东西，这时候，对未来就要做补偿，利率就会存。所以，没有货币的社会也会有利率；反对利率的社会里面其实也都有利率，只不过，在利率的价格管制之下，会出现各种问题。

# 问题6

> 产业标准由政府牵头好，还是由大企业牵头更好？

薛老师：我们先看用户“狂人先生”是怎么回答的。狂人先生说，其实无论是政府主导，还是大企业主导，都未必全面。虽然表面上看，产业标准既有可能是政府强制牵头，也有可能是大企业决定，但最终还是要回归到用户的习惯当中。

这个回答非常好。标准的产生是平衡边际的好案例。一个好标准中，需要达成的指标太多了，它要面向未来，它要把新的、现有的资源最大程度地利用起来，但是，还有个标准是，它要照顾现有用户的习惯，也就是说，它要照顾以前的设备或资源所使用的标准，它既要向上兼容，也要向下兼容，不能跳跃，因为没人愿意跟着跳。

 *这当中，我们需要平衡很多指标，这时候怎么办呢？有个办法是，竞争，也就是让不同的厂家推出不同的标准，让这些不同的标准竞争。* 今天，我们的视频标准非常复杂，视频上传后会出现各种各样的问题，为什么？因为我们面对的问题很复杂。这种乱，实际上是人们在通过竞争的方式寻找最佳的边际平衡时的一个重要过程。

# 问题7

> 从经济学的角度怎么来看存在感？

薛老师： *从经济学的角度来看，存在感就是被需要感。* 如果你知道自己有很独特的、能够奉献出去的地方，你就会知道在某个小环境里面，某个特定的关系里面，你是被需要的。如果你想被需要，你就得贡献。

每当我讲起这个问题，我总是回过头看亚当·斯密（Adam Smith）的名言， *亚当·斯密说，我从来不说我需要什么，而说这件事情对你有好处。我们从来不诉诸于每一个人的利他心，即“你应该替我做什么”，而是说，“这件事对你有好处”。*

餐桌上的面包、酒和肉，不是来自面包师、酿酒师和屠夫的善行，而是来自他们对自身的关切。你看，这虽然是大白话，却有非常深邃的思想。

那么， *你怎么才能被社会需要呢？办法是：你要关注你手头的那点事情。*

# 问题8
